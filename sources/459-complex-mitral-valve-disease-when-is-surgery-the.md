# 复杂二尖瓣疾病：何时手术是更好选择？

## 核心要点

* 讲者核心立场是“解剖决定工具”：M-TEER 适用于较窄的解剖窗口，而手术修复可按瓣叶、腱索、瓣环和钙化情况组合技术。
* 5 类复杂解剖常使 TEER 受限：Barlow 病、交界区脱垂、多发喷射、瓣环扩张合并脱垂、二尖瓣环钙化（MAC）。
* Barlow 病中 M-TEER 达到 MR 0/1+ 的比例仅 26%，非 Barlow 为 42%；3 年 MR ≤2+ 维持率 62% vs 80%。
* 交界区脱垂手术修复的 11 年免再次手术率为 97.4%，免心源性死亡率为 95.2%。
* MAC 中夹合通常依赖 6–9 mm 的可抓持瓣叶长度；外科 MAC 系列 82 例显示 30 天死亡率 1.2%，1 年 87% 存活且状态良好。
* 低危和中危退行性 MR 患者中，外科二尖瓣修复的手术死亡率约 0.21%–0.48%，10 年免再次手术率 96.6%–99.4%。

### 解剖决定工具

复杂二尖瓣解剖中，Barlow 病、交界区脱垂、多发喷射、瓣环扩张合并脱垂和 MAC 常超出 M-TEER 的理想窗口。

本页标题为“ANATOMY DICTATES THE TOOL”。为什么复杂二尖瓣解剖中更偏好手术：M-TEER 在较窄的解剖窗口内工作；5 个常见表现包括 Barlow's disease、commissural prolapse、multiple jets、dilated annulus with prolapse、mitral annular calcium，这些情况经常落在 M-TEER 窗口之外；手术没有同样的“解剖天花板”。底部给出三组关键数字：退行性 MR 修复后 10 年免再次手术率 96.6%；高容量中心孤立二尖瓣修复手术死亡率 <0.5%；PASCAL IID 中 M-TEER 适合病例里有 ≥2 个独立喷射者占 37%。

### 场景 1：Barlow 病

Barlow 病为双叶、多节段、弥漫性黏液样冗余，M-TEER 难以用一个线性夹合目标解决。

Barlow 病特点为双叶弥漫性黏液样组织冗余、腱索长度不一、瓣环扩张以及 SAM 风险，缺少单一治疗靶点。

| M-TEER：结构不匹配 | 手术：不受单一解剖形态限制 |
| --- | --- |
| 双叶、多节段脱垂难以用“夹合一个节段两点”的器械处理。 | 可按每个节段、每片瓣叶选择或组合切除、非切除 neochordae 和 edge-to-edge 技术。 |
| 最佳结果 MR 0/1+ 在 Barlow 患者仅 26%，非 Barlow 解剖为 42%，P=0.02。 | Barlow 病所有外科技术总体手术成功率 93.5%。 |
| 3 年 MR ≤2+ 维持率：Barlow 62%，非 Barlow 80%，P=0.03。 | 孤立瓣环成形、neochordae 和切除的 5 年结局相当，手术可适应瓣膜。 |
| 3 年心衰住院：Barlow 17%，非 Barlow 7%，P=0.07。 |  |
| 夹合后 10% 需要后续外科二尖瓣干预。 |  |

### 场景 2–3：交界区脱垂与多发喷射

交界区脱垂和多发独立喷射的共同问题是病变偏轴或多灶，线性夹合几何难以覆盖。

交界区脱垂与多发喷射对经导管治疗的共同缺陷是病变 off-axis 或 multifocal，单一线性夹合几何无法到达或无法覆盖缺损。

| 解剖场景 | M-TEER 挑战 | 外科能力 |
| --- | --- | --- |
| 交界区脱垂 | 喷射位于瓣叶交界，偏心且偏轴；即便在现代系列中，也需要非标准夹角和高级术者经验。 | 交界闭合加瓣环成形是快速、可靠、专门针对该病变的技术。11 年免再次手术率 97.4%，免心源性死亡率 95.2%。微创修复耐久性与标准后叶脱垂修复在统计上相当。 |
| 多发独立喷射 | PASCAL IID 注册研究中，复杂 TEER 解剖病例 37.2% 有多发独立喷射。每个喷射可能需要单独夹合，增加狭窄风险、操作时间和技术负担。若 flail gap >10 mm 或双叶病变驱动多发喷射，1 年 MR 复发率 15%–25%。 | 直视下可一次处理每个喷射来源，并按节段需要选择切除、neochordae 和瓣环成形。 |

### 场景 4：瓣环扩张合并脱垂

瓣环扩张合并脱垂时，瓣环本身是治疗靶点；M-TEER 无法按尺寸重塑瓣环。

当瓣环扩张和瓣叶脱垂共存时，瓣环本身就是治疗靶点，不只是瓣叶。M-TEER 没有机制按尺寸缩小瓣环，而外科瓣环成形环正是为此设计。

| 为什么瓣环重要 | 外科答案：按尺寸环成形 |
| --- | --- |
| 单个夹子可产生可测量的类似瓣环成形效应，即缩短 A-P 径，但这是附带效应，不是有尺寸控制、可持久的矫正。 | 刚性或半刚性环按 intertrigonal distance 选择尺寸，可明确恢复瓣环尺寸和几何。 |
| 没有真正的环成形时，扩张瓣环在术后仍会继续重构，威胁 MR 复发。 | 瓣环成形加瓣叶矫正，如切除或 neochordae，可在一次手术中同时处理脱垂及其瓣环驱动因素。 |
| 严重瓣环扩张合并脱垂是 TEER 结果不佳和再次干预率更高的已知驱动因素。 | 外科系列持续显示，环成形配合瓣叶修复后，10 年免再次手术率 >96%。 |
| 单纯 clip 策略留下瓣环几何未被矫正，这是退行性 MR 晚期修复失败的主要驱动因素。 | 无论瓣环扩张程度如何，手术可用同等精度同时矫正瓣环和瓣叶病变。 |

### 场景 5：二尖瓣环钙化（MAC）

MAC 限制瓣叶长度和活动度，而这两点正是夹合抓持组织的基础。

MAC 限制瓣叶长度和活动度，正是夹合器械抓持组织依赖的两个条件。手术可以物理去除钙化，而 TEER 只能绕开钙化工作。

| M-TEER：瓣叶长度常成为禁忌 | 手术：必须直接处理 MAC |
| --- | --- |
| 不同夹子型号要求的最小可抓持瓣叶长度约 6–9 mm。 | 直接去钙化，包括锐性切除和超声清创，可恢复原生瓣环几何和可抓持、可活动瓣叶组织。 |
| 主要综述中，重度 MAC 曾被视为经皮修复的明确禁忌。 | 当钙化浸润室壁时，可进行房室沟补片重建。 |
| 钙化延伸到瓣叶、后叶回缩、小瓣口面积都会减少可抓持组织。 | 82 例外科 MAC 系列：30 天死亡率 1.2%，1 年 87% 存活且状态良好，MAC 本身不应成为开放手术禁忌。 |
| 尝试 TEER 的 MAC 患者更常需要转外科。 | 可进行真正修复或正确坐落的置换，而不是被当前钙化量限制的操作。 |
| 即使可行，MAC 患者 1 年死亡率也呈更高趋势：19.7% vs 11.3%，P=0.05。 |  |

### 底线：全解剖谱中的优秀结局

低危和中危患者中，外科二尖瓣修复具有低死亡率和高长期耐久性。

页面结论为：在低危和中危患者中，外科二尖瓣修复不是 TEER 受挑战解剖的后备方案，而是无论上述 5 种场景如何都可提供耐久、确定性治疗。

|  |  |
| --- | --- |
| 孤立二尖瓣修复手术死亡率 | 0.21%–0.48% |
| 退行性 MR 修复 10 年免再次手术率 | 96.6%–99.4% |
| 修复后 10 年生存率 | 84%–90%，与年龄匹配一般人群相当 |
| 当代系列中修复成功率 | Barlow、交界区和双叶解剖中为 93%–100% |

最后两条信息：手术的稳定优势是可以把技术匹配到术者面前的具体解剖；对适合手术的患者，正是这种灵活性和耐久性数据，使指南继续推荐外科修复作为低危和中危退行性二尖瓣病的标准治疗，而 M-TEER 保留给手术风险禁忌者，或解剖有利、病变局灶者。
