# 逐步操作（Case-in-a-Box）与模拟器培训

## 核心要点

* 本套幻灯片是 Sapien M3 经导管二尖瓣置换（TMVR）的 live case / simulator step-by-step 训练，核心内容是从 TSP 到 dock 释放，再到 M3 瓣释放后的影像复核。
* 器械顺序为：建立跨房间隔路径，导入并定位 guide，送入 DSC，经 medial commissural access 前进 dock，完成 1/4 turn、第一圈、重捕获、三圈捕获确认，再 unsleeve dock。
* dock 释放阶段强调显露心房侧 dock、seat PVL guard、处理 dock suture slack、测量 dock height、剪断缝线并完成最终 dock release。
* 瓣膜阶段为经 dock 中央送入 pigtail，经 pigtail 将预塑形 LV wire 送至左室心尖，然后把 Sapien M3 valve 送入 guide 并跨过 dock，定位后部署。
* 最终检查包括 M3 valve post-dilation +4 cc、评估 M3 valve 和 LVOT，以及评估 iASD。
