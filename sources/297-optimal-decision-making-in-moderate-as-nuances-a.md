# 中度主动脉瓣狭窄的最佳决策：细节与表型

## 核心要点

* 中度AS不是单一疾病，而是包含一致性/不一致性分级、低流低压差、额外瓣外心脏损害、混合AS+AR、症状状态和影像/实验室高危标志的异质性谱系。
* 按EACVI/ASE定义，中度AS对应Vmax 3.0-4.0 m/s、平均压差20-40 mmHg、AVA 1.0-1.5 cm²、AVAi 0.60-0.85 cm²/m²、DI 0.25-0.50。
* 流行病学上，中度AS并不少见：PREVUE-VALVE中总体患病率6.4%，80-85岁达11.5%；NEDA显示未经治疗中度AS死亡风险升高，HR 1.94。
* 瓣外心脏损害在中度AS中常见。1,245例中度AS中，心脏损害Stage 2占42.6%，Stage 3占10.6%，Stage 4占6.9%。
* 目前默认策略仍是观察等待。干预有明确支持的场景主要是合并其他心脏手术，以及混合中度AS+AR伴高梯度或LVEF降低。
* TAVR UNLOAD纳入178例中度症状性AS、LVEF 20-50%、NYHA ≥II且接受GDMT的患者；PROGRESS和EXPAND II仍在提供TAVI治疗中度AS的新证据。

### 主动脉瓣狭窄的疾病连续谱

AS从健康瓣叶、硬化、轻度、中度到重度逐步进展，伴随脂质沉积、炎症、纤维化和钙化。

本页把AS分为Stage A、Stage B和Stage C/D。Stage A为AS风险期，风险因素包括遗传因素Lp(a)、解剖因素二叶式主动脉瓣，以及年龄较大、男性、高血压、糖尿病、高脂血症和代谢综合征。Stage B覆盖主动脉瓣硬化、轻度AS和中度AS。Stage C/D为重度AS。

| 参数 | 瓣膜硬化 | 轻度AS | 中度AS | 重度AS |
| --- | --- | --- | --- | --- |
| Vmax，m/s | ≤2.5 | 2.6-2.9 | 3.0-4.0 | ≥4.0 |
| 平均压差，mmHg | 无 | <20 | 20-40 | ≥40 |
| AVA，cm² | 无 | >1.5 | 1.0-1.5 | <1.0 |
| AVAi，cm²/m² | 无 | >0.85 | 0.60-0.85 | <0.60 |
| DI | 无 | >0.50 | 0.25-0.50 | <0.25 |

主动脉瓣硬化通常无症状，可有杂音，超声可见瓣叶局部增厚和钙化，Vmax通常<2 m/s。轻度AS通常无症状，Vmax<3 m/s。中度AS通常无症状，Vmax 3-4 m/s，左室收缩功能正常，可伴左室肥厚和舒张功能改变。重度AS可有症状或无症状，Vmax>4 m/s，有症状者若不早期AVR死亡率高；部分患者因左室收缩功能下降或小左室腔而表现为较低Vmax。页内还提示接近轻中度边界的例子可为Vmax约3.1 m/s、mPG 21 mmHg；接近中重度边界的例子可为Vmax约3.9 m/s、mPG 39 mmHg。

### 中度AS流行病学

左侧PREVUE-VALVE显示中度AS患病率随年龄升高，右侧NEDA显示中度和重度AS生存曲线下降更快。

PREVUE-VALVE研究从美国零售药房随机选择3,000人，按ASE标准使用综合TTE评估中度AS患病率。总体患病率为6.4%，65-69岁为3.8%（n=1,230），70-74岁为4.9%（n=871），75-79岁为9.0%（n=471），80-85岁为11.5%（n=428）。

NEDA为澳大利亚国家超声心动图数据库，包含241,303人。生存图横轴为随访年数0-16年，风险人数依次为241,303、169,882、101,596、59,763、33,275、16,690、6,651、1,912、175。调整模型显示年龄HR 1.07（95%CI 1.07-1.07，p<0.001），男性HR 1.38（95%CI 1.36-1.40，p<0.001），无AS为参照，轻度AS HR 1.48（95%CI 1.44-1.52，p<0.001），中度AS HR 1.94（95%CI 1.86-2.02，p<0.001），重度AS HR 2.07（95%CI 1.96-2.19，p<0.001）。页面结论是中度AS并不少见，且未经治疗的中度AS死亡率较高。

### 中度AS中的瓣外心脏损害

压力负荷可从主动脉瓣狭窄进展为替代性纤维化，并按左室、左房/二尖瓣、肺血管/三尖瓣、右室损害分期。

左侧示意图显示主动脉瓣狭窄造成压力负荷，进一步发展为替代性纤维化。心脏损害分期标准为：Stage 0无心脏损害；Stage 1为左室损害，包括左室肥厚、LVEF<50%、E/e'>14；Stage 2为左房/二尖瓣损害，包括左房扩大、房颤存在、中重度二尖瓣反流；Stage 3为肺血管/三尖瓣损害，包括肺动脉收缩压>60 mmHg、中重度三尖瓣反流；Stage 4为右室损害，包括中重度右室功能不全。

右侧数据来自1,245例中度AS患者，定义为AVA 1.0-1.5 cm²且DI≥0.25，平均年龄71岁，女性50%。心脏损害分布为Stage 0 13.1%，Stage 1 26.8%，Stage 2 42.6%，Stage 3 10.6%，Stage 4 6.9%。

### 中度AS的异质性疾病谱

中度AS应按血流-压差模式、瓣外损害、混合瓣膜病、症状和影像/实验室标志识别高危表型。

| 维度 | 潜在高危表型 |
| --- | --- |
| 血流-压差模式 | 一致性中度AS；不一致性中度AS；保留EF低流低压差LF-LG；降低EF低流低压差LF-LG；正常流低压差NF-LG。 |
| 瓣外心脏损害 | 左室舒张或收缩功能不全；左房、二尖瓣、肺血管、三尖瓣或右室损害。 |
| 混合主动脉瓣病 | 混合中度AS和中度AR。 |
| 症状状态 | 无症状；有症状，包括NYHA ≥II、运动能力下降、既往心衰事件和心绞痛。 |
| 影像/实验室标志 | 主动脉瓣钙化体积；CMR心肌纤维化；超声GLS；BNP。 |

本页总句为，中度AS需要识别潜在高危表型。

### 一致性与不一致性中度AS

1,974例中度AS按平均压差、SVI和LVEF重新分型，不一致性中度AS占40%，死亡曲线显著分离。

Stassen等研究纳入1,974例中度AS患者，AVA 1.0-1.5 cm²，来自3个学术中心，平均年龄73岁，男性51%。根据平均PG 20-40或<20 mmHg、SVI <或≥35 mL/m²、LVEF <或≥50%重新分类。

分布为一致性AS 60%，不一致性AS 40%。不一致性AS内，降低LVEF的LF-LG占12%，保留LVEF的LF-LG占6%，NF-LG占22%。DSE和CT来源的AVC评分未纳入分类。

全因死亡曲线随访至60个月，p<0.001。约60个月时，降低LVEF的LF-LG死亡率最高约64%，保留LVEF的LF-LG约51%，NF-LG约37%，一致性AS约36%。

### 中度AS评估路径

不一致性中度AS先用DSE确认，DSE不可行或不确定时用CT主动脉瓣钙化评分确认，并同时寻找高危人群。

确认“中度”严重程度时，若为不一致性分级的中度AS，先进行多巴酚丁胺负荷超声DSE。若同时满足以下全部条件，即可确认为中度AS：AVA 1.0-1.5，或AVA<1.0且AVAi>0.6；Vmax 3-4 m/s；平均PG 20-40 mmHg。若DSE不可行或结论不明确，则使用CT主动脉瓣钙化评分：女性400-1200 AU，男性800-2000 AU，符合则确认为中度AS。

高危人群识别包括：AS相关症状，即NYHA ≥II、运动能力下降、既往心衰事件；超声心动图提示瓣外心脏损害、GLS受损、混合中度AS和中度AR；实验室标志物提示BNP或NT-proBNP升高；心脏MRI显示替代性纤维化。

### 混合中度AS和AR

年龄和性别匹配比较显示，混合中度主动脉瓣病MAVD的不良事件风险接近孤立重度AS，而明显高于孤立中度AS或AR。

Egbe等研究比较无症状、LVEF≥50%的混合中度主动脉瓣病患者与孤立AS或AR患者，年龄和性别匹配；患者平均年龄63岁，男性73%，时间范围为1994-2013年。上方曲线显示至15年时，心源性死亡+AVR+症状的不良事件接近95-100%，心源性死亡+AVR接近90-95%，单独心源性死亡约20%。风险人数表每组基线均为251，约5年时为117、138、218，约10年时为53、71、159，约15年时为26、31、94。

下方三幅图显示8年不良事件：图A中MAVD高于中度AR，p<0.0001，约8年终点MAVD约85%、中度AR约40%；图B中MAVD高于中度AS，p<0.0001，约8年终点MAVD约85%、中度AS约60%；图C中重度AS与MAVD相近，p=0.49，两者约80-85%。页面结论是，混合中度AS和AR患者结局类似孤立重度AS，而不是孤立中度AS或AR。

### 中度AS决策总结

本页总结为：中度AS不是单一实体，而是包括若干预后和管理含义不同的表型。多模态影像有助于判定AS真实严重程度，并发现潜在高危人群。观察等待仍是默认策略；目前支持干预的两个场景为：1）同期心脏手术，ESC推荐IIa C；2）混合中度AS+AR伴高跨瓣压差或LVEF降低，ESC Class IB-NR。PROGRESS和EXPAND II将为中度AS患者TAVR提供更多认识。
