# A类和B类报销：CMS如何看数据和方案

## 核心要点

* FDA/CDRH负责IDE器械的安全性、有效性、A/B类归类和IDE批准；CMS独立判断Medicare覆盖。
* A类为实验性器械，常规护理可覆盖，但器械本身因法定排除不覆盖；B类为非实验性，器械和相关服务可覆盖。
* CMS最关注Medicare受益人是否纳入并被代表、是否测量Medicare人群相关结局、研究方法能否可靠回答问题。
* IDE研究不仅服务FDA批准，也应提前为未来NCD/LCD覆盖标准准备证据。

### FDA与CMS：正式协作但角色不同

2016年正式MOU下，FDA/CDRH对器械分类，CMS使用该分类确定Medicare覆盖范围。

该页主题为“FDA和CMS：正式合作，但角色不同”。2016年正式MOU规定，FDA/CDRH对器械进行分类，CMS使用该分类来做Medicare覆盖决定。

| FDA | CMS |
| --- | --- |
| 审查器械安全性和有效性 | 独立审查Medicare覆盖 |
| 根据MOU将器械归为A类或B类 | 使用FDA分类确定覆盖范围 |
| 批准IDE，决定研究能否启动 | 确保受益人被纳入，且结局相关 |
| FDA批准不等于Medicare覆盖 | CMS批准不等于FDA IDE批准 |

底部要点为：两个机构越来越多地把IDE研究设计与未来覆盖标准对齐，研究必须同时为FDA和CMS而设计。

### A类与B类的核心区别

A类实验性器械和B类非实验性器械在器械、相关服务和MAC判断范围上的差异。

| 类别 | 覆盖要点 | MAC判断范围 |
| --- | --- | --- |
| A类：Experimental | 常规护理覆盖；器械不覆盖，因为属于法定排除；研究会公开发布在CMS网站 | 仅判断常规护理覆盖 |
| B类：Non-Experimental | 常规护理覆盖；器械和相关服务覆盖；研究会公开发布在CMS网站 | 判断相关护理加常规护理覆盖 |

### CMS要求什么，以及为什么不同于FDA

CMS标准强调Medicare受益人、相关健康结局和可靠方法学，另有6项剩余标准。

核心句为：Medicare受益人必须成为试验的一部分，而且必须测量他们的相关结局。

| CMS关注点 | 要求 |
| --- | --- |
| 谁受到影响 | Medicare受益人必须被纳入，并在试验中得到代表 |
| 测量哪些结局 | 必须评估与Medicare人群相关的健康结局 |
| 方法学是否可靠 | 研究设计必须可靠回答研究问题 |

剩余标准包括：研究理由有科学证据支持；不重复已有知识；赞助方有能力完成研究；保护人类受试者；在ClinicalTrials.gov注册；所有结局均预先指定并发表。

### IDE研究是覆盖决定的构件

IDE研究从FDA IDE批准下收集安全性/有效性数据，经过CMS研究费用覆盖，最终形成NCD/LCD证据基础。

该页把IDE研究放到更大图景中。流程为：IDE Study，即在FDA IDE批准下收集安全性和有效性数据；CMS Approval，即如果满足Medicare标准，CMS覆盖研究成本；Evidence，即取得可推广、已发表结果；NCD/LCD，即CMS作出全国或地方覆盖决定。

底部警示为：赞助方如果设计IDE研究时没有考虑未来会面对的NCD标准，就会让自己暴露在覆盖风险中。
