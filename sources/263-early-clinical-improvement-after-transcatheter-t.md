# 经导管三尖瓣置换与修复后的早期临床改善比较

## 核心要点

* 单中心回顾性队列纳入 31 例严重 TR 介入患者，其中 T-TEER 22 例，TTVR 9 例，比较 30 天 NYHA 和 TR 严重度改善。
* 30 天 NYHA 改善：T-TEER 为 15/22 (68.2%)，TTVR 为 9/9 (100%)，p = 0.077。
* 30 天 TR 严重度改善：T-TEER 为 19/22 (86.4%)，TTVR 为 9/9 (100%)，p = 0.54。
* 伴随 RV-PA coupling 分析提示，随访时 preserved coupling 为 T-TEER 3/23 vs TTVR 4/9，OR 9.35，p = 0.03；初始 uncoupled 患者 recoupling 为 TTVR 3/4 vs T-TEER 1/17，OR 31.55，p = 0.01。
* 研究提示 TTVR 的早期改善更一致，但样本量小、单中心，统计效能有限，需要更大队列验证。

### 背景和研究目的

严重三尖瓣反流 TR 与生活质量差、高死亡率和住院增加相关。经导管治疗为外科高风险患者扩大了治疗选择。当前形成了两种主要经导管策略：tricuspid edge-to-edge repair (T-TEER) 和 transcatheter tricuspid valve replacement (TTVR)。

尚不清楚置换是否比修复带来更一致的早期临床改善。研究目的为比较 T-TEER 与 TTVR 的 30 天临床反应，具体终点为 NYHA class 和 TR severity 的变化。

### 三尖瓣介入关键试验背景

TRILUMINATE Pivotal 与 TRISCEND II 是不同 RCT，不能视为头对头比较。

T-TEER 的代表为 TRILUMINATE Pivotal，策略是 leaflet approximation，device + OMT 对比 OMT alone。TTVR 的代表为 TRISCEND II，策略是 bioprosthetic valve replacement，device + OMT 对比 OMT alone。

| 2 年结局 | T-TEER (TRILUMINATE) | TTVR (TRISCEND II) |
| --- | --- | --- |
| TR reduction | 84% ≤ moderate | 约 95% ≤ mild |
| KCCQ quality-of-life gain | > 15 points | +17.8 points |
| All-cause mortality (ITT) | 17.9% vs 17.1% (NS) | 19.0% vs 44.9% |
| HF hospitalization | 下降 28%，HR 0.72，p = 0.02 | 26.8% vs 37.1% |
| New pacemaker | 5.5% | 19.7% |
| Severe / major bleeding | 约 3.2% | 10.4% |

该页 takeaway 为：TTVR 获得更完全、持久的 TR reduction；T-TEER 更安全，new pacemaker 和 bleeding 更少，并在 2 年降低 HF hospitalization。页脚强调这些是 separate RCTs，不是 head-to-head；引用 TRILUMINATE 2-yr (Circulation 2025) 和 TRISCEND II 2-yr (ACC.26, 2026, preliminary)。

### 手术方式选择因素

解剖、导线、影像窗口、肾功能和抗凝耐受性影响 T-TEER 与 TTVR 选择。

| 可能更支持 T-TEER (Repair) | 可能更支持 TTVR (Replacement) |
| --- | --- |
| Septal-lateral coaptation gap ≤ 7 mm | Septal-lateral coaptation gap > 7 mm |
| Anterior-septal jet location | Anterior-posterior jet location |
| ≤ 3 TV leaflets | > 3 TV leaflets or scallops |
| Focal primary TV disease，例如 prolapse；mild leaflet tethering | Significant TV leaflet tethering or thickening |
| No CIED leads，或 leads 位于 commissure/远离 TR jet | CIED-related TR |
| Good intraprocedural windows for lateral visualization | Poor leaflet visualization |
| Advanced renal dysfunction，因为 TTVR planning 需要 contrast CT | 未列出对应 TTVR 优势 |
| Intolerance / inability to take anticoagulation | 未列出对应 TTVR 优势 |

该表改编自 2025 ACC Expert Consensus Decision Pathway，引用 O'Gara et al., JACC, Feb 2026，doi: 10.1016/j.jacc.2025.07.002。CIED = cardiac implantable electronic device。

### 研究方法和终点

研究设计为 single-center retrospective cohort study，时间为 2021-2025 年。研究对象为因 severe TR 接受 T-TEER 或 TTVR 的患者。30 天终点包括 NYHA functional class 变化和 TR severity 变化。

重新分类规则为：30 天术后 improved 定义为 change < 0，not improved 定义为 change 0。统计分析使用 Fisher's exact test 比较两组。队列共 31 例，其中 T-TEER n=22，TTVR n=9。

### 纳入和排除标准

纳入标准包括：干预时年龄 18-90 岁；干预前 60 天内经超声证实有 symptomatic severe、massive 或 torrential TR；术后 30 天有随访超声。排除标准包括：操作前既往 tricuspid valve intervention；concomitant valve intervention；术后 <30 天死亡；pulmonic valve 存在任何跨瓣压差。页脚强调需要 paired pre- and 30-day post-procedure echocardiograms 才能纳入。

### 主要结果：30 天改善

TTVR 组所有患者在 NYHA 和 TR 两个终点均改善，T-TEER 组改善比例较低。

柱状图纵轴为 Patients Improved (%)，范围 0-100。NYHA Class Improved 方面，T-TEER 为 68.2%，TTVR 为 100.0%，p = 0.077。TR Severity Improved 方面，T-TEER 为 86.4%，TTVR 为 100.0%，p = 0.54。

| 30 天结局 | T-TEER (n=22) | TTVR (n=9) | p-value |
| --- | --- | --- | --- |
| NYHA class improved | 15/22 (68.2%) | 9/9 (100%) | 0.077 |
| No symptomatic improvement | 7/22 (31.8%) | 0/9 (0%) | 未给出 |
| TR severity improved | 19/22 (86.4%) | 9/9 (100%) | 0.54 |

该页明确写道：没有任何 TTVR 患者符合任一终点的 non-improvement 标准。统计方法为 Fisher's exact test，样本量为 T-TEER n=22，TTVR n=9。

### TTVR 后改善模式一致

TTVR 组 100% NYHA 改善和 100% TR reduction，T-TEER 组分别为 68.2% 与 86.4%。

该页标题为“A Uniform Pattern After TTVR”。T-TEER 框中显示 68.2%，对应 NYHA improvement，并同时标注 86.4% TR reduction。TTVR 框中显示 100%，对应 NYHA improvement 和 100% TR reduction。文字说明每位 TTVR 患者两个终点均改善，而 T-TEER 后反应较不一致。页脚为 30-day outcomes，T-TEER n=22，TTVR n=9。

### RV-PA coupling 伴随研究

伴随超声研究显示 TTVR 后 preserved coupling 与 recoupling 比例更高。

一项平行超声心动图研究评估 T-TEER 与 TTVR 后 RV-pulmonary arterial (RV-PA) coupling 轨迹，指标包括 TAPSE/PASP 和 S'/RVSP。随访时 preserved coupling：T-TEER 3/23 (13.0%) vs TTVR 4/9 (44.4%)，OR 9.35，95% CI 0.90-150.97，p = 0.03。

所有患者中的 recoupling：TTVR 3/9 (33.3%) vs T-TEER 1/23 (4.3%)，p <0.01。Persistent uncoupling：TTVR 1/9 (11.1%) vs T-TEER 16/23 (69.6%)。在 initially uncoupled patients 中，recoupling 为 TTVR 3/4 (75.0%) vs T-TEER 1/17 (5.9%)，OR 31.55，95% CI 1.43-2588.95，p = 0.01。

| Persistent Uncoupling | 70% | 11% | <0.01 |

### 结论和局限

研究结论为：TTVR 在 30 天时显示出比 T-TEER 更一致的短期临床改善。TTVR 后 NYHA class improvement 为 100%，TR reduction 为 100%。伴随研究中，TTVR 也更倾向于获得更高的 preserved coupling，并在初始 uncoupled 患者中出现更多 recoupling。

讲者认为，replacement 可能比 repair 提供更可靠的早期临床反应。主要局限为 small single-center cohort 限制统计效能，因此需要在更大队列中验证。
