# 美元与账本：最大化 LAAC/LAAO 手术报销

## 核心要点

* 讲者把 LAAO/LAAC 从单台手术看成项目管理：成功不仅是植入成功、卒中减少和安全性，还包括正向贡献利润、流程效率、合适患者选择、准确编码与账单。
* 一个价值 $15,000 的记录错误案例显示，写“不能服用血液稀释剂”不够，必须说明既往颅内出血、具体抗凝禁忌和共同决策。
* 成熟项目中，最大可避免收入损失通常来自记录缺陷和拒付管理，而不是手术当天本身。
* 贡献利润仪表盘显示，年 LAAO 量 40、90、180+ 的项目，年贡献利润可分别约为 $80K、$495K、$1.6M。
* 收入周期目标包括授权批准率 >95%、拒付率 <5%、申诉成功率 >75%，并持续跟踪 A/R 天数。
* 四个总抓手是完美记录适应证、为早期出院标准化患者路径、追踪每一次拒付、把报销作为质量指标管理。

### 一个 15,000 美元的记录错误

成功 LAAO 手术因记录不充分被拒付，造成 120 天付款延迟。

病例患者为 81 岁男性，有慢性房颤，既往颅内出血，LAAO 手术成功。问题出在记录：术者记录只写了“Patient unable to take blood thinners”，即患者不能服用血液稀释剂。病历中没有记录既往 ICH、没有记录具体抗凝禁忌、没有记录 shared decision making。结果是理赔被拒，必须申诉，付款方要求额外医师记录，付款延迟 120 天。

财务影响包括应收账款 A/R 增加、工作人员时间增加、申诉成本增加和现金流中断。该页给出的教训是：“不能耐受抗凝”这句话不够，必须准确解释为什么不能抗凝。

### 为什么报销重要：它是项目问题

LAAO 已成为成熟的结构性心脏介入。成功不再只由植入成功、卒中减少和安全性结局定义。成功项目还需要正向贡献利润、高效周转、合适患者选择、准确编码与账单。表现最好的项目同时在临床结局和财务结局上表现良好。该页的核心信息是，报销能力不是附属环节，而是 LAAO 项目可持续运行的一部分。

### 成功项目的共同特征

成功项目在四个维度上不同。临床卓越包括选择合适患者和获得良好结局。财务卓越包括完整记录、准确编码和积极拒付管理。运营卓越包括标准化路径和高效出院。战略卓越包括整合房颤项目和结构性心脏项目。该页总结说，报销优化不是单纯账单工作，而是项目设计工作。

### 投票题：最大报销损失来自哪里

投票题问：“你的 LAAO 项目中，哪项造成最大的报销损失？”选项包括 A. prior authorization，B. documentation deficiencies，C. coding errors，D. denials and appeals，E. length of stay。正确答案说明为：在多数成熟项目中，记录缺陷和拒付管理造成最大的可避免收入损失。

### 项目收入流失位置

项目收入流失来自三大类。第一，不充分记录，缺失内容包括长期抗凝禁忌、共同决策、房颤适应证和卒中风险评估。第二，编码错误，包括诊断编码错误、合并症捕捉不完整，以及 observation 与 inpatient designation 使用错误。第三，收入周期延迟，包括 prior authorization 失败、拒付，以及没有继续推进申诉。该页强调，大多数收入流失发生在手术之后，而不是手术过程中。

### 贡献利润仪表盘示例

三个 LAAO 项目的贡献利润仪表盘，显示流程和拒付管理对年度贡献利润的影响。

| 指标 | Program A 底四分位 | Program B 中位数 | Program C 上四分位 |
| --- | --- | --- | --- |
| 年度 LAAO 量 | 40 | 90 | 180+ |
| 同日出院 | 10% | 45% | 75% |
| 拒付率 | 12% | 5% | <2% |
| 申诉成功率 | 50% | 75% | >90% |
| A/R 天数 | 85 | 52 | 35 |
| 平均每例贡献利润 | $2,000 | $5,500 | $9,000 |
| 年度项目贡献利润 | $80K | $495K | $1.6M |

页面底部提问：“哪个项目更可能成功从医院获得更多资源？”按照表格，Program C 同日出院率最高、拒付率最低、申诉成功率最高、A/R 天数最低、每例和年度贡献利润最高，因此更容易争取资源。

### 优化关键：标准化流程

优化关键是把报销嵌入术前 workup。该流程应独立于介入心脏病 IC 或电生理 EP 路径，要求清晰、标准化的文档和路径。项目应提前计划同日或次日出院，并据此安排日程。手术不应被安排，除非 prior authorization 已取得，并且文档已审核且符合标准。讲者提醒这些是择期手术，应训练团队遵循流程并主动管理上游过程。

### 服务地点经济性优化

术前 workup 应采用单一协调路径，包含文档、共同决策门诊和排程。对于同日或次日出院，应进行评估和计划。其好处包括降低成本、改善床位利用率、增加手术容量。许多中心现在已经能够提高同日出院率，运营效率会直接改善利润率。

### 驱动报销的病历记录

每份病历都应以标准格式清楚记录 AF 类型、卒中风险、出血风险、LAAO 理由和共同决策。

每份病历都应以标准化格式清楚记录以下内容。

| 记录类别 | 具体内容 |
| --- | --- |
| AF 严重程度 | Persistent、Paroxysmal、Permanent |
| 卒中风险 | CHA2DS2-VASc 评分 |
| 出血风险 | HAS-BLED 评分 |
| LAAO 理由 | 既往 GI 出血、颅内出血、反复跌倒、抗凝不耐受 |
| 共同决策 | 必需，而且可被审计 |

### 收入周期作战手册：追踪指标

收入周期关键指标和目标值：授权、拒付、申诉成功率和 A/R 天数。

| 指标 | 目标或做法 |
| --- | --- |
| 授权批准率 | 目标 >95% |
| 拒付率 | 目标 <5% |
| 申诉成功率 | 目标 >75% |
| 应收账款天数 | 持续跟踪 |

最佳实践是设置专职 structural heart reimbursement coordinator，即结构性心脏报销协调员。讲者认为这项单一投入常常能产生最高 ROI。

### 拒付管理手册

拒付管理包括五项工作：设置专职协调员；建立快速申诉流程；内部进行医师文档审查；追踪拒付指标；根据拒付原因调整流程，以降低发生率。

### 报销实用要点

要点 1：记录出血事件，而不仅仅记录出血风险。要点 2：建立强制共同决策模板。要点 3：每月审计每一次拒付。要点 4：追踪同日出院率，并创建目标。要点 5：衡量每例贡献利润，而不仅仅衡量手术量。

### 带回家的信息

最大化 LAAO 报销有四个关键点。第一，完美记录适应证。第二，标准化患者路径以实现早期出院。第三，追踪每一次拒付。第四，把报销作为质量指标来管理。讲者总结说，持续优化这四个领域的中心，通常能获得更好的临床和财务表现。
