# Fitting Index：评估 TAVR 后临床结局的重要概念

## 核心要点

* Fitting Index（FI）定义为名义 THV 直径除以 MSCT 测得主动脉瓣环直径，FI 接近 1.0 代表拟合良好。
* LANDMARK 亚组按 FI 分为 <1.00、1.00-1.07、>1.07 三组；总计 563 例中，386 例，即 68.6% 位于良好 FI 区间。
* 1 年死亡和临床有效性在三组间无显著差异：死亡为 7.0%、6.3%、4.4%。
* 瓣膜相关结局与 FI 相关：FI >1.07 组 1 年无中度及以上 PVL、无中度及以上总 AR；FI 1.00-1.07 组较 FI <1.00 组 PVL/AR 更低。
* Myval 系列的中间尺寸可能改善瓣环拟合，降低 BVD 和 BVF，但长期临床意义仍需更长随访验证。

### 背景：为什么 THV 尺寸重要

准确 THV 尺寸选择有助于稳定锚定、有效封闭和减少 PVL、移位等并发症。

准确的经导管心脏瓣膜 THV 尺寸选择与最佳临床结局相关。合适尺寸有助于安全瓣膜植入，包括稳定锚定和有效封闭；改善结局，包括更好血流动力学和增强瓣膜耐久性；减少并发症，包括更少瓣周漏和较低瓣膜移位风险。尺寸不当的影响包括：偏小导致 PVL、瓣膜移位、器械栓塞和早期瓣膜失效；偏大导致瓣环破裂、冠脉阻塞、传导障碍和起搏器需求。引用 Latib 2019、Dowling 2019、Zoghbi 2019 和 Vahanian 2022。

### 研究方法：LANDMARK 亚组

LANDMARK 试验 1:1 随机，亚组分析以 FI 评估球囊扩张瓣膜拟合。

本研究为 LANDMARK 试验亚组分析，患者 1:1 随机，随机总数 n=768。Myval THV 系列随机 n=384，按 as-treated 统计，Myval n=336，Myval Octacor n=32。Sapien THV 系列随机 n=192，按 as-treated 统计，S3 n=108，S3 Ultra n=87。两组均为球囊扩张瓣膜。FI 定义为名义 THV 直径除以 MSCT 测得主动脉瓣环直径，FI 约 1.0 表示良好拟合。随访时间为 1 年。结局包括死亡、临床有效性、中度及以上 PVL 和总 AR、生物瓣功能障碍 BVD、生物瓣失效 BVF、永久起搏器植入 PPI。引用 Baumbach 等 Lancet 2024 和 Serruys 等 JACC 2026。

### 主要结论

FI 是 TAVR 后临床结局和瓣膜表现的重要决定因素。

FI 是 TAVR 后临床结局和瓣膜性能的重要决定因素。FI=1.00-1.07 和 FI>1.07 组显示较低的中度及以上 PVL、较低的中度及以上总 AR、较少 BVD 和 BVF，并有较好的 1 年瓣膜相关结局。Myval THV 系列中间尺寸的可用性可能有助于改善瓣环拟合和优化程序结局。合适 THV 尺寸与最佳瓣环拟合可能改善瓣膜耐久性并减少 TAVR 后并发症。仍需长期随访来判断达到最佳 FI 的临床意义。缩写包括 AR 主动脉反流、BVD 生物瓣功能障碍、BVF 生物瓣失效、FI 拟合指数、PVR 瓣周反流、TAVR 经导管主动脉瓣置换、THV 经导管心脏瓣膜。

### 研究理由与目的

研究问题是 FI 是否影响球囊扩张 THV 平台的临床结局。

研究理由为：FI 决定 THV 在主动脉瓣环中的合适程度，并可能影响临床结局；目前评估 FI 在球囊扩张 THV 平台中临床影响的数据仍有限。研究目的为：LANDMARK 试验亚组分析使用 FI 评估球囊扩张 Myval 和 Sapien THV 系列的尺寸精准度对结局的影响。引用 Royen 等 EuroIntervention 2025。

### VARC-3 定义：BVD 分期

VARC-3 将 BVD 分为形态学瓣膜退化、中度 HVD 和重度 HVD。

| 分期 | 定义 |
| --- | --- |
| 1 期：形态学瓣膜退化 | 存在结构性瓣膜退化、非结构性瓣膜功能障碍，除 PVL 或 PPM 外、血栓或心内膜炎证据，但无显著血流动力学变化。 |
| 2 期：中度血流动力学瓣膜退化 | 平均跨瓣压差增加 >10 mmHg，并导致平均压差 >20 mmHg‡，同时 EOA 下降 ≥0.3 cm² 或 ≥25%，和/或 Doppler velocity index 下降 >0.1 或 ≥20%，与术后 1-3 个月超声相比；或新发或增加 ≥1 级§ 瓣内 AR，并导致中度以上 AR。 |
| 3 期：重度血流动力学瓣膜退化 | 平均跨瓣压差增加 >20 mmHg，并导致平均压差 >30 mmHg‡，同时 EOA 下降 ≥0.6 cm² 或 ≥50%，和/或 Doppler velocity index 下降 >0.2 或 ≥40%，与术后 1-3 个月超声相比；或新发或增加 ≥2 级§ 瓣内 AR，并导致重度 AR。 |

注释：‡ 该血流动力学功能障碍标准假设正常流量；§ 该标准按 3 级分级体系评估。引用 Philippe Généreux 等 European Heart Journal 2021。

### VARC-3 定义：BVD 类型

BVD 类型包括 SVD、NSVD、血栓和心内膜炎。

| 类型 | 定义 |
| --- | --- |
| SVD，结构性瓣膜退化 | 假体瓣内在永久性改变，包括磨损、瓣叶破裂、瓣叶脱垂、瓣叶纤维化和/或钙化，或支架断裂/变形。 |
| NSVD，非结构性瓣膜功能障碍 | 非假体瓣内在异常导致瓣膜功能障碍。例子包括残余瓣内或瓣周主动脉反流；瓣叶被 pannus、组织或缝线卡住；位置或尺寸不合适；无支架假体或保瓣手术后主动脉根部扩张；PPM；栓塞。 |
| 血栓 | 亚临床血栓指影像发现低衰减瓣叶增厚 HALT 或瓣叶运动减少 RLM，但无或仅轻微血流动力学变化且无症状/后遗症。临床显著血栓包括：有血栓栓塞事件，如卒中、TIA、视网膜闭塞、系统性栓塞，或瓣膜狭窄/返流加重导致心衰、晕厥，并且存在 HVD 2-3 期或影像确认 HALT/RLM；或无临床后果但同时有 HVD 3 期和影像确认 HALT/RLM。 |
| 心内膜炎 | 至少符合一项标准：满足 Duke 心内膜炎标准；再次手术时组织学或微生物学证实感染相关脓肿、脓液或赘生物；尸检证实脓肿、脓液或赘生物。 |

### VARC-3 定义：BVF 分期

BVF 1 期为有临床表达的 BVD 或不可逆 3 期 HVD，2 期为再手术/再干预，3 期为瓣膜相关死亡。

| BVF 分期 | 定义 |
| --- | --- |
| 1 期 | 任何与临床表达标准相关的 BVD，包括新发或加重症状、左室扩张/肥厚/功能障碍或肺动脉高压，或不可逆 3 期血流动力学瓣膜退化 HVD。 |
| 2 期 | 主动脉瓣再次手术或再次介入。 |
| 3 期 | 瓣膜相关死亡。 |
