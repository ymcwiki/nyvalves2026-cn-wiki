# 失败 THV 合并冠脉错位中的 ShortCut 双侧瓣叶改造

## 核心要点

* 病例为 85 岁女性，2017 年曾植入 23 mm S3，因生物瓣严重 AS 退化，计划 20 mm S3 TAV-in-TAV。
* 术前 CT 显示左右冠脉均有阻塞风险：LC VTC 2.0 mm、VTS 0.0 mm；RC VTC 2.8 mm、VTS 1.1 mm。
* 右冠风险更突出且存在偏心，RCA 偏心角度标注为 54°，因此操作中进行冠脉保护，并以导引系统作为偏心冠脉定位标志。
* ShortCut 完成双瓣叶切开，用时 16 分钟，冠脉保护没有妨碍 ShortCut 操作。
* 随后植入 20 mm S3UR THV；术后冠脉造影显示有保护和撤除保护后血流均良好，RCC 保持通畅。

### 结论

ShortCut 在高风险解剖中实现了安全、流程化的双瓣叶切开，用时 16 分钟；该解剖中 RCA 偏心为 54°。冠脉保护不会干扰 ShortCut 操作。一旦确认血流通畅，冠脉保护可顺利撤除；对于偏心冠脉，可将导引系统作为定位标志。随着 valve-in-valve 操作预期增加，ShortCut 机械瓣叶改造为有冠脉阻塞风险的复杂 TAVR 患者提供了可预测、可重复的治疗选择。
