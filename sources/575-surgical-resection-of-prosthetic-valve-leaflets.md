# 直视下外科切除人工瓣叶（SURPLUS）用于高风险 redo TAVR

## 核心要点

* 患者为80 岁男性，2021 年因症状性二叶式重度 AS 合并中度反流接受 29 mm CoreValve Evolut TAVR，术前 BAV 20 mm，术后 BAV 22 mm。
* 随后出现重度生物瓣 AS 和中重度至大量 PVL，超声显示峰速4.2 m/s、平均压差50 mmHg、DVI 0.19、连续方程 VTI 面积0.49 cm²。
* TEE 和 4D CT 未见 HALT，抗凝试验未改善跨瓣压差；问题更指向钙化瓣叶、瓣架和 redo TAVR 解剖风险。
* CT 风险分析显示 redo TAVR 的冠脉距离很小，例如 RCA-VTC 2.5 mm、RCC-VTSTJ 2.3 mm，DASI 评审认为常规经皮 redoTAV 缓解策略不可靠。
* 最终采用 SURPLUS 混合策略并行 rSVG-RCA bypass；PVL 仍存在但改善，保留未来 TAVR 选项，患者无症状且恢复良好。

### 病史和基础资料

患者为80 岁男性。既往病史包括青光眼、良性前列腺增生（BPH）、高血压、血脂异常、2 型糖尿病、左视网膜静脉阻塞、重度症状性二叶式主动脉瓣狭窄和中度主动脉反流。2021 年治疗经过为：术前以 20 mm 球囊预扩张（pre-BAV），随后植入 29 mm CoreValve Evolut，术后使用 22 mm true balloon 再扩张。

此后出现重度生物人工主动脉瓣狭窄，并伴中重度至大量瓣周漏，导致进行性失代偿性心力衰竭，发生在初次 TAVR 后约5 年。超声显示中度主动脉瓣关闭不全；重度主动脉瓣狭窄，连续方程面积0.8 cm²，平均压差44 mmHg，峰速4.2 m/s。

### 可选方案和陷阱

可选方案包括：对右冠状动脉进行电外科处理；考虑低位（inflow）着陆以理论上封堵瓣周漏；为处理短期和长期瓣叶悬垂问题而高位着陆；外科取出 TAVI 瓣并置换为外科生物瓣；采用 SURPLUS 这类混合策略。

### DASI 报告和 proctor 评审

DASI/proctor 认为这是一个非常困难的场景：这是狭窄的 Evolut，瓣叶在 node 5 和 node 6 处有严重钙化，因此需要在 node 5 或 node 6 高位植入 S3。这样植入时存在 RCA 闭塞风险，而 LCA 无论如何都被认为是安全的。

评审指出连合方向不理想，VTA/VTC 非常小，因此没有可靠的风险缓解策略，snorkeling 看起来也不可靠。若在 node 4 低位植入 S3，可能只会通过 S3 递送球囊扩张撞击钙化瓣叶而短暂改善主动脉瓣压差；但 Evolut 的钙化瓣叶仍会悬垂在位于 node 4 的 S3 上方，中期可能再次导致更高压差。

评审还强调，在 node 4 部署并不一定能降低 RCA 闭塞风险。原因有两点：第一，Evolut 瓣叶可能被顶起并堵塞开放网孔；第二，Evolut 支架框架至少会扩张 1 mm，在 Evolut 连合 tab 所在位置可能把 VTA 降至0-1 mm，而该处是封闭单元而非开放单元。

关于 PVL，评审认为其可能改善，也可能不改善，且很难预测，与 S3 位置并不完全相关。node 3 或 node 4 的较低位置可能更有机会改善 PVL，但机会仍不理想。评审明确表示，不会为了增加修复 PVL 的机会而牺牲对 AS 的充分解决。因此，除非 RCA 被认为不重要或已被旁路，否则本例没有合理的经皮 redoTAV 方案，应强烈重新考虑外科。

### 混合策略：SURPLUS

SURPLUS 混合手术影像：直视下处理人工瓣叶，并展示切除组织。

本页展示 Hybrid（SURPLUS）过程。左图和中图为开放手术视野下器械进入主动脉根部/瓣膜区域，对既有人工瓣叶进行直视下处理和切除；右图显示取出的钙化瓣叶/组织标本。该策略的核心是通过外科直视下切除悬垂或钙化的人工瓣叶，降低未来或同步经导管策略中的冠脉阻塞和瓣叶遮挡风险。

### 总结

最终采用 SURPLUS，并行 rSVG-RCA bypass。PVL 仍存在，但已改善。未来 TAVR 选项得到保留。患者目前无症状，状态良好。
