# 突出至主动脉的冠脉支架患者采用改良序贯球囊扩张成功 TAVR

## 核心要点

* 病例为 80 岁女性，身高 154 cm、体重 74.8 kg、BMI 30.0、BSA 1.69，主诉 DOE（NYHA III-IV）和胸痛，合并 HTN、DM。
* 重度 AS 参数为 Vmax 4.9 m/s、平均压差 58 mmHg、AVA 0.7 cm²；风险评分为 STS 8.42%、Euroscore I 8.0%、Euroscore II 2.81%。
* 既往 RCA PCI 后，使用 Selethru NC 4.0 × 10辅助球囊扩张导致 RCA 支架 elongation，并有支架突出至主动脉的问题，形成 TAVR 时冠脉阻塞和支架变形风险。
* CT 显示瓣环面积 435.4 mm²、面积推导直径 23.5 mm，瓣叶钙化总量 590 mm²（HU 600）；LCA 高度 8.2 mm，STJ 18.5 mm，RCA 高度 14.0 mm，RCA 支架突出约 5.0 mm。
* 团队选择 SAPIEN 3 系列 23 mm瓣膜，并借鉴 stepwise/two-step inflation 与 sequential ballooning 思路，采用 modified sequential ballooning 进行 BEV 植入，以保护突出冠脉支架和冠脉血流。
* 术中进行了冠脉保护/导丝定位、NC 4.0 × 15球囊、Z-med 18 mm预扩张、改良序贯球囊瓣膜植入和术后再次 NC 球囊处理，最终冠脉造影和主动脉造影显示结果可接受。

### 备选介入方案

在处理突出至主动脉的冠脉支架时，团队考虑了若干非外科替代方案：Guide Extension Catheter（GEC）、buddy wire，以及 Rotablation。页面配有导引延长导管、buddy wire 和旋磨系统示意图，核心问题是如何在 TAVR 中维持冠脉通路、防止支架形变并避免冠脉阻塞。

### 备选外科或混合策略

团队还考虑了 CABG + AVR，以及 CABG + TAVR 两种外科或混合策略。页面用 CABG 手术示意、机械瓣/生物瓣图像和经导管主动脉瓣植入示意图表达这些备选方案。结合患者年龄、风险评分和既有冠脉支架问题，最终策略转向经导管治疗中的冠脉保护和改良球囊释放。

### Stepwise 或 sequential ballooning 的文献依据

两篇文献示意分步扩张或序贯球囊策略，用于控制瓣膜释放和保护关键结构。

本页引入 stepwise（two-step）inflation 或 sequential ballooning 概念。左侧引用 Freek Nijhoff 等，Eurointervention 2013;9(5):555-63，展示球囊扩张和瓣膜逐步展开的透视序列。右侧引用 Takuma Ohi 等，Eur Heart J Case Rep 2023;7(12):1-6：第 1步使用 4.0-10 mm non-compliant balloon，同时对 23 mm SAPIEN 3 deployment balloon 进行 2 mL underfilling；第 2步再进行 1 mL more overfilling，并将 deployment balloon 向 LVOT 推进。该思路被用于本病例的 modified sequential ballooning。

### 结论：突出冠脉支架患者可考虑改良序贯球囊 BEV 策略

TAVR 后冠脉阻塞仍是最严重且危及生命的并发症之一。当对冠脉支架突出至主动脉的患者进行 TAVR 时，存在支架变形和潜在冠脉阻塞风险。在这类病例中，BEV 植入时采用 modified sequential ballooning 可以成为预防冠脉阻塞的有效策略。
