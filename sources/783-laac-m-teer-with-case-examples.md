# 左心耳封堵联合二尖瓣经导管缘对缘修复：含病例示例

## 核心要点

* 接受 M-TEER 的患者中，超过 50% 合并房颤；这类患者通常年龄更大、合并症更多、出血风险更高，因此 LAAC 可能适合与 M-TEER 同期或序贯完成。
* 病例为 73 岁女性，气短合并房颤且复律失败，肥胖，使用 apixaban 时出现瘀斑，手术风险中等。
* TEE 显示左心耳大且小梁丰富，无血栓。患者先用单枚 NTW G4 MitraClip 成功行经导管二尖瓣修复。
* 二尖瓣反流纠正后，患者成功行 AF 消融，并计划 LAAC；置入 40 mm 左心耳封堵器，术后 6 周停用抗凝。
* 随访显示患者维持窦律、无心衰；末次超声为微量 MR，二尖瓣平均压差 4 mmHg。
* WATCH-TEER 研究页总结了 n=24 例同期 LAAC 与 M-TEER：器械成功 100%，总操作时间 103.6 ± 33.7 分钟，残余 MR ≤2+ 为 96%，45 天主要复合终点 21% (n=5)。

### 背景：为什么考虑 LAAC 与 M-TEER 合并

超过 50% 接受二尖瓣经导管缘对缘修复 M-TEER 的患者合并房颤。这些患者年龄较大，合并多种共病，且出血风险高。对这类患者，同期或序贯左心耳封堵 LAAC 会有帮助。作者还提出，需要建立单独的 Medicare DRG，使这些操作能够同期完成；该页最后一行原文显示到“concomitantly du”，句尾未完整显示。

### MR 纠正后：AF 消融和计划 LAAC

二尖瓣反流纠正后，患者成功完成 AF 消融，并计划进行 LAAC。放置 40 mm 左心耳封堵器。患者在术后 6 周停用抗凝药。随访中患者维持窦性心律，没有心力衰竭。末次随访超声显示二尖瓣反流为 trace，二尖瓣平均压差为 4 mmHg。

### M-TEER 同期 LAAC：优点与缺点

同期 M-TEER 和 LAAC 的 advantages 与 disadvantages 对照表。

| 优点 | 缺点 |
| --- | --- |
| ACC 指南建议在二尖瓣外科手术中关闭 LAA。 | 成本增加。 |
| 两种操作都需要后方经房间隔入路。 | 操作时间增加。 |
| Clip 后左心耳内常出现血液淤滞。 | 如果 M-TEER 导致中度二尖瓣狭窄，可能增加器械相关血栓风险。 |
| 符合患者偏好。 |  |
| Clip 操作不需要造影剂，因此急性肾衰风险低。 |  |
| 如果 LAAC 器械栓塞，Clip 可防止器械进入心室。 |  |

### 需要新的 MS-DRG

作者呼吁为 M-TEER 同期 LAAC 建立新的 MS-DRG。理由包括：许多成功 M-TEER 后的患者会在以后再行 LAAC；分两次操作会增加患者风险；分两次完成会增加总体成本。同期操作可由同一团队完成，即介入心脏病医生和影像医生；两者都需要麻醉和 TEE 引导；无需额外造影剂，因为 M-TEER 不需要造影剂。

### 最后观点

作者认为，当 M-TEER 和 LAAC 均有适应证时，应考虑同期完成。医保编码方面需要一个合并 MS-DRG code。可以设计单臂研究，在合适患者人群中证明这种联合操作的安全性。
