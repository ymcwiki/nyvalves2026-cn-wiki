# PASCAL 经导管二尖瓣缘对缘修复中单叶装置附着的补救处理

## 核心要点

* SLDA 是 mTEER 少见但有临床意义的并发症，比较研究中发生率约 1-2%，PASCAL 与 MitraClip 系统的 SLDA 率相近。
* 病例为 82 岁男性，NYHA III，LVEF 60-65%，退行性重度原发 MR，病变包括 P2 连枷和 P1-P2 裂隙。
* 初次 P10 处理后仍有显著残余 MR，策略改为部署 2 枚 PASCAL ACE，第二枚外侧 ACE 后出现高度活动的 SLDA。
* 补救先计划第 3 枚 ACE，位于第二枚装置外侧、P1/A2 区，用来锁定 SLDA；但第二枚 ACE 在第 3 枚 ACE 存在下仍不稳定。
* 最终使用经房间隔辅助支撑：将 AVP2 16 mm 血管塞置于可调弯 Baylis 鞘内，下压第二枚 ACE 并稳定器械，随后完成第 3 枚夹合器成功释放。

### SLDA 背景

单叶装置附着，single leaflet device attachment，简称 SLDA，仍是 mTEER 术中少见但临床意义明确的并发症。比较研究中 SLDA 发生率约 1-2%。PASCAL 与 MitraClip 系统的 SLDA 发生率相近。早期识别和补救策略是关键。

### 病例基本资料

患者为 82 岁男性，劳力性呼吸困难加重，功能分级为 NYHA III。TTE/TEE 显示双心室收缩功能正常，LVEF 60-65%；存在重度原发性二尖瓣反流，病因为退行性病变，具体为 P2 连枷以及 P1-P2 裂隙。

### SLDA 伴残余反流

SLDA 的彩色多普勒图像，显示装置旁仍有残余 MR。

### 补救策略：计划第 3 枚 ACE 锁定 SLDA

计划通过第 3 枚 ACE 稳定受损装置，位置设在第二枚外侧、P1/A2 区域，用于锁定 SLDA。

页面题为“SLDA: fighting the Devil”。为稳定受损装置，团队计划植入第 3 枚 ACE。策略要点包括：第 3 枚装置放在第二枚装置外侧；目标解剖位置为 P1/A2；目的为“locking the SLDA”，即通过额外夹合器固定发生 SLDA 的装置。

### 第 3 枚 ACE 后第二枚仍不稳定

第三枚夹合器策略下，第二枚 ACE 仍显示高度活动和不稳定。

页面标题为“3rd Clip strategy: highly mobile SLDA”。左侧透视和右侧三维 TEE 显示第 3 枚装置已参与稳定，但页脚明确写道：2nd ACE device unstable, despite 3rd ACE device。也就是即使有第 3 枚 ACE，第二枚 ACE 仍不稳定，常规额外夹合没有完全解决 SLDA。

### 高度活动 SLDA 的升级清单

该页题为“Highly mobile SLDA”，提出“Fighting the devil, a user guide”。讲者列出四步升级思路：1. Fire，快速起搏；2. ICE，腺苷；3. Faith，呼叫外科；4. Other，寻找其他办法。此处将快速起搏和腺苷作为降低运动和血流影响的稳定化手段，同时保留外科转换作为兜底选项。

### 经房间隔血管塞辅助支撑

将 AVP2 16 mm 置于可调弯 Baylis 鞘内，经房间隔下压第二枚 ACE，以稳定 SLDA。

页面标题为“3rd Clip: Sheath stabilization with Vascular Plug”。具体技术为“Adjunctive transseptal”：使用 AVP2 16 mm 血管塞装置，置于可调弯 Baylis 鞘内，用来下压第二枚 ACE并稳定它。左侧和中间透视图展示鞘管、血管塞和夹合器之间的相对位置；右侧三维 TEE 显示装置在二尖瓣瓣口的空间关系。

### 第 3 枚夹合器成功释放

第 3 枚夹合器成功部署后的彩色多普勒图像，残余反流较前受控。

### 讨论：导管内系统升级可避免外科转换

讨论页右下三维 TEE 标注 AVP2/ACE、Amplatzer Vascular plug II 和 SLDA 的相对位置。

讲者总结：SLDA 可通过有创造性的导管内稳定策略有效处理，而不一定需要外科转换。关键是早期识别并按系统化升级路径处理，包括额外装置植入、快速起搏和腺苷等稳定技术、经房间隔辅助支撑，以在复杂退行性二尖瓣解剖中保留手术成功机会。外科仍是升级路径中的重要选项。右下图标注了 AVP2/ACE、Amplatzer Vascular plug II 和 SLDA 的关系，说明血管塞在补救中承担机械支撑作用。
