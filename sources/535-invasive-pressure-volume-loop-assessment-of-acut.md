# 重度症状性主动脉瓣狭窄 TAVI 后急性双心室血流动力学变化的侵入性压力-容积环评估

## 核心要点

* 研究纳入 12 例重度症状性 AS 患者，在 TAVI 前后用 7F conductance catheter 进行实时左右心室压力-容积环测量。
* TAVI 后 LV 卸载明显：LV ESP、LV dP/dt+、LV PRSW、LV SW、LV PVA 和 LV Ea 均下降，提示机械负荷和能量需求立即降低。
* 尽管 LV 收缩性指标即时下降，LV 动脉-心室耦合改善，LV Ea/Ees 从约 0.65 降至约 0.57。
* RV 也出现急性后负荷下降和 RV-PA coupling 改善，RV Ea 从约 0.48 降至约 0.38，RV Ees/Ea 从约 1.10 升至约 1.27。
* 研究结论是，TAVI 的急性效应不局限于左室，还包括右室后负荷和双心室能量转移效率的改善。

### 研究背景：AS 的瓣膜外心脏损害

重度 AS 造成 LV 压力负荷，并沿左房、肺循环、三尖瓣和右室形成瓣膜外心脏损害谱。

重度主动脉瓣狭窄导致左室压力负荷增加，并可引起一系列瓣膜外重构。TAVI 解除增加的左室后负荷，并促进长期左室、肺循环和右室逆重构。讲者指出，关于 TAVI 急性血流动力学变化的侵入性研究仍然稀少。

| 心脏损害分期 | 定义 |
| --- | --- |
| Stage 0，无心脏损害 | 无瓣膜外心脏损害 |
| Stage 1，LV 心脏损害 | LV mass index 升高，男性 >115 g/m² 或女性 >95 g/m²；E/e’ >14；LVEF <50% |
| Stage 2，LA 或二尖瓣损害 | LA volume index >35 mL/m²；中重度 MR；房颤 |
| Stage 3，肺循环或三尖瓣损害 | 收缩期肺动脉高压 >60 mmHg；中重度 TR |
| Stage 4，RV 损害 | 中重度 RV 功能障碍 |

### 研究目的

研究目标是用金标准侵入性 PV loop 评估重度症状性 AS 患者 TAVI 后的急性双心室反应。

本研究为前瞻性研究，目标是量化重度症状性主动脉瓣狭窄患者在 TAVI 后急性左室和右室反应。方法学上使用金标准侵入性 pressure-volume loop 评估。页面展示 CD Leycom/ConductNT 相关设备和软件界面。

### 研究人群与纳排标准

研究纳入 12 例重度症状性 AS 患者，均于 2023-2025 年接受择期 TAVI。重度 AS 定义包括 AVmax ≥4 m/s、跨主动脉瓣平均压差 ≥40 mmHg、AVA <1 cm²，或 indexed AVA ≤0.6 cm²/m²。

排除标准包括束支传导阻滞；侵入性评估期间出现节律问题，例如频发室早；除 AS 外存在至少中度瓣膜病；既往瓣膜手术；植入永久起搏器；肺病史；LVEF <50%；显著 CAD；既往 MI；按 VARC-3 定义的主要术后并发症；以及经胸超声声窗差。

### TAVI 流程与 PV loop 采集

在清醒镇静、镇静水平恒定的 TAVI 过程中，使用 7F conductance catheter 进行左右心室 PV loop 采集。

研究同时包含球囊扩张瓣和自膨瓣。TAVI 在清醒镇静下进行，并保持镇静水平恒定。实时 PV loop 采集使用 7F conductance catheter，系统为 CD Leycom。TAVI 前后分别校准压力和 LV/RV 容积。

| 步骤 | 内容 |
| --- | --- |
| 1 | TAVI 前 LV/RV 容积校准 |
| 2 | TAVI 前右心导管 RHC |
| 3 | TAVI 前 RV PV loop 采集 |
| 4 | TAVI 前 LV PV loop 采集 |
| 5 | TAVI 瓣膜释放 |
| 6 | 重复 LV/RV 容积校准 |
| 7 | TAVI 后 LV PV loop 采集 |
| 8 | TAVI 后 RV PV loop 采集 |

### PV loop 分析指标

PV loop 使用 CD Leycom 的 ConductNT 专用软件分析，分析者 blinded。计算指标包括四类。

| 类别 | 指标 |
| --- | --- |
| 收缩功能 | Ees、dP/dt+、PRSW |
| 舒张功能 | Tau、dP/dt-、LV 的 V30 或 RV 的 V10 |
| 能量学 | SW、PVA、SW/PVA |
| 心室-动脉耦合 | RV Ees/Ea、LV Ea/Ees |

### 结论与局限

研究结论为，TAVI 引起的 LV 卸载在急性期改善心室-动脉耦合，尽管 LV 收缩性指标立即下降。卸载效应不局限于左室，RV 后负荷下降与 RV-PA coupling 增强相关。双心室前向血流动力学改善，即全局心脏能量转移效率增加，与总机械功和心肌代谢需求下降相关。

需要注意的局限包括样本量仅 12 例，研究关注 TAVI 前后即刻变化，不能直接替代长期随访结论；同时纳入标准排除了 LVEF <50%、显著 CAD、既往 MI、较重要并发症和超声窗差等患者，因此结果主要适用于较稳定、保留 EF 的择期 TAVI 人群。
