# Redo TAVR 中瓣架结构的影响（高架与短架）

## 核心要点

* Redo TAVR 规划首先要识别原 TAV 的瓣架设计和交界柱位置，因为这决定新裙部平面 NSP 的判定。
* 第二枚 TAV 的尺寸选择取决于原瓣类型、目标 landing zone、组合方式以及原瓣失效机制；不能只按标称瓣径处理。
* 短架置入高架失败 Evolut 时，如 BEV 在 landing zone 有超过 1 个 inflow node 偏小，或失效模式为 AR/PVL，可能需要上调 BEV 尺寸以获得锚定。
* LAO/CRA frame alignment 透视可去除原 TAV 视差，辅助确认 CT 分析、识别冠脉风险区 CRZ，并判断 NSP/CRP 与左冠关系。
* 冠脉风险规划需把最小 VTA 距离分层为 >4 mm、2-4 mm、<2 mm，高风险时考虑冠脉保护、瓣叶改造、替代第二瓣策略或外科。

### 第二枚 TAV 尺寸取决于原瓣类型和 landing zone

第二枚 TAV 的测量区随原瓣平台改变，短架、Lotus、Evolut、Navitor 和 ACURATE 的算法不同。

本页同样引用 Pop A、Tang GHL 等 JACC Intv 2026，主题为“TAV #2 sizing varies by index TAV type and landing zone”。一般原则是先用 center-to-center tracing 测量支架框架面积。不同平台的测量方式如下：

| 原瓣平台 | 第二瓣 sizing 方法 |
| --- | --- |
| 通用测量 | 按中心到中心描迹测量 stent frame area。 |
| Sapien 3 | 平均 inflow、mid-frame、outflow 三处面积。 |
| Lotus | 平均目标 implant zone 内各面积。 |
| Evolut | 平均 implant zone 内 4 个 node 的面积，即 NSP 和其下方 3 个 node。 |
| Portico/Navitor | 平均 implant zone 内 3 个 node 的面积，即 NSP 和其下方 2 个 node。 |
| ACURATE neo/neo2 | 平均目标 implant zone 的面积。 |

### 第二瓣植入位置、尺寸和锚定注意事项

失败 Evolut 中第二瓣位置取决于 AS 与 AR 失效机制，红框强调目标 landing zone 和下方 inflow 区域。

本页标题为“CAUTION re: TAV #2 implant position, sizing & anchoring”。植入位置取决于 AS 与 AR。若失效机制为 AS，在 SIT 中可接受一定 leaflet overhang，但在失败 Evolut 中不要低于 Node 4。若失效机制为 AR，可以采用更低植入和 leaflet overhang，但必须避免 LV embolization。

尺寸和锚定方面，失败 Evolut 需要在目标 landing zone 处用所选 BEV 检查底部 2 个 node 的百分比 oversizing。若 landing zone 有超过 1 个 inflow node 对 BEV 来说偏小，因 Evolut inflow 向外张开，BEV 若落得更深，可能需要 UPSIZE 以获得锚定。若 AR/PVL 是失效模式，也应考虑上调 BEV 尺寸。

### SIS 与 SIT TAV-in-TAV 的第二瓣尺寸算法

第二枚 TAV sizing algorithm：按 short-in-short 与 short-in-tall 组合使用 downsize 或 upsize 修正。

本页给出“Second TAV Sizing Algorithm for SIS and SIT TAV-in-TAV”的组合特异性修正。Short-in-short 情境下，如原 TAV 因 AS 或 underexpanded 失败、第二 TAV 受限或担心 HALT、原 TAV 扩张可能增加冠脉风险，可考虑 DOWNSIZE；如原 TAV undersized 且 PVL 需要更大径向力以减少 PVL，可考虑 UPSIZE。

Short-in-tall 情境下，DOWNSIZE 因锚定顾虑而不推荐。UPSIZE 条件分为两类：若 Evolut 作为原 TAV，条件包括 AR/PVL 失败、TAV #2 landing zone 有超过 1 个 inflow node 对 BEV 偏小（因 Evolut frame 向外张开），以及较深 TAV #2 植入存在 LV migration 风险；若 Navitor 作为原 TAV，AR 或 PVL 失败也支持考虑 UPSIZE。

### 透视分析的价值

本页说明“Value of Fluoro Analysis”。透视的优势包括：分辨率高于 CT，且没有伪影；可在手术中直接指导目标 outflow 植入位置与左冠的相对关系；在 LAO-CRA frame alignment 视角中去除原 TAV parallax，理想情况下使 cells 重叠，从而更容易观察 TAV #2 与原 TAV 的相对位置；主动脉造影可评价 NSP 与 CRP 的相对关系。

### Take Home Messages

总结信息共有 4 点。第一，理解原 TAV 设计和交界柱位置是确定 NSP 的关键。第二，第二枚 TAV 的 sizing 随组合方式、目标 landing zone 和原 TAV 失效机制而变。第三，在失败 Evolut 的 SIT 中，如 BEV 在 landing zone 有超过 1 个 inflow node 偏小，或失效模式为 AR/PVL，可能需要 UPSIZE。第四，LAO/CRA frame alignment 透视有助于确认 CT 分析、识别 coronary risk zone（CRZ）和 NSP/CRP 与 LCA 的关系，并帮助定位第二枚 TAV。

### 短架/高架 TAV-in-TAV 最佳实践流程

左侧为 short-in-short 与 short-in-tall，右侧为 tall-in-short 与 tall-in-tall 的最佳实践流程。

本页整合两套 best-practice recommendations。左侧适用于 short-in-short 和 short-in-tall TAV-in-TAV。第 1 步为 CT planning for coronary risk assessment：定义 CRP 与计划 NSP 的关系，在 CRZ 内测量最小 VTA，风险分层为 >4 mm、2-4 mm、<2 mm。如冠脉风险高，附加选择包括 coronary protection、leaflet modification、alternative TAV#2 strategy 和 surgery。第 2 步为 TAV #2 device selection and sizing：CT-guided TAV #2 sizing 需考虑原 TAV 类型/尺寸、体内 landing zone 尺寸、原 TAV 失效机制（AS、AR/PVL、AS/AR）、若有 pre-index CT、以及厂家 sizing 建议；随后判断 upsize 或 downsize 是否有益，若是则应用组合特异性修正，若否则选择 CT 推荐尺寸。第 3 步为预扩策略：狭窄或 mixed AS/AR 或 PVL 可考虑预扩，pure AR 通常不需要常规预扩；球囊 sizing 的目标为 frame optimization，若适合则 CT-guided balloon sizing，若不适合则保守或小球囊。第 4 步为 TAV #2 deployment：使用 frame alignment view，保持 TAV #2 与原 TAV 同轴，按原 TAV 内的 outflow target 定位 TAV #2，采用 slow 2-step BEV inflation，并在释放过程中调整 TAV #2 位置。第 5 步为后扩策略：目标包括优化 TAV #2 expansion/apposition、改善 anchoring/hemodynamics、纠正 canting/asymmetry、减少 inter-frame leak/PVL；球囊选择从 double tap/volume add（TAV #2=BEV）开始，若在 volume-add 后仍有 persistent constraint 或 migration concern，则转向 CT-guided NC balloon。

右侧适用于 tall-in-short 和 tall-in-tall TAV-in-TAV。第 1 步同样为 CT 冠脉风险评估：定义 CRP 与预测 NSP 的关系，在 CRZ 内测量最小 VTA；如冠脉风险高，需要确认可行性，包括 VTC/VTA/VTSTJ、sinus/STJ constraints 和 commissural alignment，并考虑 coronary protection、leaflet modification、alternative TAV#2 strategy 或 surgery。第 2 步为 CT-guided tall TAV #2 sizing，考虑原 TAV 类型/尺寸、体内 landing zone 尺寸、失效机制、coronary risk/future access、pre-index CT 和厂家建议；再判断 downsize 是否有益。第 3 步为预扩：狭窄、mixed AS-AR 或 PVL 可考虑预扩，pure AR 通常不需；球囊 sizing 若可行则 CT-guided balloon sizing，若否则保守或小球囊，并避免超出 native/root limits 的 oversizing。第 4 步为释放：使用 frame alignment view，保持同轴，适当时 target inflow-to-inflow，计划 commissural/cell alignment，释放前评估冠脉血流，需要时使用 controlled pacing。第 5 步为后扩：目标仍为优化扩张/贴壁、改善 hemodynamics、纠正 canting/asymmetry、减少 inter-frame leak/PVL；球囊选择包括 CT-guided NC balloon，结合厂家 sizing 与体内尺寸，且尽量避免在 chimney stent 后后扩。
