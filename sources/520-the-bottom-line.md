# 底线观点：CAD 与 AS

## 核心要点

* CAD 在 AS 患者中很常见，AS 患者可以安全接受 PCI。
* 是否存在 CAD、是否有既往血运重建史，并不与 TAVR 更差围手术期结局相关。
* CAD 与 TAVR 后较差长期结局相关，但常规血运重建可能并不改变 TAVR 后预后。
* 讲者建议：CAD 是否血运重建应基于 CAD 本身适应证，而不是因为患者合并 AS 就常规处理。
* TAVR 前血运重建主要考虑 ACS，或计划使用可能导致冠脉再进入困难的 SEV；TAVR 后血运重建主要考虑症状持续且病变严重度足够高。

### 题目与讲者

题目为“The Bottom Line: CAD and AS”。讲者 Samir Kapadia, MD，Cleveland Clinic 医学教授、介入心脏病学 Section Head、Sones Cardiac Catheterization Laboratory 主任。

### Bottom Line：CAD、PCI 与 TAVR 结局

CAD 在 AS 患者中常见。PCI 可以在 AS 患者中安全实施。CAD 是否存在以及既往血运重建史，并不与 TAVR 更差的操作相关结局相关。CAD 与 TAVR 后更差的长期结局相关。常规血运重建可能并不会改变 TAVR 后预后。

### 讲者建议：按 CAD 适应证决定血运重建时机

讲者建议，只有当 CAD 本身有适应证时才进行血运重建，与是否合并 AS 无关。

TAVR 前血运重建的情形包括：ACS，其中列出 USA、NSTEMI、STEMI；以及计划 SEV 且未来冠脉通路可能困难的情形。

TAVR 后血运重建的情形包括：症状持续，例如 CP 或 SOB，且病变严重度大于年龄，原文为 lesion severity > age。
