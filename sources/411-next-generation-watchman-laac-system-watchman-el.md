# 新一代 Watchman 左心耳封堵系统（Watchman Elite）

## 核心要点

* Watchman 系列从早期概念验证、FLX 安全性改进、FLX Pro 简化术后抗栓方案，继续演进到 Watchman Elite，目标集中在稳定性和封堵密封。
* 45 天 TEE 核心实验室判定显示，Watchman FLX Pro 和 Watchman FLX 的无漏率分别为 82.7% 与 82.8%，均高于 Amulet 的 63% 和早期 Watchman 的 46%。
* FLX Pro 仍有可改进空间，包括器械或织物漏、肺叶未覆盖、器械移位或偏移，以及复杂左心耳解剖中的植入成功率差异。
* Watchman Elite 的概念设计强调单件式顺应性框架，增加支架和锚数量，增加近端与远端节段，取消主体锥度，以改善贴壁、填充、即时固定和长期封堵区。
* 本器械在幻灯片中标注为概念器械或技术，尚未上市销售；前瞻性研究预计即将启动。

### 即使优秀器械仍可改进

现有器械的三类可见问题：器械或织物漏、未覆盖肺叶、器械迁移或移位。

本页以超声、彩色多普勒和造影图像说明，Watchman 系列已有良好基础，但仍存在可优化问题。左侧图像标注为 device/fabric leak，即器械或织物相关漏；中间造影图像标注为 uncovered lobe，即左心耳某一肺叶未被覆盖；右侧两幅超声图像标注为 device migration/shift，即器械迁移或位置偏移。

### Watchman Elite 的单件式顺应性设计

Watchman Elite 概念设计：完整填充 LAA、适应不同几何形态，并强调长期密封和无创远端。

本页展示 Watchman Elite 左心耳封堵器的概念设计。页面中心为器械框架，主标题写明这是单件式、顺应性设计，可填充完整左心耳并适配所有几何形态。

* Optimized Strut Architecture，优化支架结构：填充并适配左心耳的独特特征。
* Stability，稳定性：通过主动贴壁实现更好的锚定接合。
* Chronic Complete Seal，长期完全密封：扩大密封范围，提高成功机会。
* Next Generation Ball，下一代球形远端：进一步改善无创伤远端。

### 增强型框架结构

同尺寸器械中，WATCHMAN FLX 与 WATCHMAN Elite 的框架和左心耳填充对比。

本页标题为 WATCHMAN Elite: Enhanced Frame Architecture。左侧为 WATCHMAN FLX，中央和右侧为 WATCHMAN Elite，底部注明为同尺寸器械。图像对比显示，Elite 设计在左心耳中的填充更充分，框架形态更贴合。

* 支架和锚数量更多，从而增加接触和稳定性。
* 近端和远端节段增加，从而提高框架适应性。
* 主体零锥度，目标是获得更强的即时固定和更大的填充。

### 总结

本页总结 Watchman Elite 的设计方向。第一，设计改进旨在进一步提高 Watchman 器械封堵密封和植入成功率。第二，改进点包括支架和框架结构、形状以及织物长度。第三，改进目标还包括更好的顺应性、更好的器械贴壁接触和更好的稳定性。第四，前瞻性研究即将启动。
