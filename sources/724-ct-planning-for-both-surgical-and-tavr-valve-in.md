# 同时用于外科瓣中瓣与 TAVR 瓣中瓣的 CT 规划

## 核心要点

* CT 规划要同时回答四个问题：第二枚 TAV 尺寸、冠脉风险、第二枚 TAV 选择和第二枚 TAV 植入位置。
* TAV-in-SAV 中，外科瓣通常保留原始几何，True ID 决定 THV 尺寸；推荐 THV 尺寸为 True ID 加至少 0.5 mm oversize。
* TAV-in-TAV 中，原 TAV 体内欠扩张和不均匀扩张常见，因此体内 CT 测量可能优于台架尺寸或原生瓣环面积。
* 冠脉风险评估围绕 neoskirt 与 neoskirt plane（NSP）：风险开始于冠脉开口低于 NSP，需要测量冠脉风险区内最窄 VTA。
* VTA 阈值示意为：>4 mm 低风险，2-4 mm 中间区间，<2 mm 高风险；TAV-in-TAV 中若第二枚为 BEV 或计划扩张，要考虑进一步扩张。

### CT 术前规划需要评估什么

术前 CT 规划框架：第二枚 TAV sizing 与冠脉风险是主轴，同时受第二枚 TAV 选择和植入位置影响。

Pre-CT Planning 需要评估四个彼此关联的问题：第二枚 TAV sizing、coronary risk、第二枚 TAV choice 和第二枚 TAV implant position。图中箭头提示，第二枚 TAV 的选择与植入位置会同时影响尺寸策略和冠脉风险判断。

### 第二枚 TAV sizing：外科瓣与 TAV 的差异

Index SAV 通常保留出厂几何；Index TAV 则常见欠扩张，实际尺寸因病例而异。

Index SAV 与 index TAV 在二次瓣中瓣规划中的核心差异是几何稳定性。Index SAV 保留原始几何，可以参考 “out-of-the-box” 尺寸。Index TAV 常见 under-expansion，且不同病例的实际尺寸不同，因此不能简单用名义尺寸替代体内测量。

### TAV-in-SAV：True ID 决定 THV 尺寸

支架外科瓣中，True ID 与 Stent ID 不同；True ID 用于决定第二枚 THV 尺寸。

对于 stented surgical valve 的 TAV-in-SAV，True ID 决定 THV size。确认 True ID 的方法有两步：第一，获得 SHV 型号和尺寸，来源可以是外科记录、厂家记录或 CT 测量；第二，使用 ViV App 获取 True ID 与推荐 THV 尺寸。幻灯片给出的公式是：THV size = True ID + minimum 0.5 mm Oversize。

### TAV-in-TAV：体内 CT sizing 可能更好

TAV-in-TAV 可用体内 CT 测量真实 index TAV 支架框架；台架配对和原生瓣环面积也被列出。

对于 TAV-in-TAV，幻灯片强调 in-vivo CT sizing may be better。左侧方法是测量真实 index TAV stent frame。中间列出台架配对，即 failed short TAV 与第二枚 tall TAV 的匹配：short TAV #20 对 tall TAV #23，#23 对 #26，#26 对 #29，#29 对 #34。右侧则提示可使用 pre-index TAVI data 的 native annulus area，示例图中显示 Annulus Dimensions Area 366.1 mm²。

### TAV-in-TAV：欠扩张和尺寸变化

TAV 体内欠扩张和不均匀扩张常见；S3 尺寸可能因 Evolut 尺寸与植入位置而改变。

讲者列出两个理由支持体内 CT sizing。第一，under-expansion 与 uneven expansion 很常见。第二，第二枚瓣膜尺寸可能随 TAV 尺寸和位置变化而变化。示例显示 Evolut 26 和 Evolut 34 的节点位置不同，图中标注 Node 6 和 Node 4；对于 Evolut 26，S3 size may not change；对于 Evolut 34，S3 size may change。

### TAV-in-TAV：每类 TAV 有独特 sizing 策略

短瓣失败与高瓣失败的测量层面不同，分别按 3 个层面或 NSP 加下方节点平均。

对于 failed short TAV，建议取 3 个层面的面积平均：outflow、waist（narrowest）、inflow。对于 failed tall TAV，建议取 NSP 以及 NSP 下方节点的面积平均；图中给出两种规则，一种是 NSP 加下方 3 个 nodes，另一种是 NSP 加下方 2 个 nodes。右侧节点图分别标注从 0 到 6 或从 0 到 3 的 node 层级。

### 冠脉风险：neoskirt 与 NSP

TAV-in-SAV 和 TAV-in-TAV 共享 neoskirt 与 neoskirt plane 概念。

冠脉风险评估的共同概念是 neoskirt 和 neoskirt plane。Neoskirt 指 ViV 后形成的 tube graft；neoskirt plane（NSP）指 neoskirt 顶端所在平面。图中蓝线为 NSP，红色竖向区域示意新形成的 neoskirt。

### 冠脉风险机制：直接阻塞与窦隔绝

冠脉风险可表现为直接冠脉阻塞或 sinus sequestration，TAV-in-SAV 与 TAV-in-TAV 均适用。

冠脉风险机制分为两类。第一是 direct coronary obstruction，TAV-in-SAV 与 TAV-in-TAV 均可发生，示意为 neoskirt 或瓣架将冠脉开口直接遮挡。第二是 sinus sequestration，即窦部被新形成的 neoskirt 隔绝，图中同时标出 STJ。页脚图例再次说明浅蓝色区域代表 neoskirt。

### NSP 在哪里

TAV-in-SAV 中 NSP 是支架柱顶端；TAV-in-TAV 中 NSP 取决于 TAV 类型和第二枚 TAV 植入位置。

对于 TAV-in-SAV，NSP 是 top of stent post。图中示例显示 post 15.0 mm、neoskirt 15.0 mm，以及 post 16.0 mm、neoskirt 15.0 mm。对于 TAV-in-TAV，NSP 取决于 TAV 类型和第二枚 TAV 植入位置。右侧分别以 Sapien3 与 Evolut 说明 NSP 的差异。引用文献为 Akodad et al, JACC Imaging 2022; 15:1663，以及 Akodad et al, JACC Intervention 2021; 14:2298。

### TAV-in-TAV 的四种组合

TAV-in-TAV 按原瓣与第二枚瓣的高低组合分为四类。

四种组合为：Short in Short、Tall in Short、Tall in Tall、Short in Tall。图中分别展示短瓣中放短瓣、高瓣中放短瓣、高瓣中放高瓣、短瓣中放高瓣的几何关系。这一页为后续判断 NSP 位置的分型基础。

### 每种组合的 NSP 不同

四种 TAV-in-TAV 组合形成不同高度的 neoskirt plane。

对于 Short in Short，NSP 位于短瓣顶部附近；Tall in Short 的 NSP 随高瓣进入短瓣后的 neoskirt 顶端确定；Tall in Tall 的 NSP 更高；Short in Tall 的 NSP 位于高瓣中较低且可调整的位置。页脚再次定义 neoskirt 是 ViV 后形成的 tube graft，neoskirt plane 是 neoskirt 的顶部。

### Short-in-Tall 的特殊性

Short in Tall 组合中 NSP 水平可通过节点位置调整，示例为 Node 6、Node 5、Node 4。

Short in Tall 是 TAV-in-TAV 中特别独特的组合，因为 NSP level can be adjusted。图中显示同一短瓣置入高瓣时，NSP 可调整到 Node 6、Node 5 或 Node 4。这意味着第二枚 TAV 的植入深度会直接改变 neoskirt 顶部位置，从而影响冠脉风险区。

### 冠脉开口低于 NSP 时风险开始

当冠脉开口在 NSP 以上时低风险；当冠脉开口低于 NSP 时需要冠脉评估。

本页同时适用于 TAV-in-SAV 和 TAV-in-TAV。核心规则是：risk begins when coronary ostium below NSP。如果冠脉开口高于 NSP，图中标为 Low risk of coronary；如果冠脉开口低于 NSP，图中标为 Need coronary assessment。蓝线为 NSP，橙线和红点表示冠脉开口位置。

### 在冠脉风险区测量最窄 VTA

冠脉风险区位于 NSP 与冠脉开口之间；需要在该区域内寻找最窄 VTA。

Coronary Risk Zone 定义为 NSP 与 coronary ostium 之间的区域。VTA 指 valve-to-aorta distance。对于 TAV-in-SAV，最窄 VTA 通常位于 NSP；对于 TAV-in-TAV，图中提示最窄 VTA 可能等于 VTSTJ。页脚图例说明蓝线为 NSP，橙线为冠脉开口底部。

### 横断面测量最窄 VTA

横断面测量示例：TAV-in-SAV 中 VTC 为 3.1 mm 和 3.6 mm；TAV-in-TAV 中用第二枚 TAV 或球囊虚拟圆测量到 ostium。

在 TAV-in-SAV 中，除非计划 BVF 或扩张，一般不预计进一步扩张；示例图标出 VTC=3.1 mm 与 VTC=3.6 mm。中间箭头给出风险阈值：>4 mm 为低风险，2-4 mm 为中间范围，<2 mm 为高风险。对于 TAV-in-TAV，如果第二枚 TAV 计划使用 BEV 或计划 dilatation，则预期可能进一步扩张；VTA 的测量为从第二枚 TAV 尺寸或球囊尺寸的虚拟圆到冠脉 ostium。

### 术前 CT 清单

术前 CT checklist：理解失败瓣、决定第二枚 TAV sizing、评估冠脉风险。

术前 CT checklist 分为三部分。第一，理解 index 或 failed SAV/TAV：确认瓣膜型号和尺寸；理解第二枚 TAV 选择、植入位置以及潜在 NSP。第二，进行第二枚 TAV sizing：TAV-in-SAV 遵循 True ID；TAV-in-TAV 采用体内 CT sizing。第三，评估冠脉风险：评估 NSP、冠脉和 STJ 的相对关系；在冠脉风险区测量最窄 VTA。当 NSP 位于 STJ 之上时，测量 VTSTJ；当 NSP 位于 STJ 之下时，测量 VTA at NSP。右侧还展示 Aortic ViV app 和 Redo TAV app 的二维码。
