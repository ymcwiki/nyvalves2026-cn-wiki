# 特色讲座：经房间隔 TMVR 真实世界结局，MIRACLE 注册研究启示

## 核心要点

* MIRACLE 是 22 个 心脏瓣膜中心参与的真实世界、研究者发起、回顾性、多中心注册研究，纳入 2018-2025 年经房间隔 TMVR 病例。
* 全队列 N=144，最常用平台为 Sapien M3，81 例，占 56.3%。
* 程序结局良好：MVARC 技术成功 95.1%，器械植入成功 92.4%，程序死亡 1.4%，程序卒中 0.0%。
* MR 控制持久：出院 MR ≤1+ 为 96%，1 个月为 97%，1 年为 93%。
* 中位随访 116 天，12 个月全因死亡为 13.3%，心衰住院为 17%，死亡或心衰住院复合终点为 25.7%。

### 研究背景与目的

TMVR 是二尖瓣疾病患者的替代治疗选择，目标是完全且持久消除 MR，并带来临床获益。该领域快速发展，但当代经房间隔 TMVR 的真实世界数据仍有限。本研究目的为评估当代经房间隔 TMVR 的安全性和有效性。页面引用 Guerrero 等 Lancet 2025 年文献。

### MIRACLE 注册研究设计与中心

MIRACLE 注册研究来自美国、欧洲和加拿大 22 个心脏瓣膜中心。

### TMVR 后 MR 演变

TMVR 后出院、1 个月和 1 年 MR ≤1+ 比例均超过 90%。

基线 n=144：MR 3+ 为 31%，MR 4+ 为 69%。出院 n=141：MR ≤1+ 为 96%，其中 0 级 79%，1+ 17%，2+ 2%，3+ 1%，4+ 1%。1 个月 n=89：MR ≤1+ 为 97%，其中 0 级 78%，1+ 19%，2+ 1%，3+ 1%，4+ 1%。1 年 n=42：MR ≤1+ 为 93%，其中 0 级 69%，1+ 24%，2+ 7%，未见更高级别 MR。

### 研究局限

局限包括：观察性回顾性注册研究；临床事件由中心报告；无独立事件判定；数据异质性较高；随访时间有限。

### 结论

经房间隔 TMVR 与较高程序成功率和截至 1 年的持久 MR 消除相关。患者有明显临床改善，表现为 NYHA 分级改善。死亡和心衰住院结果良好，并与 ENCIRCLE 枢纽研究相符。TMVR 后出现早期心脏反向重构信号。讲者邀请加入 MIRACLE Registry，并给出联系邮箱 [邮箱已隐去] 与 [邮箱已隐去]。
