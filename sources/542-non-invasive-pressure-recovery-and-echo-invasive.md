# TAVR 后无创压力恢复与超声-有创梯度不一致

## 核心要点

* 研究纳入 189 例重度原生主动脉瓣狭窄 TAVR 患者，比较术后超声平均跨瓣压差与同步双 pigtail 有创测量的差异。
* T0 即刻术中，超声 MG 5.4 ± 2.4 mmHg，有创 MG 4.2 ± 2.3 mmHg，ΔMG 为 +1.2 ± 2.2 mmHg；T1 出院前超声 MG 升至 8.0 ± 3.4 mmHg，ΔMG 为 +4.0 ± 3.6 mmHg。
* PR 与 ΔMG 明显相关：T0 r=0.50、T1 r=0.80，均 p<0.001。
* PR 可能高估真实不一致幅度：ε = PR − ΔMG，T0 为 +3.4 ± 2.2 mmHg，T1 为 +3.1 ± 2.2 mmHg。
* 23 mm 自膨瓣的 ΔMG 和 PR 均高于 25-34 mm 大瓣，提示小瓣 TAVR 后看到高超声梯度时要主动考虑压力恢复。

### 背景：压力恢复机制与公式

压力恢复示意图：Doppler 在 vena contracta 处测得较高梯度，而有创测量反映净跨瓣梯度。

压力恢复（Pressure Recovery, PR）发生在血流经过狭窄孔口后，在升主动脉远端减速，动能重新转化为压力。Doppler 超声测量的是 vena contracta gradient，因此会系统性高于有创测得的净跨瓣压差。TAVR 后，人工瓣口几何形态可能放大 PR，但其对超声-有创梯度不一致的机制贡献此前没有被系统量化。

PR 公式为：PR = MGecho × 2 × EOA / AA × [ 1 − (EOA / AA) ]。其中 EOA 为有效瓣口面积，单位 cm²；AA 为升主动脉横截面积，单位 cm²。图示中 LVOT、Aortic valve、VC、SJ、Aorta、Pmax、Pnet、PR 和 flow direction 用来说明从最小压力点到远端压力恢复的过程。

### 研究设计与定义

研究设计为前瞻性、双中心研究，时间从 2024 年 1 月至 2025 年 12 月。研究人群为 189 例重度原生主动脉瓣狭窄并接受 TAVR 的患者。

关键定义包括：有创梯度使用同步双 pigtail catheter technique，一根位于主动脉、一根位于左心室；ΔMG 即 echo-invasive discordance，定义为 MGecho − MGinvasive；PR 使用 MGecho × 2 × EOA / AA × [ 1 − (EOA / AA) ]；ε 即 residual discordance，定义为 PR − ΔMG。

时间点包括 T0（intra-procedural），即瓣膜释放后即刻；T1（pre-discharge），即出院前经胸 Doppler 超声。研究目的有两项：第一，首次系统评估 PR 作为 TAVR 后超声-有创梯度不一致 ΔMG 的机制决定因素；第二，按瓣膜尺寸进行亚组分析，比较 23 mm 与 25-34 mm。

### 结果：超声与有创压差

T0 和 T1 的超声、有创压差、ΔMG、PR 与 ε；右侧柱状图显示 echo MG 高于 invasive MG，均 p<0.001。

| 参数 | T0（术中） | T1（出院前） |
| --- | --- | --- |
| 超声平均跨瓣压差 | 5.4 ± 2.4 mmHg | 8.0 ± 3.4 mmHg |
| 超声峰值跨瓣压差 | 9.5 ± 4.7 mmHg | 14.5 ± 6.1 mmHg |
| 主动脉峰速 | 1.6 ± 1.3 m/s | 1.9 ± 0.4 m/s |
| 有创峰值跨瓣压差 | 5.9 ± 4.1 mmHg | - |
| 有创平均跨瓣压差 | 4.2 ± 2.3 mmHg | - |
| ΔMG = 超声 − 有创 | +1.2 ± 2.2 mmHg | +4.0 ± 3.6 mmHg |
| Pressure Recovery | 4.7 ± 2.3 mmHg | 7.1 ± 3.2 mmHg |
| ε = PR − ΔMG | +3.4 ± 2.2 mmHg | +3.1 ± 2.2 mmHg |

右侧柱状图进一步显示：T0 时 echo MG 约 5.4 mmHg、invasive MG 约 4.0 mmHg，图中 ΔMG 标注为 +1.20 mmHg，p<0.001；T1 时 echo MG 约 8.0 mmHg、invasive 参考约 4.0 mmHg，图中 ΔMG 标注为 +3.93 mmHg，p<0.001。

### 结果：PR 与 ΔMG 相关性

PR/NIPR 与 ΔMG 的散点相关：T0 为中等相关，T1 更强。

左侧 T0（intra-procedural）散点图以 NIPR（mmHg）为横轴、ΔMG（mmHg）为纵轴，Pearson 相关 r = 0.50，p < 0.001。横轴大致覆盖约 1 至 14 mmHg，纵轴大致从 −4 至 10 mmHg。

右侧 T1（pre-discharge）散点图同样以 NIPR（mmHg）为横轴、ΔMG（mmHg）为纵轴，Pearson 相关 r = 0.80，p < 0.001。横轴约 2.5 至 17.5 mmHg，纵轴约 −5.0 至 12.5 mmHg。

页面结论为：超声-有创梯度不一致 ΔMG 可由 PR 解释；压力恢复倾向于高估真实压差，ε（PR − ΔMG）在 T0 为 +3.4 ± 2.2 mmHg，在 T1 为 +3.1 ± 2.2 mmHg。

### 亚组：小瓣与较大瓣

23 mm SEV 与 25-34 mm 大瓣的 ΔMG、PR 和 PR-ΔMG 相关性比较。

| 变量 | 23 mm SEV（N = 23） | Large 25-34 mm（N = 166） | p 值 |
| --- | --- | --- | --- |
| ΔMG at T0 | +2.8 ± 2.9 mmHg | +1.1 ± 1.9 mmHg | <0.001 |
| ΔMG at T1 | +6.5 ± 4.9 mmHg | +3.9 ± 3.5 mmHg | 0.004 |
| PR at T0 | 6.9 ± 3.1 mmHg | 4.0 ± 1.5 mmHg | <0.001 |
| PR at T1 | 9.5 ± 3.5 mmHg | 6.7 ± 3.0 mmHg | <0.001 |
| Pearson r at T0 | 0.55（p = 0.010） | 0.28（p = 0.005） |  |
| Pearson r at T1 | 0.85（p < 0.001） | 0.79（p < 0.001） |  |

小瓣亚组在术中和出院前均显示更大的 echo-invasive discordance，也显示更高 PR。T1 时 23 mm SEV 的相关性达到 0.85，提示小瓣术后高超声压差尤其可能包含压力恢复成分。

### 结论

结论包括四点。第一，echo-invasive gradient discordance（ΔMG）与 PR 强相关，T0 的 r = 0.50，T1 的 r = 0.80，两者均 p < 0.001。第二，PR 在 T0 与 T1 均可能高估不一致幅度，T0 的 ε 为 +3.4 mmHg，T1 的 ε 为 +3.1 mmHg。第三，在 23 mm SEV 中这种效应被放大，23 mm 瓣的 discordance 和 PR 均显著高于较大瓣，p 值至少达到 <0.001 或 0.004。第四，TAVR 后若超声提示高梯度，应考虑 pressure recovery，因为它解释了观察到的不一致中的相当一部分。
