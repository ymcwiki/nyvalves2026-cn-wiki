# TAV-in-TAV：瓣膜选择、尺寸、瓣叶改造与冠脉通路

## 核心要点

* 病例为 63 岁男性，既往二叶式主动脉瓣狭窄伴右-左冠瓣嵴钙化，因肝硬化和 NASH 外科风险高，2019 年接受 34 Evolut TAVR；本次因瓣膜性主动脉瓣反流和心衰症状就诊。
* 术前 CT 显示左冠开口对应 node 5，右冠开口对应 node 3；从 node 0 到 node 6 的内径、面积和周长逐层变化，node 6 距离 27.4 mm，面积 438.4 mm²，周长 74.5 mm。
* 冠脉风险判断显示 node 4 的 VTC 足够，读数约 6.8 和 6.9 mm；但右冠侧 node 6 的 VTA 很小，仅 1.6 和 1.8 mm，因此需要主动降低右冠阻塞风险。
* 瓣膜方案比较了 23 S3 @n6、23 S3 @n4 和 26 S3 @n6，最终计划为右侧瓣叶 TELLTALE 改造并在 node 6 植入 26 S3。
* 术中影像展示了 TELLTALE 导管、圈套和瓣叶改造步骤，随后在既有自膨瓣内植入球扩瓣；患者术后第 1 天出院，状态良好。
* 讲者强调 TAV-in-TAV 时要考虑球扩瓣在自膨瓣内的上下移动，并提前降低冠脉阻塞或心室侧栓脱等风险。

### 病史与本次表现

患者为 63 岁男性。既往二叶式主动脉瓣狭窄，右-左冠瓣嵴（R-L raphe）钙化。合并肝硬化和 NASH，因此外科风险高。患者于 2019 年接受 TAVR，植入 34 Evolut。目前因瓣膜性主动脉瓣关闭不全（valvular AI）和心衰症状再次就诊。

### VTC 足够，但右冠 VTA 偏小

node 4 的 VTC 测量约 6.8-6.9 mm，右冠侧 node 6 的 VTA 仅 1.6-1.8 mm。

标题写明“VTC (node 4) adequate. R VTA node 6 small”。左侧 CT 横断面标注 NC、RC、LC，node 4 水平的 VTC 读数为 Ø 6.8 mm 和 Ø 6.9 mm，提示虚拟瓣膜到冠脉距离足够。右侧图集中在右冠相关空间，标注 1.6 mm 和 1.8 mm，提示 R VTA node 6 很小，存在冠脉阻塞风险，需要在计划中处理右侧瓣叶。

### 瓣膜尺寸和释放高度模拟

三种 TAV-in-TAV 模拟：23 S3 @n6、23 S3 @n4 和 26 S3 @n6。

本页比较三种球扩瓣植入方案。左图为 23 S3 @n6，中图为 23 S3 @n4，右图为 26 S3 @n6。三维图中显示现有 Evolut 瓣架、拟植入 SAPIEN 3 的位置、冠脉/参考点和释放平面。结合前页 VTC/VTA 结果，最终策略倾向于在 node 6 位置使用较大的 26 S3，并通过右侧瓣叶改造降低冠脉阻塞风险。

### 治疗计划

计划包括两项。第一，使用 TELLTALE 进行右侧瓣叶改造（Right leaflet modification with TELLTALE）。第二，在 node 6 植入 26 S3。

### TELLTALE 瓣叶改造系统

TELLTALE Leaflet Modification System 的导丝、绝缘涂层、金标记、裸露段、kinker-denuder 和 wire grippers。

系统标题为“TELLTALE Leaflet Modification System”。导丝规格标注为 0.014" OD x 310 cm。导丝主体有“Electrically Insulative Coating”，即电绝缘涂层；有 Gold Marker 金标记；两端或关键工作段标注为 Un-insulated 非绝缘。下方器械包括 Kinker-denuder、形成 Denuded & Kinked (100° Angle) 的裸露并弯折导丝段，以及 Wire grippers 导丝夹持器。该系统的目的，是在目标瓣叶上建立可控的电切/改造路径，以避免再瓣膜植入后瓣叶覆盖冠脉。

### TELLTALE Pachyderm 导管

Pachyderm 导管包括左瓣叶导引导管、圈套导引导管和右瓣叶 3D 导引导管。

本页展示三类 TELLTALE Pachyderm Catheters。第一是 LEFT LEAFLET GUIDE CATHETER，即左瓣叶导引导管；第二是 SNARE GUIDE CATHETER，即圈套导引导管；第三是 RIGHT LEAFLET 3D GUIDE CATHETER，即右瓣叶三维导引导管。由于本病例计划做右侧瓣叶改造，右瓣叶 3D 导引导管是关键器械之一。

### 结论与操作要点

结论包括三点。第一，患者术后第 1 天出院（POD 1），恢复良好。第二，在自膨瓣（SEV）内植入球扩瓣（BEV）时，需要考虑瓣膜可能向上或向下移动。第三，需要主动降低风险，尤其是冠脉阻塞和瓣膜向心室侧栓脱。对本病例而言，风险控制路径包括术前 CT 节点测量、VTC/VTA 评估、右侧瓣叶 TELLTALE 改造，以及在 node 6 精准释放 26 S3。
