# 使用 EVOQUE 系统经导管三尖瓣置换的单中心经验

## 核心要点

* 这是一项单中心回顾性研究，纳入 2024 年 6 月至 2026 年 4 月 接受 EVOQUE TTVR 的连续重度 TR 患者。
* 共 96 例，年龄中位数 80 岁（IQR 76-85），女性 72%，NYHA III/IV 82%。
* 1 年全因死亡率为 16%（n=15）；30 天内新永久起搏器植入率为 9%（n=9）。
* 中重度 RV 功能障碍从术前 30% 增至术后 57%，提示 TTVR 后右室功能监测很关键。
* PVL 分布为无 45%、trivial/trace 36%、轻度 16%、中度或以上 3%。

### 题目与单位

题目为“Single-Center Experience with Transcatheter Tricuspid Valve Replacement Using the EVOQUE System”，中文为“使用 EVOQUE 系统经导管三尖瓣置换的单中心经验”。作者为 Qais Radaideh, MD 和 Tanvir Bajwa, MD，单位为 Aurora St Luke Medical Center, Milwaukee, WI。

### 研究背景

使用 EVOQUE 系统进行经导管三尖瓣置换（TTVR）是治疗重度三尖瓣反流（TR）的新兴治疗方式，但真实世界结局数据仍有限。

### 研究方法

研究者回顾性分析了本机构接受 EVOQUE TTVR 的连续重度 TR 患者。时间范围为 2024 年 6 月至 2026 年 4 月。评价结局包括 1 年全因死亡率、TTVR 后 30 天内新永久起搏器（PPM）植入、TTVR 前后右室（RV）功能，以及瓣周漏（PVL）。

### 基线特征与 1 年死亡

| 项目 | 结果 |
| --- | --- |
| 样本量 | 96 例 |
| 年龄 | 中位数 80 岁，IQR 76-85 |
| 女性 | 72% |
| NYHA III/IV | 82% |
| LVEF | 60%，IQR 52-64% |
| TriScore | 中位数 4，IQR 2-7 |
| Torrential 或 massive TR | 23% |
| 术前 RV failure | 39% |
| 术前中重度 RV dysfunction | 30% |
| 胆红素升高 | 31% |
| eGFR <30 mL/min/1.73m² | 16% |
| 既有 RV lead | 40 例，其中 7 例为 CRT-D，8 例为 ICD lead |
| 1 年全因死亡率 | 16%，n=15 |

该队列为高龄、症状重、合并右室和终末器官问题较多的真实世界患者。

### PPM、RV 功能、HF 住院和 PVL

| 结局 | 结果 |
| --- | --- |
| 30 天内新 PPM 植入 | 9%，n=9 |
| 中重度 RV dysfunction | 从术前 30% 增至术后 57% |
| 至少一次 HF 住院 | 25/96 例，在植入后不同时间发生至少一次心衰事件住院 |
| PVL 无 | 45% |
| PVL trivial/trace | 36% |
| PVL 轻度 | 16% |
| PVL 中度或以上 | 3% |
| 轻度至中度 PVL 且既有 RV lead | 9 例 |

页面特别强调中重度右室功能障碍从术前到术后明显增加，这可能反映重度 TR 消除后的右室后负荷变化，也提示术后右室评估和随访不能省略。

### 结论

EVOQUE TTVR 能够有效降低 TR，但伴随术后 RV 功能障碍显著增加、有临床意义的新 PPM 发生率，以及较高死亡率。死亡率略高于临床试验数据，原因可能是该真实世界队列患者更重、更高龄。研究提示需要谨慎选择患者、充分评估 RV，并建立结构化术后监测流程。
