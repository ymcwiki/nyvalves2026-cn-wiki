# 介入视角：重度 AS 合并 CAD 时的 PCI 时机、瓣膜选择与瓣叶改造

## 核心要点

* 病例核心是低外科风险但强烈拒绝开胸的重度主动脉瓣狭窄患者，同时存在开口左主干狭窄，介入方案必须同时处理冠脉保护、瓣膜选择和未来冠脉再进入。
* 冠脉造影显示左主干约 70% 狭窄且导管嵌入时压力阻尼，IVUS 最小管腔面积约 4.44 mm²，支持左主干病变有血流动力学意义。
* CT 显示主动脉瓣环面积 351 mm²、周长 67.9 mm，右冠开口高度仅 9.9 mm，窦部偏小，使自膨瓣和未来 TAV-in-TAV 更受限制。
* 选择先做开口左主干支架，再行 TAVR；术者在 TAVR 后复查 IVUS，发现开口支架轻度受压并再次后扩张。
* 讲者强调，对于这类病例，现在会更积极考虑 native BASILICA 等瓣叶改造，以保护左主干通路并保留长期治疗选择。

### 病例标题与讲者信息

本页为题名页。英文标题为 “IC Perspective on CAD and AS: Case Presentation”。讲者为 Paul Mahoney, MD, FACC, FSCAI；职务为 Banner University Medical Center, Phoenix 的 Director Transcatheter Therapies，同时任职 University of Arizona College of Medicine。

### 临床背景：重度 AS 合并开口左主干病变

病例起点：患者出现气短和胸痛，外院检查提示重度主动脉瓣狭窄与开口左主干病变。

患者为 70 岁女性，因进行性加重的气短和胸痛就诊，既往已知主动脉瓣狭窄逐渐进展。既往史包括颈动脉内膜剥脱术、HTN 高血压和 HLD 高脂血症。她的兄弟 5 年前在同一机构接受 CABG 后因并发症死亡，因此患者极度害怕开放心脏手术。

外院导管检查显示重度主动脉瓣狭窄和开口左主干病变。风险和功能状态方面，STS 评分为 3.3%，NYHA 分级为 II 级，体表面积/体重指数标注为 1.8/31。

### 超声评估主动脉瓣狭窄严重程度

经胸超声显示高跨瓣压差、高流速和偏小瓣环。

超声提示重度主动脉瓣狭窄：平均跨瓣压差为 43 mmHg，主动脉瓣最大流速 Vmax 为 4.4 m/sec。二维超声测量主动脉瓣环为 18 mm。这一页的重点是确认症状性重度 AS，同时提示瓣环尺寸偏小。

### 冠脉造影与 IVUS：左主干开口重度狭窄

冠脉造影和 IVUS 显示左主干开口狭窄，导管嵌入时出现压力阻尼。

| 血管/检查 | 结果 |
| --- | --- |
| 左主干 | 约 70% 狭窄，导管嵌入时有 damping 压力阻尼 |
| LAD | 正常 |
| LCx | 正常 |
| RCA | 正常 |
| IVUS 左主干 MLA | 4.44 mm² |

这一页建立了左主干病变的治疗必要性：病变位于开口，造影有明显狭窄，导管压力阻尼提示病变具有临床意义，IVUS 最小管腔面积进一步支持需要血运重建。

### CT 主动脉根部评估：小窦部与低右冠高度

CT 评估瓣环、冠脉高度和窦部尺寸，用于判断 SAVR/CABG 与 TAVR 的解剖风险。

| CT 参数 | 数值 |

| 瓣环直径 | 23 × 20 mm |
| 瓣环面积 | 351 mm² |
| 瓣环周长 | 67.9 mm |
| 左冠开口高度 | 14.3 mm |
| 右冠开口高度 | 9.9 mm |

讲者强调窦部偏小。若进行 SAVR/CABG，可能需要主动脉根部扩大以避免患者-人工瓣不匹配。若进行 TAVR，尺寸上可考虑 23 mm Sapien 或 26 mm Evolut，但窦部对于 Evolut 来说过窄，且右冠高度低，使冠脉相关风险需要特别审视。

### 股动脉入路评估

CT 下肢血管重建显示经股动脉通路可行。

本页显示腹主动脉、髂动脉和股动脉通路评估。讲者结论为经股动脉入路良好，因而经股 TAVR 从血管通路角度可实施。

### 心脏团队讨论：推荐手术但患者拒绝

心脏团队讨论后，认为最佳选择是外科手术。推荐方案包括主动脉根部扩大和主动脉瓣置换，LIMA 至 LAD，以及 SVG 至 OM。患者拒绝手术，并且已充分知晓手术与经皮治疗的风险和获益；若经皮方案中出现危及生命的并发症，患者愿意接受外科抢救。

讲者提出接下来的实际问题：现在应该怎么做？是否因患者拒绝手术而拒绝经皮治疗？是否转诊到其他中心寻求第二意见？左主干支架和 TAVR 叠加时，风险是简单相加还是互相放大？

### 低风险 AS 合并左主干病变：证据空白与操作顺序

左主干支架治疗已经成为常规介入治疗的一部分，但 EXCEL 研究及其解读仍有争议。低风险主动脉瓣狭窄中，PARTNER 3 显示 TAVR 在 1 年死亡和再住院方面较外科有优势，但 未保护左主干病变是 PARTNER 3 的排除标准。

因此，对于低危或中危 TAVR 患者合并左主干支架，目前缺乏直接数据。讲者提出三个问题：经皮策略是否合适？患者偏好应占多大权重？如果采用经皮策略，是分期还是同期，先 PCI 还是先 TAVR？是否需要瓣叶改造以降低冠脉风险并方便未来再进入？

### TAVR 与左主干 PCI 注册研究经验

TAVR-LM 注册研究显示，高危/不能手术患者中左主干 PCI 与 TAVR 可行，但需要关注血管并发症和左主干阻塞。

| 项目 | 结果 |
| --- | --- |
| 左主干支架患者数 | N=204 |
| PCI 在 TAVR 前 | 82% |
| PCI 与 TAVR 同期 | 13% |
| PCI 在 TAVR 后 | 5% |
| 开口左主干 | 54% |
| 非开口左主干 | 46% |
| 平均支架数 | 1.1 枚/患者 |
| 平均 EF | 51% |
| 30 天死亡率 | 3.1% |
| 左主干阻塞 | 1.6% |
| 血管并发症 | 16%，较高 |

中心示意图列出 TAVR 加左主干 PCI 的关键议题：解剖相互作用、介入过程中的血流动力学不稳定、左主干 PCI 的时机、开口或非开口左主干支架，以及计划内或非计划左主干 PCI。Kaplan-Meier 曲线比较 TAVR 合并左主干 PCI n=128 与单纯 TAVR n=128，p=0.83，约 1 年生存曲线基本重叠。

### 左主干支架与 TAVR 瓣架的相互影响

开口左主干支架和 TAVR 瓣架之间可能发生直接或间接相互作用。

本页讨论特殊注意事项。左主干支架和 TAVR 瓣膜需要评估支架与瓣架的相互影响。直接风险是支架变形或被压瘪；间接风险是瓣膜钙化被推移后造成冠脉口阻塞。开口支架理想位置应与开口齐平，若支架伸入主动脉则会给 TAVR 释放和未来再进入带来额外问题。

讲者还强调未来左主干再进入的问题：短框架瓣膜与高框架瓣膜的差异、原生瓣叶位置、以及是否应进行瓣叶改造都需要在第一次 TAVR 前规划。

### 短框架与高框架瓣膜：冠脉再进入差异

短框架瓣膜开放网格较大，理论上更利于冠脉再进入；高框架瓣膜还需关注连合定位。

左侧示意短框架瓣膜的特点：瓣叶位于瓣环内，框架高度较低，网格开口较大。图中文字提示一组病例中 TAVI 后冠脉再进入成功率为 100%，即 68/68 例成功。

右侧为 Evolut 连合和标记物定位示意，标出 NCC、LCC、RCC。图中说明 C-tab 位于右侧，hat marker 位于正前方中央，C-tab 对应 THV 连合，也对应 flush port。对于小窦部、左主干支架和未来冠脉操作，这些空间关系会影响瓣膜选择。

### 术前规划：左主干开口支架和 23 mm Sapien S3

CT 中虚拟植入 23 mm Sapien S3，用于判断覆膜框架和左主干开口的关系。

本页以 CT 重建进行术前规划。虚拟放置 23 mm Sapien S3 后，覆膜框架看起来位于左主干开口下方；但原生瓣叶可能在开放网格周围延伸得更高。图中标尺显示 5 mm/格，用于估计瓣架、瓣叶和冠脉口间距离。

讲者说明，Evolut 因窦部太小而不理想，高框架也会使冠脉再进入和未来 TAV-in-TAV 更困难。Navitor 的冠脉风险平面低于 Evolut 但高于 Sapien，因此仍需要把长期治疗路径纳入决策。

### 基线造影与 IVUS：确认左主干开口狭窄

基线左主干造影和 IVUS 再次确认开口病变，导管嵌入出现明显阻尼。

术中基线造影显示左主干开口病变。IVUS 测得左主干管腔尺寸约 4.1 × 4.4 mm，开口最小管腔面积约 4.8 mm²。导管进入左主干时发生严重压力阻尼，进一步提示开口病变不宜忽略。

### 开口左主干支架定位与释放

左主干开口支架释放过程，重点是控制支架与冠脉开口的相对位置。

本页展示开口左主干支架的定位和释放。支架需要覆盖开口病变，同时尽量避免明显伸入主动脉腔内，因为过多主动脉内突出可能在后续 TAVR 时被瓣架影响，也会增加未来再进入难度。

### 左主干支架后扩张

支架释放后进行球囊后扩张，以优化开口左主干支架形态。

支架置入后，术者对左主干开口支架进行后扩张。对开口病变而言，后扩张的目的包括改善支架贴壁、保证开口充分扩开、减少后续 TAVR 瓣架或原生瓣叶对支架入口的影响。

### 左主干支架即时结果

左主干开口支架后造影，左冠系统血流保持通畅。

造影显示左主干支架后血流保持良好，左冠系统无明显残余受限。该结果为下一步 TAVR 提供了条件，但并未消除 TAVR 释放时对左主干支架和冠脉开口的潜在风险。

### TAVR 前再次规划：瓣叶改造的角色

讲者用另一例图像说明 native BASILICA 可用于保留左主干通路和未来治疗选项。

本页讨论 lifetime management。TAVR 前规划不仅要完成当前瓣膜置换，还要考虑未来冠脉通路和未来 valve-in-valve。讲者指出，左冠瓣 native BASILICA 用于保留左主干进入通路和未来治疗选择，在这个病例中会是很合理的策略。

讲者以 “If I did this case today...” 引出观点变化：若今天处理同样病例，会更强烈考虑瓣叶改造，而不仅是完成左主干支架和 TAVR。

### 左主干支架后、TAVR 前的 IVUS 与保护策略

术者在左主干支架后思考 TAVR 时是否保留导丝、是否预置保护球囊，以及 TAVR 后是否复查支架。

本页讨论 TAVR 植入对左主干支架的风险。问题一是是否在左主干内保留导丝。优点是可预置球囊来保护支架，一旦支架受压可立即处理；缺点是导丝或球囊可能被困在变形支架和瓣架之间。问题二是 TAVR 后是否重新进入左主干并检查支架。讲者后续选择了 TAVR 后复查并处理。

### 左主干支架后的 TAVR 操作

左主干支架已经置入后进行 TAVR，需持续关注瓣膜输送系统与支架的空间关系。

本页显示 TAVR 输送系统通过主动脉瓣并定位。由于左主干开口已存在支架，释放前必须确认瓣膜框架、原生瓣叶、钙化和支架开口之间不会造成急性左主干阻塞或支架塌陷。

### 开口左主干支架后的 TAVR 释放

TAVR 瓣膜释放过程中，左主干支架和左冠通路仍被重点监测。

图中显示瓣膜释放后的框架位置，左主干支架附近仍有导管或导丝监测。此时的关键是确认 Sapien 瓣架上缘和原生瓣叶位置不会阻塞左主干入口，同时支架本身没有被明显压缩。

### TAVR 后即时影像

TAVR 后瓣膜框架位置稳定，随后进行左主干支架复查。

本页为 TAVR 后透视图像，显示人工瓣膜已释放并处于稳定位置。由于病例核心风险在于左主干支架和 TAVR 的相互作用，术者随后没有仅凭造影结束，而是继续检查左主干开口支架。

### TAVR 后复查：左主干支架轻度受压并后扩张

TAVR 后 IVUS 显示左主干开口支架轻度受压，术者进行后扩张并再次 IVUS 评估。

TAVR 后重复 IVUS 发现开口支架有轻度压缩。术者对左主干支架进行后扩张，并再次进行 IVUS 复查。该步骤说明，在左主干开口支架加 TAVR 的病例中，瓣膜释放成功并不等于冠脉处理已经完成，支架形态和开口面积仍需确认。

### 临床经过与随访

病例临床经过良好，短流程、早期下床、术后第 1 天出院，并在 3 年随访时状态良好。

手术皮肤到皮肤时间为 38 分钟。患者术后 4 小时下床活动，术后第 1 天出院回家。随访 3 年时患者状况良好。
