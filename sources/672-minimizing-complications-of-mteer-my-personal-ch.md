# 最小化 mTEER 并发症：我的个人清单

## 核心要点

* mTEER 总体属于低风险、高成功率操作，但仍需主动防范 SLDA、器械栓塞、卒中、出血、心包积液、血管损伤、残余 MR 和术后 MS。
* 术前首要步骤是患者选择：必须确认症状性重度 MR、GDMT 已优化、解剖适合、手术风险评估合适，并形成 heart team 共识。
* 原发 MR 不应因药物治疗过久而延误干预；继发 MR 要确认容量状态、最大耐受 GDMT、CRT 评估，以及必要时低输出和 OHT 筛查。
* 术中关键是“慢下来确认”：经房间隔高度至少距瓣环 4 cm，必要时用 RF 系统或球囊房间隔成形，释放前确认夹持、组织桥、MR 降低和器械稳定。
* 术后立即记录 TTE、残余 MR、跨瓣压差、心包积液和入路情况；出院前明确抗栓策略、随访和患者教育。

### MV TEER 的风险：低风险，高成功率

MV TEER 常见风险和发生率总结；右侧示意包括瓣叶损伤、开胸转换、缺血事件、心包填塞、感染性心内膜炎、肾衰、血管并发症、院内死亡和 SLDA。

该页列出的 MV TEER 风险和结局如下：

| 项目 | 发生率或结局 |
| --- | --- |
| Single Leaflet Device Attachment, SLDA | 0–2% |
| Device embolization，器械栓塞 | 0.1% |
| Stroke，卒中 | 2% |
| Major bleeding，重大出血 | 5–7% |
| 30-day mortality，30 天死亡率 | 3% |
| Procedural mortality，操作相关死亡率 | <1% |
| Commercial success，商业化真实世界成功率 | >95% |

右侧风险示意中还标注了 isolated leaflet damage、conversion to open heart surgery、ischemic events、pericardial tamponade、infective endocarditis、renal failure、major vascular complications、in-hospital death 和 single leaflet device attachment。

### mTEER 成功依赖什么

讲者把 mTEER 成功归纳为患者选择、影像评估、操作计划、沟通和早期识别并发症。

总体清单包括 5 项：appropriate patient selection，合适的患者选择；detailed imaging assessment，详细影像评估；procedural planning，操作计划；communication，团队沟通；early recognition of complications，早期识别并发症。

### 术前清单：患者选择

讲者的术前清单首先是患者选择，具体包括：症状性重度 MR；指南指导药物治疗已经优化；解剖条件有利；手术风险评估恰当；heart team 达成共识。

### 操作清单：临床评估

术中或围术期临床评估清单包括：容量状态；血红蛋白；凝血状态；食管病变；衰弱评估；血管入路评估。

### 患者选择：获得完整病史

患者选择的第一步是获得照护史，并全面评估和记录症状。讲者强调要 FULLY assess symptoms and DOCUMENT，即充分评估症状并写入病历。

与外科二尖瓣置换或修复不同，目前 MV TEER 仅获批用于症状性二尖瓣反流。如果考虑超说明书使用，例如无症状但左室扩大或 LVEF 降低患者接受 MV TEER，应清楚告知患者：获益推断来自外科文献的外推。

### 患者选择：原发 MR 的药物优化

对于原发 MR，如果高血压或房颤控制不佳，应处理 HTN 或 afib；利尿剂仅用于症状控制。讲者提醒，药物治疗在原发 MR 中获益有限。

应与心胸外科，CTS，建立合作关系，便于快速转诊和多学科讨论。治疗时机要谨慎评估，尤其当症状起病为“acute”或“subacute”时，不要延误，应加快治疗。

还要事先讨论备选方案，明确“如果 MV TEER 失败 then…”之后如何处理。

### 患者选择：继发 MR 与全身状态

对于继发 MR，应确认患者处于 euvolemic 状态，并使用最大耐受 GDMT。幻灯片列出的药物类别为 ACE/ARB/ARNI、MRA、SGTL2、BB，其中 SGTL2 为页面原文写法。

如合适，应转诊 CMP；若担心低输出，可考虑 RHC。讲者提出一个关键问题：患者是否需要在 MV TEER 前预先筛查 OHT。若适合 CRT，也应确认患者已经接受相关评估。

此外，应处理衰弱和其他共病，以便设定现实预期。

### 影像评估

影像方面要预判挑战，包括声影、显像不足，以及是否需要 MultiPlanar Reconstruction。

需要提前处理小 MVA、高 MG、钙化、瓣叶短等解剖问题。策略要提前规划，包括 NT、NTW、XT、XTW、Pascal，ACE vs P10，以及是否需要 ICE。

基于解剖情况设定预期：成功机会是否高，是否可能残余 MR，是否有 MS 风险；还要评估 leaflet integrity 和 commissural MR。

### 术中防守：通路、穿隔、气体和血栓

操作中要预判并发症。血管入路方面，可考虑 preclose 或 figure 8 缝合。

经房间隔穿刺方面，在穿刺前可考虑给予全剂量或半剂量全身抗凝；操作要慢，确认穿刺高度，确保至少在瓣环上方 4 cm。可考虑使用 RF 系统。为避免撕裂房间隔，应对球囊房间隔成形保持较低使用阈值。

操作过程中还要监测空气和血栓形成。

### 器械定位清单

释放前需确认转向、垂直取向、瓣叶插入、组织桥、MR 降低和器械稳定；避免过早释放和瓣叶夹持不足。

器械定位清单包括：device steering optimized，器械转向已优化；perpendicular orientation confirmed，确认与接合线垂直；leaflet insertion visualized，可视化瓣叶插入；adequate tissue bridge，有足够组织桥；MR reduction confirmed，确认 MR 减少；stable device before release，释放前器械稳定。

该页特别提醒要避免两件事：premature deployment，过早释放；inadequate leaflet capture，瓣叶夹持不足。

### 术中系统选择与抓瓣策略

应使用自己最熟悉、最有把握的 MV TEER 系统，但也要理解每个系统的优点和局限，包括 Abbott 与 Pascal。

如果可能，应限制抓瓣尝试次数。很小的取向调整也可能对 MR 产生明显影响。

目标是在不造成显著 MS 的前提下最大程度减少 MR。讲者提示，如果 MR 结果为 MILD，较高 MG 往往可以接受；但应避免中度 MR 加 MS，因为这是最差组合。

### 术后照护清单

立即评估包括：完成 TTE；记录残余 MR；记录跨瓣压差；排除心包积液；检查血管入路部位。

出院前应完成：明确抗栓策略；安排随访；完成患者教育。

### 常见并发症与预防

常见并发症对应预防动作：夹持确认、影像定位、压差监测、安全穿隔、超声引导入路和释放前稳定性测试。

| 并发症 | 预防措施 |
| --- | --- |
| SLDA | 确认瓣叶抓握 |
| Residual MR，残余 MR | 仔细影像和定位 |
| Mitral stenosis，二尖瓣狭窄 | 监测跨瓣压差 |
| Pericardial effusion，心包积液 | 安全经房间隔技术 |
| Vascular injury，血管损伤 | 超声引导入路 |
| Device embolization，器械栓塞 | 释放前进行稳定性测试 |

### 黄金规则：5 个不要急的时刻

讲者总结出 5 个“Don’t Rush”时刻：不要急于患者选择；不要急于经房间隔穿刺；不要急于瓣叶抓握；不要急于 clip release；不要急于术后评估。

核心提醒是：多数并发症发生前都有警示信号。

### 带回信息

认真计划可以预防大多数并发症。影像是操作成功的中心环节。结构化清单可提高一致性。团队沟通很重要。器械释放前要暂停并核实。

讲者最后一句话是：最安全的 mTEER，是每一个关键步骤都被有意确认过的 mTEER。
