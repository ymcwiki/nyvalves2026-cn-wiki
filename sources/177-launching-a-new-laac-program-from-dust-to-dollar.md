# 启动新的左心耳封堵项目：从尘土到美元

## 核心要点

* LAAC 项目失败往往不是临床技术失败，而是吞吐量、文档和转诊系统失败；应先搭系统，再排手术。
* LAAC 在美国已是标准治疗路径：截至 2025 年 12 月 NCDR 记录约 552,985 例 LAAO，涉及约 4,000 名医生和 917 家医院。
* 最高 ROI 的早期投入是 coordinator。它负责漏斗、检查、SDM、prior authorization 和可复制的文书包。
* 共同决策 SDM 是拒付高发点：必须由独立非介入医生完成，使用并记录循证工具，不能由植入者事后补写。
* 报销逻辑要锚定 NCD 20.34、CHA₂DS₂-VASc ≥3、不适合 OAC、登记入组，以及 2023 AF 指南 Class 2a 或 2b。
* 同日出院、ICE、患者分批和专用麻醉/影像资源，是扩容和利润率管理的关键。

### LAAC 是一门业务，不只是一个操作

讲者强调，LAAC 项目很少因为临床质量而失败，更多失败在 throughput、documentation 和 referrals。LAAC 现在已是标准治疗路径：NCDR 截至 2025 年 12 月记录约 552,985 例美国 LAAO 操作，约 4,000 名医生、917 家医院参与，并且仍在增长。

临床卓越只是基本要求，项目利润率取决于流程。讲者给自己的第一天建议是：先建立系统，再建立手术日程。联系邮箱为 [邮箱已隐去]。

### 先建转诊漏斗，再建导管室量能

手术量来自 referral engine，而不是导管室本身。显而易见的来源包括电生理/房颤门诊、初级保健、抗凝门诊。不能漏掉的来源包括神经科，尤其是 ICH 或卒中后患者；消化科；结构性心脏团队；以及跌倒门诊。

应利用 EMR 完成患者识别和 best-practice advisory。讲者给自己的建议是：把重点放在建立需求，而不仅是器械本身。并强调“Don’t make this mistake”，不要只做设备驱动而忽视需求建设。

### 团队就是项目本身

NCD 要求多学科团队，并要求已建立 SHD 或 EP 项目。操作者门槛包括完成厂家培训、已完成 ≥25 次 transseptal 操作，并维持 ≥25 次/2 年的持续操作量，其中包括 ≥12 次 LAAC。

讲者的建议是：最高 ROI 的岗位是 coordinator，应作为第 1 优先级招聘。

### 把术前工作前置

程序日之前应完成工作：由 APP/coordinator 与 implanter 进行 shared visits。CT 术前规划应把门诊访问和 CT 合并在同一次出行中。讲者建议“rent the CT bay”，以减少 no-show 并缩短从评估到手术的时间。

讲者的建议是：越靠近手术日才安排的检查，越可能成为取消手术的原因。

### 共同决策：拒付的头号陷阱

共同决策和抗凝禁忌文档表：每一种 OAC 不适合情境都要有可审计证据，并能转换为覆盖理由。

SDM 必须由独立、非介入医生完成。必须使用并记录循证工具。项目需要提前画出抗凝禁忌 playbook。SDM 不是 informed consent。由植入者撰写或未使用工具的 SDM，可能导致病例完成后被拒付。

| 禁忌示例 | 文档要点 | 用于覆盖理由 |
| --- | --- | --- |
| 1. 重大出血史 | 记录日期、类型（GI、ICH）、住院记录 | 证明抗凝造成直接伤害 |
| 2. 高跌倒风险 | PT/OT 记录、跌倒史、行动能力评估、虚弱指数 | 支持创伤相关出血风险 |
| 3. 用药依从性差 | 药房续方记录、EMR 记录 | 提示抗凝无效风险 |
| 4. 药物相互作用 | 用药清单、提供者记录 | 因多重用药构成禁忌 |
| 5. 肾功能或肝功能受损 | 实验室结果（eGFR、LFTs）、肾脏科/肝病科记录 | 限制 OAC 安全使用，改变药物代谢和安全性 |
| 6. 慢性 NSAID 使用 | 用药史、疼痛管理记录 | 增加 GI 出血风险 |
| 7. 职业风险 | 雇主信或提供者记录，例如重型机械操作 | 现实世界中的出血危险 |
| 8. 认知受损 | 神经科记录、MMSE 评分、照护者输入 | 抗凝自我管理不安全 |
| 9. 患者拒绝 | 共同决策记录、签署的同意文件 | 证明咨询后形成的知情选择 |
| 10. 既往出血性卒中 | 影像报告、神经科会诊 | 抗凝绝对禁忌，并与指南一致 |
| 11. HAS-BLED 高分 | EMR 中的评分计算 | 以数值量化出血风险 |
| 12. 合并症相关禁忌 | 肿瘤科、消化科或外科记录 | 具体情境下的风险放大 |

讲者的建议是：SDM 拒付的成本超过任何并发症。

### “美元”：为工作获得支付

NCD 20.34 的核心骨架是 CHA₂DS₂-VASc ≥3、不适合 OAC、登记入组。必要性说明要锚定 2023 年房颤指南：Class 2a（B-NR）或 Class 2b（B-R）。

干净的 medical-necessity note 和 prior authorization 应在病例前完成，而不是病例后补做。讲者的建议是：写给 auditor 看，而不仅是写给病历系统看。

### EP 路径：同期消融加 LAAC

对于 OPTION-eligible 患者，可在同一设置中完成节律控制和卒中预防，一次通路、一次恢复。CMS 现在支持这一模式：新的住院 MS-DRG 317，约 $44K，用于同期 LAAC + ablation；PFA、RF 和 cryo 均符合条件。

应为合适患者合并操作，而不是为了编码而合并。专业费用会被 MPPR 降低；门诊规则，尤其 ASC，仍在成熟。讲者的建议是：联合病例是临床意义和支付模型终于一致的地方。

### 标准化文书

三份表格构成一条流水线：Medical Necessity Assessment、Prior Authorization Cover Sheet 和 SDM attestation。需要 coordinator 拥有的 checklist，在每次 consult 前提前运行。通过这一点，手术量可以扩展，而拒付不会随之同步扩展。

讲者的建议是：可重复的 packet 胜过一次英雄式的 peer-to-peer。

### 手术效率

应把患者成组安排，专用 anesthesia 和 imaging，并保护 operator time。ICE 可减少对麻醉和影像排班的依赖，使病例可以安排在任何一天，而不只是在 block day。讲者建议先获得 TEE 熟练度，然后再用 ICE 提高灵活性。

讲者的建议是：半天或零散手术日比任何临床障碍都更限制手术量。

### 同日出院：释放容量

讲者建议把 same-day discharge 从安全顾虑重新定义为利润率和吞吐量杠杆。应早期筛选，优先安排 SDD 候选者，并在 24 小时内电话随访。没有占用的床位，就是项目增长所需的床位。

讲者的建议是：SDD 不是需要忍受的风险，而是腾出空间、扩展项目和最大化利润率的方法。

### 总结：项目从来不是器械，而是流程

最后一页总结：“It was never the device, It was always the process”。中文理解是，LAAC 项目的成败核心不是单个器械，而是转诊漏斗、团队角色、术前检查、SDM、报销文档、排班、麻醉影像资源、ICE 和同日出院组成的流程体系。
