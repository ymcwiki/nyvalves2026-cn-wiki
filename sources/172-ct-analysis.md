# CT 分析

## 核心要点

* Redo-TAV 的 CT 规划围绕四件事：第二枚 TAV 兼容性、植入位置、冠脉风险、第二枚 TAV sizing。
* TAV-in-TAV 不等同于 THV-in-SHV：第一枚 TAV 植入深度可变、瓣架结构不同，体内形态和尺寸会随瓣型、尺寸和位置变化。
* Neoskirt 由第一枚 TAV 固定瓣叶加第二枚 TAV 内裙形成，NSP 是 neoskirt 顶部；NSP 越高，冠脉阻塞或冠脉再进入风险越高，NSP 越低，leaflet overhang 风险越高。
* 冠脉风险从 NSP 高于冠脉开口时开始，需在 NSP 与 CRP 之间测量最窄 VTA；短瓣示例中 RCA 2.8 mm、LCA 5.5 mm，总体按中等冠脉风险处理。
* 病例中 84 岁女性 8 年前植入 Evolut R 29 mm，Redo-TAV 次日发生多发栓塞性卒中，超声峰速 4 m/s、平均跨瓣压差 41 mmHg，后续 CT 见 S3 瓣叶和 native sinus 血栓。

### 看起来简单，真的简单吗

透视图看似是简单 redo-TAV，但讲者提示需要系统 CT 规划。

本页问题为 “Appeared simple! But is it?”，通过两张透视图提示：redo-TAV 表面上可能只是把新瓣放入旧瓣，但实际风险来自旧瓣结构、植入深度、冠脉关系和第二枚瓣扩张形态。

### THV-in-SHV 的相对可预测性

外科生物瓣内植入 THV 时，外科瓣通常在瓣环，瓣高相似，尺寸较稳定。

THV in SHV 的特点是：瓣膜植入在 annulus；不同外科瓣高度相近；瓣膜尺寸维持。图中列出多种外科瓣示例，包括 Perimount、Magna、Mosaic、Hancock II、Mitroflow、Trifecta、Epic 等。

### THV-in-THV 与 THV-in-SHV 的差异

TAV-in-TAV 的深度、结构和体内尺寸变化使 CT planning 成为成败关键。

THV in SHV：瓣膜植入于 annulus，高度相似，尺寸维持。THV in THV：植入深度可变；瓣膜结构不同；体外和体内的形态、尺寸可不同。因此 CT planning 对病例选择和成功非常关键。

### TAV 器械上的标志点

TAV 节点和 commissure tab 可帮助定位尺寸、NSP 和冠脉关系。

TAV devices 上的 landmarks 告诉我们三类信息：重要尺寸、neo-skirt plane、冠状动脉关系。图中以 node # 0 至 6 标注瓣架，并指出 top of the commissure tab。

### Redo-TAV 规划四要素

第二枚 TAV 兼容性、植入位置、冠脉风险、第二枚 TAV sizing 四要素互相影响。

Redo-TAV 规划的 4 Key Elements 为：2nd TAV Compatibility、Implant Position、Coronary Risk、2nd TAV Sizing。

### TAV 兼容性：第一枚短或高

Index TAV 可为 short 或 tall，第二枚 TAV 的选择由组合决定。

本页定义 Index TAV 为 first implanted TAV，Second TAV 为 new TAV inside index TAV。第一枚 TAV 可为 short，也可为 tall，因此会形成四类组合。

### TAV-in-TAV 四种组合

Short in Short、Tall in Short、Short in Tall、Tall in Tall 四种 redo-TAV 组合。

四种组合分别为 Short in Short、Tall in Short、Short in Tall、Tall in Tall。每种组合的 neoskirt 高度、冠脉风险和第二枚 TAV sizing 策略不同。

### Neoskirt 与 NSP

Neoskirt 是 ViV 后形成的管状结构，NSP 是 neoskirt 顶部。

Neoskirt 指 ViV 后产生的 tube graft，neoskirt plane (NSP) 是 neoskirt 的 top。图中展示 Short in Short 与 Tall in Tall 的 NSP，Tall in Tall 例中 NSP 约在 Node 6.5。

### NSP 的组成

NSP 由 index TAV 被固定的瓣叶和第二枚 TAV 的内裙共同决定。

NSP 来源于 index TAV pinned leaflets 与 inner skirt of second TAV 的组合。Short in Short、Tall in Tall、Tall in Short 的 NSP 水平不同，决定潜在冠脉风险平面。

### Short-in-Tall 中 NSP 可调整

Short in Tall 可通过第二枚短瓣位置调整 NSP 至 Node 6、Node 5 或 Node 4。

对于 Short in Tall，NSP level 可以调整。图中展示将短瓣放在 tall index TAV 中时，NSP 可位于 Node 6、Node 5、Node 4。

### Leaflet overhang 与 NSP

NSP 越低，leaflet overhang 越多；图中示例为 4%、38%、90%。

Leaflet overhang 与 NSP 位置相关。图中以 Node 6、Node 5、Node 4 的位置展示 overhang 比例约为 4%、38%、90%。底部说明 acceptable hemodynamics 包括 mean gradient 和 regurgitant fraction。

### 如何选择植入位置

植入位置是在冠脉阻塞/冠脉再进入风险与 leaflet overhang 之间取平衡。

位置从 Node 6.5、Node 6、Node 5 到 Node 4。较高位置增加 coronary obstruction or access 风险；较低位置增加 leaflet overhang。

### Short-in-Short 的低位植入问题

Short in Short 中较低植入会造成 leaflet overhang。

Short in Short 的 NSP 在旧短瓣顶部附近。若第二枚瓣植入过低，图中提示会发生 lower implant with leaflet overhang。

### 第二枚 TAV sizing 的目标

Sizing 要同时满足固定和功能，避免过小或过大。

Fixation 方面不能太小，否则有 embolization risk。Function 方面不能太大，否则会 underexpansion，并增加 valve thrombosis risk。

### 第二枚 TAV sizing 的三个来源

体内 CT sizing、体外 bench sizing 和 native annulus area 三种 sizing 依据。

In-vivo CT sizing 是测量真实 index TAV stent frame。In-vitro bench sizing 使用 index 与 second TAV 的匹配对，表中 short TAV #20、#23、#26、#29 对应 tall TAV #23、#26、#29、#34。Native annulus area 则使用 pre-index TAVI data。

### 为什么体内 sizing 可能更好

欠扩张和不均匀扩张常见，且尺寸可随 TAV 尺寸和位置改变。

理由有两点：第一，under-expansion 与 uneven expansion 常见；第二，尺寸可随 TAV size 和 position 改变。示意图比较 Evolut 26 与 Evolut 34，并标出 Node 6 与 Node 4。S3 size 在某些情形可能不变，在另一些情形可能变化。

### 不同失败 TAV 的 sizing 策略

短瓣和高瓣的第二枚 TAV sizing 测量层面不同。

Failed short TAV：取 3 个层面 面积平均，即 outflow、waist 或 narrowest、inflow。Failed tall TAV：取 NSP 和其下方 3 个 nodes 的面积平均，或取其下方 2 个 nodes。

### 冠脉风险类型

冠脉风险包括直接冠脉阻塞和 sinus sequestration。

Coronary Risk Assessment 包括 Direct Coronary Obstruction 和 Sinus Sequestration。直接阻塞来自被固定瓣叶和新瓣架对冠脉开口的覆盖；sinus sequestration 则与 NSP 和 STJ 之间形成封闭窦腔有关。

### 风险从 NSP 高于冠脉开口开始

当 NSP 位于冠脉开口以上时，需要进入冠脉风险评估。

CRP 是 coronary risk plane，NSP 是 neoskirt plane。若 NSP 低于冠脉开口，图中标注为 low risk of coronary；若 NSP 高于冠脉开口，则需要 coronary assessment。

### 在哪里测量冠脉风险

在 NSP 与 CRP 之间寻找最窄 valve-to-aorta 距离。

分析位置为 NSP 和 CRP 之间的 narrowest valve-to-aorta (VTA) distance。图中阈值表达为：VTA >4 mm 倾向低风险，2 至 4 mm 为中间区域，<2 mm 倾向高风险。

### 如何测量 VTA

BEV 可扩张 index TAV，因此需用第二枚 TAV 的 virtual circle。

BEV could expand index TAV and increase coronary risk，因此应使用 chosen TAV size 的 virtual circle。当第二枚 TAV 的 virtual circle 位于 index TAV 外侧时，VTA 从第二枚 TAV virtual circle 至 ostium 测量；当 virtual circle 在 index TAV 上或内侧时，VTA 从 index TAV 至 ostium 测量。

### 失败高瓣的四要素

高瓣失败时，兼容性、植入位置、冠脉风险和 sizing 更依赖第二枚 TAV 类型及位置。

Failed tall valves 中，第二枚可为 short-in-tall 或 tall-in-tall。Coronary Risk 取决于 implant position 与 2nd TAV size。2nd TAV sizing 取决于 2nd TAV type 和 implant position：短瓣策略为 NSP 与其下 3 nodes 面积平均；高瓣策略为与 index TAV 相同或小一号。

### 术前 CT checklist

Redo-TAV 术前 CT checklist。

Key Home: Pre-procedure CT checklist 包括三步。第一，identify index failed TAV：确认 valve model and size，理解 2nd TAV options，理解 2nd TAV implant position and potential NSP。第二，2nd TAV sizing：测量 In-vivo CT sizing。第三，assess coronary risk：评估 NSP 与 coronaries 的相对关系，测量 narrowest VTAoS；当 NSP above STJ 时测 VTSTJ，当 NSP below STJ 时测 VTAoS at NSP。
