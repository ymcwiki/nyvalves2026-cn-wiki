# 后叶二尖瓣修复标准化人工腱索技术的早期结果

## 核心要点

* 该研究评估一种“不做术中测量”的后叶二尖瓣修复策略：所有后叶病变均使用预先测量的 16 mm 固定长度人工腱索，并同步行瓣环带成形。
* 队列为 149 例连续后叶病变二尖瓣修复患者，主要为低危、退行性人群：男性 69.1%，平均年龄 62.7±10.7 岁，STS 死亡风险 1.67±1.03%。
* 病因以退行性为主，占 95.7%；感染性心内膜炎占 4.3%；76.5% 的修复为微创完成，固定长度策略在微创和常规手术中同样应用。
* 后叶节段分布中，孤立 P2 占 51.7%，P1-P2 占 20.1%，P2-P3 占 19.5%，多节段组合也使用同一固定长度。
* 出院超声显示无或微量 MR 占 85.9%，无中度或重度 MR；平均跨二尖瓣压差 2.81±1.27 mmHg，未提示固定长度修复造成二尖瓣功能性狭窄。

### 背景与目的

二尖瓣修复中人工腱索长度测量仍然有较高技术要求。不同外科医生之间、不同病例之间的手术实践存在明显差异。后叶病变是最常见的病变基础，同时也是最难标准化的场景。该页提出的问题是：能否把后叶修复中的人工腱索长度用一个固定值统一起来，并保持早期修复质量。

### 研究策略

研究采用标准化 16 mm 固定长度人工腱索，应用于每一例后叶二尖瓣修复，且不做术中长度测量。研究目标是评估这种标准化策略在后叶二尖瓣病变中的安全性、通用性和早期超声心动图结果。

### 方法：连续病例、固定腱索、瓣环带与出院超声

方法页用四个模块概括研究设计：149 例后叶病变修复、16 mm 固定人工腱索、瓣环带成形、出院早期超声结局。

方法包括 149 例患者，均为后叶病变的连续二尖瓣修复病例。每例均使用 16 mm 人工腱索，这是预先测量、固定长度的策略。所有患者在人工腱索植入的同时接受 band annuloplasty。早期超声结局在出院时评估，重点观察残余二尖瓣反流 MR 和跨二尖瓣压差。

| 方法模块 | 具体内容 |
| --- | --- |
| 病例数 | 149 例后叶病变连续二尖瓣修复患者 |
| 人工腱索 | 16 mm 预先测量固定长度，每例均使用 |
| 瓣环处理 | 所有患者均在人工腱索植入同时行 band annuloplasty |
| 早期超声终点 | 出院时残余 MR 和跨二尖瓣压差 |

### 基线特征：低危、退行性为主

基线人群以退行性病变为主，手术风险低，多数采用微创入路。

队列被描述为低风险、以退行性病变为主的人群。男性占 69.1%，对应 n=103。平均年龄 62.7±10.7 岁。STS 预测死亡率 1.67±1.03%。基线 EF 为 62.4±8.86%。LVEDD 为 54.9±13.7 mm。病因方面，退行性病变占 95.7%，感染性心内膜炎占 4.3%。手术入路方面，76.5% 的修复以微创方式完成；标准化长度在微创和常规病例中以相同方式应用。

### 一个长度覆盖所有后叶节段

后叶病变节段分布：孤立 P2 最常见，多节段组合也以同一固定腱索长度处理。

柱状图显示，孤立 P2 占 51.7%，是经典的 P2 主导脱垂模式，也是超过半数病例的病变类型。P1-P2 占 20.1%，P2-P3 占 19.5%，孤立 P1 占 3.4%，孤立 P3 占 3.4%，P1-P2 与 P2-P3 组合占 2.0%。右侧强调三点：该技术可用于多节段病变，P1-P2 与 P2-P3 组合使用同一固定长度；不需要自定义 sizing；不需要针对不同节段调整腱索长度。

| P2-P3 | 19.5% |
| 孤立 P1 | 3.4% |
| 孤立 P3 | 3.4% |
| P1-P2 与 P2-P3 组合 | 2.0% |

### 出院早期超声结局

出院时残余 MR 以无和微量为主，跨二尖瓣平均压差低。

出院时残余二尖瓣反流：无 MR 为 26.2%，n=39；微量 MR 为 59.7%，n=89；轻度 MR 为 14.1%，n=21。无或微量 MR 合计 85.9%。中度或重度 MR 为 0%。平均跨二尖瓣压差为 2.81±1.27 mmHg，幻灯片注明没有固定长度修复导致功能性狭窄的证据。出院 EF 为 56.9±11.7%，早期收缩功能保留；基线 EF 为 62.4%。

### 结论

标准化人工腱索技术是后叶二尖瓣病变修复的一种可靠策略。该方法可跨越不同后叶节段、不同病因，并适用于微创与常规两类手术入路。出院超声未见超过轻度的残余 MR，跨二尖瓣压差低。作者提出，这一策略也可作为标准化经导管人工腱索方法的概念基础。
