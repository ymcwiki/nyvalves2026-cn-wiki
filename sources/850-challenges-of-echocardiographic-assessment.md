# 三尖瓣反流超声评估挑战

## 核心要点

* 三尖瓣反流（TR）不能只靠彩色多普勒判断，讲者把挑战分为定义严重程度、定义机制和预测“高风险”三部分。
* 2017 ASE流程要求综合VC、PISA、CW形态、肝静脉流、RV/RA大小和定量指标；严重TR阈值包括VC ≥0.7 cm、EROA >0.4 cm²、RVol ≥45 mL。
* “严重”TR已扩展为severe、massive、torrential三级，二维EROA阈值为40-59、60-79、>80 mm²，3D VCA阈值则为75-94、95-114、≥115 mm²。
* AFTR、VFTR、CIED相关TR和原发性TR的定义并不统一，机制判断需要同时看瓣叶牵拉/限制、RA/TA扩张、RV扩张与RV功能。
* 严重TR中RV功能测量受角度、负荷、图像质量和纵向功能偏倚影响，TTVR后可能出现RV失代偿，病例中TAPSE降至0.6 cm，FWS为-9.1%。
* 患者选择必须综合解剖特征、瓣膜团队经验、患者期望和替代治疗，超声贯穿所有环节。

### 挑战一：定义TR严重程度

本页列出超声评估的第1个挑战：定义三尖瓣反流（TR）的严重程度。这为随后关于彩色多普勒、ASE定量流程和严重TR分级扩展的内容铺垫。

### 彩色多普勒不够

从轻度TR到torrential TR的彩色多普勒示例被红色禁止符号覆盖，提示单靠彩色喷流面积不足以分级。

本页标题为“Color Doppler is not Enough”。图中从左到右标注Mild TR (1+)、Moderate TR (2+)、Severe TR (3+)、Massive TR (4+)和Torrential TR (5+)。大红色禁止符号覆盖在图上，强调TR分级不能只依靠彩色多普勒喷流面积或主观印象。

### 2017 ASE慢性TR定量流程

2017 ASE慢性三尖瓣反流定量流程，按轻度、可能中度、重度和不确定TR进行分流。

2017 ASE指南建议先判断TR是否符合多数轻度或重度特异标准。若符合轻度标准，走向轻度TR；若符合重度标准，走向重度TR；若只满足少数标准或处于中间值，则判断为可能中度TR，并建议测量VC，必要时尽可能进行定量PISA。

| 类别 | 特异标准 |
| --- | --- |
| 轻度TR特异标准 | 薄而小的中心彩色喷流；VC宽度<0.3 cm；Nyquist 30-40 cm/s时PISA半径<0.4 cm；CW喷流不完整或较淡；肝静脉血流以收缩期为主；三尖瓣A波主导流入；RV/RA正常。 |
| 重度TR特异标准 | 瓣环扩张且瓣叶不对合或瓣叶连枷；大中心喷流>50% RA；VC宽度≥0.7 cm；Nyquist 30-40 cm/s时PISA半径>0.9 cm；CW喷流致密、三角形或正弦波样；肝静脉收缩期反向；RV扩张但功能保留。 |

| 最终分级 | VC宽度 | EROA | 反流量RVol |
| --- | --- | --- | --- |
| 轻度TR | <0.3 cm | <0.2 cm² | <30 mL |
| 中度TR | 0.3-0.69 cm | 0.2-0.4 cm² | 30-44 mL |
| 重度TR | ≥0.7 cm | >0.4 cm² | ≥45 mL |

若TTE质量差、对多普勒定量参数信心低，或定量与定性参数及临床资料不一致，则归为不确定TR，并考虑进一步检查：TEE或CMR定量。引用来源为J Am Soc Echocardiogr. 2017 Apr;30(4):303-371。

### 严重TR扩展为三级

二维超声参数将严重TR扩展为severe、massive和torrential三个等级。

| 二维超声参数 | Mild | Moderate | Severe | Massive | Torrential |
| --- | --- | --- | --- | --- | --- |
| EROACONV (mm²) | <20 | 20-39 | 40-59 | 60-79 | >80 |
| RegVolCONV (mL) | <30 | 30-44 | ≥45 | - | - |

本页说明传统“严重TR”进一步扩展为Severe、Massive和Torrential，二维EROA阈值分别为40-59 mm²、60-79 mm²和>80 mm²。

### 二维超声局限

在同一二维阈值表下，讲者列出二维超声的阴影、图像质量、角度、容量状态和房颤逐搏变异局限。

本页重复上一页的二维分级阈值，并补充二维超声局限：声影、图像质量、角度依赖、容量状态依赖，以及房颤时逐搏变异。这些因素会影响PISA、VC、反流量和EROA的稳定性。

### TR口非圆形：3D MPR证据

二维计算默认反流口为圆形，但3D MPR显示三尖瓣反流口常为非圆形。

二维超声计算常假设反流口为圆形，但三维多平面重建（3D MPR）显示TR反流口可为非圆形，因此二维VC和PISA可能低估或误判严重程度。

| 变量 | Severe | Massive | Torrential |
| --- | --- | --- | --- |
| VC，双平面 | 7-13 mm | 14-20 mm | ≥21 mm |
| EROA，PISA | 40-59 mm² | 60-79 mm² | ≥80 mm² |
| 3D VCA或定量EROA | 75-94 mm² | 95-114 mm² | ≥115 mm² |

缩写：VC为vena contracta，EROA为有效反流口面积，3D VCA为三维vena contracta面积。引用为RT Hahn, JL Zamorano. Eur H J CV Imaging 2017;18(12):1342-43。

### 不同模态下严重TR阈值不同

二维超声、三维超声和CMR用于TR分级的阈值并不相同。

| 参数 | Mild | Moderate | Severe | Massive | Torrential |
| --- | --- | --- | --- | --- | --- |
| **2D Echocardiographic Parameters** | | | | | |
| EROACONV (mm²) | <20 | 20-39 | 40-59 | 60-79 | >80 |
| RegVolCONV (mL) | <30 | 30-44 | ≥45 | - | - |
| **3D Echocardiographic Parameters** | | | | | |
| 3D VCA (mm²) | 25-49 | 50-74 | 75-94 | 95-114 | ≥115 |
| RegVol3DVCA (mL) | <30 | 30-44 | 45-59 | 60-74 | ≥75 |
| RegFrCORR (%) | <16 | 16-26 | 27-49 | >49 | - |
| **CMR Parameters** | | | | | |
| RegVolCMR (mL) | <30 | 30-44 | 45-59 | 60-74 | ≥75 |
| RegFrCMR (%) | <15 | 16-25 | 26-48 | 49-57 | ≥58 |

本页核心信息是：同样称为“严重TR”，不同成像模态使用的切点不一致。因此在病例讨论和介入选择中，需要明确阈值来自二维、三维超声还是CMR。

### 挑战二：定义机制

本页把超声评估挑战扩展为两项：第1，定义TR严重程度；第2，定义TR机制。后续页面分别讨论不同疾病过程、心房功能性TR、心室功能性TR和CIED相关TR。

### 不同疾病过程导致TR

TR机制分类包括心房功能性、心室功能性、CIED相关和原发性TR，各自影像和解剖特征不同。

本页根据Hahn等2022年综述显示，TR可来自不同疾病过程：功能性/继发性TR中的心房型和心室型、CIED相关TR，以及器质性/原发性TR。原发性TR进一步示意为Prolapse (I)和RHD (IIIA)。

| 参数 | Atrial FTR | Ventricular FTR | CIED-Related | Primary TR: Prolapse (I) | Primary TR: RHD (IIIA) |
| --- | --- | --- | --- | --- | --- |
| Leaflet Tethering | - | +++ | ++ | - | - |
| Leaflet Restriction | - | Systole | Systole/Diastole | - | Diastole |
| RA/TA Dilatation | +++ | ++ | +/- | ++ | ++ |
| RV Dilatation | +/- | +++ | +/- | +/- | +/- |
| RV Dysfunction | +/- | +++ | +/- | +/- | +/- |

引用：Hahn R et al. Eur Heart J Cardiovasc Imaging, Volume 23, Issue 7, July 2022, Pages 913-929。

### 心房功能性TR的多个定义

AFTR定义存在机制、病因、Carpentier、超声表型、Schlotter 2022、RA/RV比值和排除式定义等多个版本。

| 定义类别 | 核心标准 |
| --- | --- |
| Mechanistic | RA enlargement + annular dilation |
| Etiologic | AF/atrial cardiomyopathy without ventricular disease |
| Carpentier | Type I TR with normal leaflet motion |
| Echo phenotype | Large RA, large annulus, minimal tethering |
| Schlotter 2022 | Tenting ≤10 mm + RV diameter ≤38 mm + LVEF ≥50% |
| RA/RV ratio | RA/RV area ratio >1.132 |
| Exclusion-based | RA dilation with absence of RV remodeling/PH |

该表说明AFTR不是单一概念，不同研究或团队可能依据心房扩大、瓣环扩张、房颤/房性心肌病、低牵拉或排除RV重构/肺高压来定义。

### 心室功能性TR的多个定义

VFTR定义强调RV重构、扩张、功能障碍、瓣叶牵拉、tenting和对合间隙。

| 定义类别 | 核心标准 |
| --- | --- |
| Mechanistic | RV remodeling with leaflet tethering |
| Consensus / Contemporary | RV dilation or dysfunction causing malcoaptation |
| RV-overload | Pressure/volume overload-induced RV geometric distortion |
| Echocardiographic | Leaflet tethering, tenting, and coaptation gap |
| Etiologic | TR secondary to RV disease |
| Pathoanatomic | Papillary muscle displacement and tethering |
| Carpentier | Multiple depending on mechanism |

### CIED-TR的多重机制

三维和二维TEE示例显示CIED导线可通过多种方式影响三尖瓣结构和反流。

本页标题为“Multiple Mechanisms of CIED-TR”。左侧三维TEE图像和右侧二维TEE图像显示植入式心脏电子装置（CIED）导线与三尖瓣结构关系复杂。导线相关TR可能来自瓣叶受限、穿越或压迫瓣叶、干扰对合、诱发瓣下结构牵拉，以及长期导致局部结构改变。该页强调不能把所有CIED相关TR简化为一个机制。

### 挑战三：预测高风险

本页把超声评估挑战完整列为三项：第1，定义TR严重程度；第2，定义机制；第3，预测“高风险”。后续内容聚焦TTVR后RV功能和右心衰风险。

### 严重心房功能性TR病例

严重心房功能性TR病例，显示心腔扩大、TR喷流和RV功能参数。

本页展示严重心房功能性TR（Severe atrial functional TR）病例。影像包含二维心尖四腔观、彩色多普勒TR喷流和RV strain/功能分析图。右下角列出RV参数：RV FAC 34%，RV TAPSE 1.3 cm，RV FW GLS -20%，RVSP 42 mmHg。这些数字提示在重度TR背景下，RV功能评估需谨慎，因为负荷状态会影响外观和指标。

### TTVR后RV功能障碍

TTVR前后RV GLS对比：术前-14.3%，术后-7.0%，提示RV收缩功能下降。

本页题为“Post TTVR, RV dysfunction”。左图标注术前RV GLS为-14.3%，右图标注术后RV GLS为-7.0%。数值绝对值降低，提示三尖瓣反流被处理后，RV面临新的后负荷环境，潜在RV功能障碍变得更明显。

### 术后第1天右心衰

术后第1天出现右心衰，strain和TAPSE显示RV功能严重受损。

本页标题为“Post-op Day 1 -> Right Heart Failure”。右侧图像标注FR 75 fps、HR 85 bpm，并给出RV功能参数：GSv -10.0%，FWS -9.1%，TAPSE 0.6 cm。这些结果提示术后第1天出现明显右心功能衰竭，TAPSE和自由壁应变均明显异常。

### TTVR治疗窗口概念框架

图形摘要显示TTVR最佳治疗窗口：TR进展早期受RV扩张影响，晚期受RV功能障碍限制。

本页为Eur Heart J图形摘要，标题为“A conceptual framework illustrating the optimal therapeutic window of TTVR”。纵轴为获益效应（Beneficial effect，Win ratio），范围从0.0到3.0，刻度间隔0.5。横轴为TR等级：Moderate TR、Severe TR、Massive TR、Torrential TR。曲线从中度TR时接近0逐渐上升，在severe/massive区间达到高峰，峰值约超过2.0，随后在torrential和RV dysfunction阶段下降接近0。图中左侧箭头标注RV dilatation，右侧箭头标注RV dysfunction。引用为Eur Heart J, ehaf954, https://doi.org/10.1093/eurheartj/ehaf954。

### 严重TR中RV功能测量局限

不同RV功能指标在严重TR中受角度、负荷、图像质量和评估范围限制。

| 指标 | 超声局限 | 对RV功能评估的限制 |
| --- | --- | --- |
| RV linear diameters | 角度依赖；负荷依赖；图像依赖 | 只观察RV心尖 |
| RV FAC | 角度依赖；负荷依赖；图像依赖 | 只观察RV心尖 |
| TAPSE / s’ TD Velocity | 角度依赖；负荷依赖；图像依赖 | 只评估纵向功能 |
| RV GLS | 不依赖角度；负荷依赖；较少图像依赖 | 只评估纵向功能 |
| RVEF (3D) | 图像依赖 | 反映整体RV功能，但严重TR时可能被假性抬高 |
| RV-PA Uncoupling | 图像依赖；负荷依赖 | 反映“后负荷储备”，依赖PASP准确评估 |

### RV-PA耦合评估

RV-PA耦合指RV在后负荷升高时维持收缩功能的能力，可作为后负荷储备评估。

RV-PA coupling定义为：在后负荷增加时，右心室仍能以保留的RV收缩功能进行代偿的能力。讲者也称其为“Afterload Reserve”。右侧列出RV-PA耦合评估方式：

|  |
| --- |
| TAPSE / RVSP |
| TAPSE / mean PAP |
| RV FAC / mean PAP |
| RV ESV / SV (3D TTE) |
| RV Strain / RVSP |
| RVEF / RVSP (3D TTE) |
| TAPSE / PVR |

### RV功能、TR与瓣环扩张循环

RV功能障碍、TR和进行性瓣环扩张形成相互促进的循环，同时提出RV衰竭和无效治疗预测问题。

本页用循环箭头表示RV Dysfunction、Tricuspid Regurgitation和Progressive Annular dilation之间的相互作用。右侧提出三个临床问题：如何在严重TR中识别RV dysfunction？什么是“轻度”或“重度”RV dysfunction？能否预测RV failure或futility？这些问题直接关系到介入时机和患者选择。

### 超声在患者选择中至关重要

患者选择需整合解剖特征、瓣膜团队经验、患者期望和替代治疗，超声贯穿全程。

本页总结超声在患者选择的所有方面都很关键。右侧循环图列出四个组成部分：Anatomic Features、Valve Team Experience、Patient expectations和Alternative treatments。含义是，超声不仅用于分级，还用于机制判断、解剖筛选、治疗策略讨论和风险预测。
