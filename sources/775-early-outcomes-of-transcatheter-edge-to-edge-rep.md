# MitraClip G5 经导管缘对缘修复的早期结局：MITRAGEN5 注册研究

## 核心要点

* MITRAGEN5 是研究者发起、行业独立的真实世界多中心注册研究，纳入 578 例连续患者，来自 21 个中心，时间为 2025–2026 年。
* 主要终点 MVARC 技术成功率为 95.7%；组成项包括无术中死亡 578/578，100%、无急诊手术或器械/入路相关再干预 575，99.5%。
* 出院时残余 MR ≤1+ 的总体比例为 82.2%；复杂性分层样本中，残余 MR ≤1+ 为总体 272/334，82%，Replacement? 组较低，为 52/73，71%。
* 出院时 MPG <5 mmHg 的总体比例在复杂性分层样本中为 273/334，83%；各组为非复杂 85%、复杂 87%、非常复杂 90%、Replacement? 74%。
* 院内死亡率为 1.8%。研究局限包括观察性真实世界设计、本地超声评估且无核心实验室、随访不完整所以尚无 30 天数据。

### 研究背景

**M-TEER** 已经是有症状重度二尖瓣反流（**MR**）且外科手术风险不可接受患者的一种成熟治疗。M-TEER 技术正在经历迭代改进。近期，第五代 **MitraClip（G5）** 被引入，其输送平台进行了增强。

### 研究目的

研究目的为评估使用 **MitraClip G5** 进行治疗时的真实世界多中心数据，重点关注操作结局和院内结局。

### 研究设计

MITRAClip GENeration 5 Mitral Valve Transcatheter Edge-to-Edge Repair Registry 的研究设计，地图显示多中心分布。

研究全名为 **MITRAClip GENeration 5 Mitral Valve Transcatheter Edge-to-Edge Repair Registry**。这是一项研究者发起、行业独立的注册研究。纳入 N = 578 例连续患者，来自 21 个中心，研究时间为 2025–2026 年。所有患者接受使用 **MitraClip G5** 的 M-TEER。

### 研究终点

主要终点为 **MVARC 定义的技术成功**。次要终点包括三项：出院时残余 MR ≤1+；出院时残余平均跨瓣压差（**MPG**）<5 mmHg；院内死亡率。

### 复杂性标准：从 Repair 到 Replacement

基于 Hausleiter 等的复杂性标准：解剖适合度从适合 M-TEER 到可能需要置换，中心经验需求逐步升高。

图示从左到右呈现 **Repair!** 到 **Replacement?** 的谱系，上方白线表示 **Anatomical suitability for M-TEER** 与 **Centre experience** 的关系：越往右，M-TEER 解剖适配性下降，对中心经验要求增加，最终进入偏向置换的区域。引用来源为 **Hausleiter et al. EuroIntervention 2023;23;18(12):957-976**。

| 类别 | 标准 |
| --- | --- |
| Non-complex，Ideal for M-TEER | 中央病变；无钙化；MVA >4.0 cm2；后叶 >10 mm；tenting height <10 mm；flail gap <10 mm；flail width <15 mm。 |
| Complex，Suitable for M-TEER | 孤立交界区病变（A1/P1 或 A3/P3）；MVA 3.5–4.0 cm2；后叶长度 7–10 mm；tenting height >10 mm；flail width >15 mm；flail gap >10 mm。图中同列另有若干连字符空白条目。 |
| Very complex，Challenging for M-TEER | 交界区病变伴多个反流束；MVA 3.0–3.5 cm2；后叶长度 5–7 mm；Barlow 病；裂隙；外科瓣环成形失败。图中同列另有若干连字符空白条目。 |
| Criteria favouring replacement，M-TEER hard or impossible | MVA <3.0 cm2；相关二尖瓣狭窄，平均压差 >5 mmHg；后叶 <5 mm；夹持区钙化；深反流裂隙；瓣叶穿孔；多发或宽反流束；风湿性二尖瓣狭窄。图中开头另有一个连字符空白条目。 |

### 研究局限

局限包括三点。第一，本研究为观察性真实世界注册研究。第二，超声心动图在各中心本地完成，没有核心实验室。第三，随访尚不完整，因此目前还没有 30 天数据。

### 结论

结论为：使用第五代 MitraClip 进行 M-TEER 是有效的，表现为出院时残余 MR ≤1+ 比例较高，且该比例受复杂性影响；出院时 MPG <5 mmHg 的比例较高；院内死亡率较低，为 1.8%。研究仍需要更长期随访数据。
