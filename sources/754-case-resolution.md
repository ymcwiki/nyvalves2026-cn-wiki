# 病例解决方案

## 核心要点

* 这是一例高龄、极高外科风险、多瓣膜病患者，基线EF仅15%–20%，STS手术风险为16.2%。
* 心脏团队没有按常规先处理主动脉瓣，而是先以单个MitraClip XTW完成M-TEER，将重度功能性MR降至轻度。
* M-TEER后第1天TTE显示低流低压差AS被进一步“显影”：平均跨瓣压差14.9 mmHg、峰速2.8 m/s、AVA 0.9 cm²、DI 0.19。
* 两周后患者呼吸困难和下肢水肿改善，但仍疲乏，复查提示重度AS，因此进入第二阶段TAVR，植入23 mm Sapien 3。
* TAVR后2个月，患者呼吸显著改善，EF升至20%–25%，无AS/AI，残余MR为轻至中度。

### 可选策略

本页列出治疗选项：1，继续药物治疗；2，外科主动脉瓣置换加二尖瓣置换，即AVR+MVR；3，先TAVR；4，先M-TEER；5，TMVR。

### 心脏团队决策

心脏团队评估认为患者为高危手术人群，STS风险为16.2%，并合并低EF和高龄。主动脉瓣病变为低流低压差AS，严重程度介于中度至重度之间，钙化积分为816。二尖瓣病变为重度功能性MR。最终计划选择二尖瓣介入，即M-TEER。

### 术后第1天TTE定量结果

| 项目 | 结果 |
| --- | --- |
| 左室EF | 15% |
| 主动脉瓣 | 重度AS；平均跨瓣压差14.9 mmHg；峰值速度2.8 m/s；AVA 0.9 cm²；AVAi 0.5；DI 0.19。 |

### 两周随访：下一步问题

两周随访时，患者呼吸困难改善，下肢水肿改善，但仍有疲乏。利尿剂使用呋塞米40 mg bid。复查TTE仍提示重度AS。幻灯片以“Now what?”提出下一步策略问题。

### TAVR治疗

第二阶段TAVR，植入23 mm Sapien 3瓣膜。

本页显示TAVR操作，治疗选择为Sapien 3 #23。影像中可见瓣膜递送和释放位置，策略是在M-TEER稳定MR后处理主动脉瓣，以降低急性后负荷变化带来的风险。

### 两个月随访结果

TAVR后两个月随访，临床症状和TTE指标均改善。

TAVR后2个月，患者自述呼吸较TAVR前显著改善，感觉“像换了个人”。她恢复为家人做饭，并能进行短距离步行。复查TTE显示EF为20%–25%，无AS/AI，残余MR为轻至中度。

### 病例总结

基线情况为86岁女性，EF 15%–20%，外科风险不可接受，合并重度功能性MR和中度至重度AS。传统思路通常认为应先TAVR，通过降低后负荷改善继发性MR。本病例采用分阶段策略：第一阶段用单枚MitraClip XTW行M-TEER，残余MR降至轻度。随后出现血流动力学“显影”效应，失去MR这个“pop-off valve”后，真实AS严重程度暴露，表现为Vmax 2.8 m/s、AVA 0.9 cm²、DI 0.19。第二阶段行TAVR，植入Sapien 3 #23，避免急性后负荷不匹配。最终结果为EF 20%–25%、无AS/AR、MR轻至中度，并获得显著功能恢复。
