# 结构性心脏介入时代心脏和主动脉假性动脉瘤经皮封堵技术指南

## 核心要点

* 本研究为 Mayo Clinic Rochester 单中心回顾性描述研究，时间为 2010-01-01 至 2026-02，纳入 27 例尝试经皮封堵心脏或主动脉假性动脉瘤患者。
* 队列中位年龄 73 岁，IQR 64-82；男性 52%，即 14/27。病变位置为 LV 16 例（59%）、主动脉 9 例（33%）、RV/RVOT 2 例（7%）。
* 技术成功率 100%（27/27）；分期或改用替代入路 7%（2/27）；围术期并发症 11%（3/27）。
* 无手术相关死亡、无卒中、无急诊转外科、无冠脉阻塞、无人工瓣损伤；30 天生存 96%（26/27），1 年生存 89%（24/27）。
* 器械选择按解剖定制：Amplatzer Vascular Plug II 多用于窄到中等颈部，Amplatzer Septal Occluder 多用于宽颈需更大覆盖/锚定，弹簧圈用于小而离散且邻近关键结构的交通口。

### 背景

心脏和主动脉假性动脉瘤是心脏外科手术和结构性心脏介入后的少见但严重并发症。许多患者因既往胸骨切开、合并症或手术风险过高而不是理想外科候选者。经皮封堵提供了一种侵入性较低的选择。

### 研究目标

研究目标：总结单中心经验、解剖驱动技术，以及成功率、安全性和随访结局。

本研究想补充三方面内容：第一，单中心病例系列的经验；第二，解剖驱动的技术选择；第三，成功、安全和随访结局。

### 研究方法

研究设计、纳排标准和数据收集分析框架。

研究为回顾性、单中心、描述性研究，地点为 Mayo Clinic Rochester，时间范围为 2010-01-01 至 2026-02。病例识别通过系统性电子病历检索关键词“pseudoaneurysm”。研究队列为 27 例尝试经皮封堵的患者。数据来源为详细人工病历审阅。

纳入标准：影像确认的左室、右室/RVOT 或主动脉假性动脉瘤，并尝试经皮封堵。排除标准：外科修补或保守治疗。收集临床、影像、操作、并发症和随访数据。分析仅采用描述性统计，连续变量报告中位数（IQR），分类变量报告 n（%）。研究获得 IRB 批准。

### 操作结局

技术成功、替代入路和围术期并发症。

|  |  |
| --- | --- |
| 技术成功 | 100%（27/27） |
| 分期或替代入路 | 7%（2/27） |
| 围术期并发症 | 11%（3/27） |
| 入路血管损伤 | 2 例 |
| 器械移位 | 1 例 |

替代入路的 2 例包括：1 例初次逆行尝试时导管接合不稳定；1 例器械移位，需要经房间隔路径重复干预。两例最终均处理成功。

### 生存和主要不良事件

围术期死亡、卒中、30 天生存和 1 年生存。

|  |  |
| --- | --- |
| 操作相关死亡 | 0 |
| 卒中 | 0 |
| 30 天生存 | 96%（26/27） |
| 1 年生存 | 89%（24/27） |

观察到没有操作相关死亡或卒中。没有发生急诊转外科、冠状动脉阻塞或人工瓣损伤。随访中有 3 例死亡， adjudication 判定与本次假性动脉瘤封堵操作无关。作者强调，观察到的死亡反映了这一高风险队列的竞争性合并症，而不是操作并发症。

### 器械选择

用于假性动脉瘤封堵的 off-label 器械选择。

| 器械 | 例数 | 何时选择 |
| --- | --- | --- |
| Amplatzer Vascular Plug II | n=17 | 多数病例优先选择；适用于窄到中等颈部；尺寸通常比颈部大 30%-50%。 |
| Amplatzer Septal Occluder | n=9 | 适用于宽颈假性动脉瘤；需要更大的表面覆盖或锚定时选择。 |
| Embolic Coils | n=1 | 适用于小而离散的交通口，尤其靠近关键结构时。 |

三类器械均为假性动脉瘤封堵的 off-label 使用。器械选择的核心是颈部宽度、交通口离散程度、是否靠近关键结构，以及是否需要更大覆盖和锚定。

### 结论

作者总结：在选择合适的高风险患者时，如果根据解剖、入路和器械策略进行个体化设计，经皮假性动脉瘤封堵是可行且安全的。
