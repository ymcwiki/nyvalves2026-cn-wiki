# 解剖表型如何影响我的 TTVI 决策过程

## 核心要点

* TTVI 的基本前提是 OMT 下仍有持续大于或等于重度 TR、症状与 TR 相关、预期寿命超过 1-2 年，并且影像质量足够。
* 基线 TR 越重、合合缘间隙越大，术后残余 TR 风险越高；在 n=171 的示例中，合合缘间隙 <10 mm 占 85%，≥10 mm 占 15%。
* 瓣叶构型影响 T-TEER：三叶构型占 71.1%，四叶构型占 28.9%；四叶构型和复杂喷射/大间隙时残余 TR ≥3+ 最高可达 45.5%。
* 残余 TR 等级影响预后：resTR ≥3+ 的免心衰住院生存和总生存均低于 resTR 2+ 或 ≤1+，图中显示 p 值为 0.005、<0.001、0.013 和 <0.001。
* 决策不只看影像，还要纳入 RV 功能、LV 功能、肺高压、出血风险和肾功能；中等肺高压以 WU 3-5 或 WU >3 被单独提示。
* 作者的实用排序为：预期可降至 <中度 TR 时选 T-TEER；合并 RV/LV 功能不全、出血风险、中等 PH、CKD 且预期可降至 <重度 TR 时仍可选 T-TEER；否则考虑原位 TTVR，再不适合原位时考虑异位 TTVR。

### TTVI 适应证框架

TTVI 的起点包括四个条件：在最佳药物治疗 OMT 下仍存在持续大于或等于重度 TR；症状与 TR 相关；预期寿命超过 1-2 年；影像质量高，足以支持介入规划和术中引导。

### GLIDE 评分系统

GLIDE 评分用间隙、喷流位置、影像质量、腱索密度和瓣叶构型评估 T-TEER 复杂程度。

| 参数 | Straightforward，0 分 | Complex，1 分 |
| --- | --- | --- |
| Septolateral Gap | 0-5 mm | ≥6 mm |
| Predominant Jet Location | Anteroseptal / Central | Posteroseptal / Anteroposterior / Diffuse |
| Image Quality | Good | Limited |
| Chordal Structure Density | Modest | High |
| Leaflet Configuration，En Face TR Jet Morphology | Oval / Linear | Star-Shaped |

### TR 表型驱动的治疗算法

算法按 Secondary、CIED 和 Primary TR 分流，并据 TEER 可行性、原位 TTVR 可行性和瓣叶病变类型选择策略。

算法首先把 TR 分为 Secondary、CIED 和 Primary。CIED 又区分 lead-related 与 lead-associated；lead-related 情形图中提示 Reposition/Extraction。

| 表型/分支 | 判定条件 | 推荐方向 |
| --- | --- | --- |
| Secondary，-- TEER | GLIDE-Score ≥2 points；以下 2 项或更多存在：Gap >8.5 mm、anteroposterior jet location、short/immobile leaflet、Type IV morphology。 | 再问 Orthotopic TTVR? 若不适合，OMT 或 Heterotopic TTVR；若适合，TTVR。 |
| Secondary，+/- TEER | GLIDE-Score ≥2 points；以下 1 项存在：Gap >8.5 mm、anteroposterior jet location、short/immobile leaflet、Type IV morphology。 | 再问 Orthotopic TTVR? 分支指向 TTVR，或 Complex T-TEER、Heterotopic TTVR、Conservative Treatment。 |
| Secondary/CIED，++ TEER | GLIDE-Score 0-1 points；Gap ≤5 mm；anteroseptal jet location；long, mobile leaflets；linear jet morphology。 | T-TEER。 |
| Primary，prolapse/flail | 瓣叶脱垂或 flail。 | T-TEER。 |
| Primary，leaflet restriction/perforation/destruction | 瓣叶限制、穿孔或破坏。 | Orthotopic TTVR 或 Heterotopic TTVR。 |

### Secondary 与 CIED 分支中的全身因素

在 Secondary 和 CIED 相关 TR 中，RV/LV 功能、PH、出血和肾功能会改变策略选择。

该页保留 Secondary 和 CIED 分支，强调在 TEER 可行性之外还要看 RV function、LV Function、PH (WU 3-5)、Bleeding、Renal function。-- TEER 分支仍以 GLIDE-Score ≥2 且多项不利因素为特征；+/- TEER 分支为 GLIDE-Score ≥2 且有 1 个不利因素；CIED 相关的较理想分支可进入 T-TEER。底部治疗选项仍包括 OMT/Heterotopic TTVR、TTVR、Complex T-TEER/Heterotopic TTVR/Conservative Treatment 和 T-TEER。

### 作者的最终决策顺序

最终流程：先确认 TTVI 适应证，再按预期残余 TR 和患者风险选择 T-TEER、原位 TTVR 或异位 TTVR。

最终页先重复四个前提：OMT 下仍持续大于或等于重度 TR；症状与 TR 相关；预期寿命 >1-2 年；影像质量高。随后给出四步治疗排序：

1. 孤立 TR 中，如果预期 T-TEER 后可达到 <中度 TR，选择 T-TEER。
2. 如果存在 RV/LV dysfunction、bleeding risk、moderate PH、CKD，但预期 T-TEER 后可达到 <重度 TR，仍可选择 T-TEER。
3. 考虑 Orthotopic TTVR。
4. 如果解剖不适合 Orthotopic TTVR，考虑 Heterotopic TTVR。
