# 病例 3：瓣架不对称或变形：我们需要完美圆形吗？

## 核心要点

* 病例为 67 岁女性，重度主动脉瓣狭窄，平均跨瓣压差 57 mmHg，EOA/AVA 0.72 cm2，EF 50%，合并 COPD、肺高压 60 mmHg。
* CTA 显示三叶均有均匀中重度钙化；主动脉瓣环最小径 21.4 mm、最大径 29.4 mm、面积 495.5 mm2、周长 81.7 mm。
* 原计划右股主入路、桡动脉辅助入路，23 mm 非顺应性 True Dilatation 预扩，植入 26 mm Sapien Ultra Resilia，标称容量 23 mL，并用标称容量 double-tap。
* 首次植入后瓣架扩张测得直径 21.7 mm，double-tap 后增加至 23.8 mm，增加 2.1 mm。
* 最终超声平均跨瓣压差 7 mmHg，侵入性平均压差 0 mmHg；病例重点不是追求几何上“完美圆”，而是确认扩张、反流、梯度和冠脉/入路结果。

### 封面与问题设定

### 病例基本情况

患者为 67 岁女性，因气短 NYHA III 和心绞痛 CCS II 就诊，近期因心脏失代偿入院。诊断为高度主动脉瓣狭窄，平均压差 57 mmHg，EOA 0.72 cm2；冠心病但无相关阻塞，EF 50%。

合并肺气肿、COPD 和肺高压，肺动脉压力标注为 60 mmHg。用药包括阿司匹林 100 mg、Ramipril/Amlodipin 2.5 mg、阿托伐他汀 20 mg、二甲双胍 1000 mg、L-Thyroxin 75 μg。风险和体型参数：EuroScore II 10.7%，STS 5.2%，身高 167 cm，体重 79 kg。

超声结论为重度主动脉瓣狭窄：平均压差 57 mmHg，AVA 0.72 cm2。左室 EF 轻度受损，左房和右房轻度扩大，右室功能正常。页面右下方连续波多普勒图与左侧彩色多普勒图共同支持狭窄严重度评估。

### 术式计划

* 右股动脉作为主入路，桡动脉作为辅助入路。
* 使用 23 mm True Dilatation 非顺应性球囊进行预扩张。
* 选择 Sapien Ultra Resilia 26 mm。
* 以标称容量 23 mL 植入 Sapien 3UR。
* 计划以标称容量进行 double-tap。
