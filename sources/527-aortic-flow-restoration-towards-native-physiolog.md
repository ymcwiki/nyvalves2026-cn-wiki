# 主动脉血流恢复趋向原生生理：新型仿生 TAVR 首次人体 CMR 影像结果

## 核心要点

* DurAVR 是首个 biomimetic THV，采用单片牛心包、原生瓣叶轮廓、球囊扩张钴铬支架，并设计为恢复更接近生理的层流。
* 专门 CMR 研究为前瞻性单臂影像研究，DurAVR 基线 N=60、6 月 N=46、配对 N=39，外部匹配对照 N=35。
* DurAVR 后 indexed systolic flow reversal ratio 从术前约 4.0 %/cm² 降至术后约 2.5 %/cm²，P<0.0001；与匹配对照约 2.2 %/cm² 相比差异接近但未达传统显著性，P=0.06。
* Indexed systolic flow displacement 从术前约 4.1 %/cm² 降至术后约 3.5 %/cm²，P<0.005；术后与匹配对照几乎一致，P=0.81。
* 6 月配对 CMR 显示 LV mass 从 136 ± 45 g 降至 112 ± 33 g，P<0.0001；PCWP 从 20 ± 4 降至 18 ± 4 mmHg，P<0.0001。
* 新发现：平均跨瓣压差改善与主动脉流场改善均为 LV mass regression 的独立预测因子；combined 模型解释/预测比例 43%，P=0.0005。

### DurAVR：首个仿生 TAVR

DurAVR 使用单片牛心包组织，塑造成独特原生瓣叶轮廓，目标是模仿健康主动脉瓣。

DurAVR 被定义为“First Biomimetic TAVR”。其核心结构为 single-piece bovine pericardial tissue，并塑造成 unique, native leaflet contour，以 mimic a healthy valve。幻灯片列出三项设计目标：在 easy-to-use balloon-expandable system 上展示 favorable hemodynamic performance；设计用于 restore physiologic laminar flow；通过 tissue、shape、flow 三方面支持 durability。

### 方法：用 2D 流量量化描述 4D flow 特征

方法页定义 SFD、FRR 和 flow rotation，并展示示例流量曲线。

该研究使用 2D flow quantification 来获得 4D flow signatures。Systolic flow displacement，SFD，单位百分比，代表 eccentricity；flow reversal ratio，FRR，单位百分比，代表 vorticity；flow rotation 代表 helicity。该方法已针对 translational science 与 4D flow MRI 验证。

右侧流量曲线示例显示：SFD 39%，FRR 29%，forward flow，FF，为 99 ml，backward flow，BF，为 29 ml。纵轴为 FLOW ML/SEC，刻度约从 -200 到 500 ml/sec；图中标出 systole。引用：Mehmood Z et al. Open Heart. 2024 Mar 8;11(1):e002451。

### 专门 CMR 研究设计

前瞻性单臂影像研究，并使用外部血流动力学匹配对照来评估 DurAVR 对血流恢复的影响。

| 项目 | 内容 |
| --- | --- |
| Design | Prospective single-arm imaging study with external hemodynamics-matched controls。 |
| Population | DurAVR：重度、有症状主动脉瓣狭窄，经判断可从 TAVR 获益的患者。Control：年龄、性别和血流动力学匹配，且无主动脉瓣疾病的对照。 |
| Cohorts | DurAVR：来自 EMBARK FIH 和 US-EFS，且有 MRI imaging。Control：来自 PREFER-CMR Registry 的外部队列。 |
| Sample Size | DurAVR MRI imaging：baseline N=60，6 months N=46，paired N=39。Control MRI imaging：N=35。 |
| Outcomes | Flow：indexed aortic flow displacement 和 indexed aortic flow reversal ratio。另包括 left ventricular mass、structure、function。 |
| Independent Data Oversight | Imaging core lab，MRI。 |

对照纳入标准：LV ejection fraction >50%，right ventricular ejection fraction >50%，absence of late gadolinium enhancement，native T1 <1100 ms。对照按 age 与 sex 与 post-DurAVR 队列做 frequency matching；血流动力学匹配基于 NT-proBNP 和 CMR-derived pulmonary capillary wedge pressure，作为 neurohormonal activation 和 left-sided filling pressure 的替代指标。

### 研究队列

DurAVR 组来自 EMBARK FIH 和 US-EFS，对照组来自 PREFER-CMR Registry。

### 结果：恢复前向生理血流，FRR 下降

Indexed systolic flow reversal ratio 降低，提示旋涡/逆向血流减少，术后接近无主动脉瓣疾病对照。

纵轴为 indexed FRR，单位 %/cm²，刻度从 0 到 6。图中说明 turbulent flow 对应 higher FRR，laminar flow 对应 lower FRR。Pre-DurAVR severe AS N=60 的柱高约 4.0 %/cm²；Post-DurAVR N=46 约 2.5 %/cm²；Matched controls N=35 约 2.2 %/cm²。Pre-DurAVR 到 Post-DurAVR 的比较 P<0.0001；Post-DurAVR 与 matched controls 比较 P=0.06。

FRR 定义：systolic flow reversal ratio = systolic backward flow / systolic forward flow，并 indexed to aortic area。

### 结果：血流中心化，FD 下降

Indexed systolic flow displacement 降低，提示射流偏心度下降，术后与匹配对照无差异。

纵轴为 indexed FD，单位 %/cm²，刻度从 0 到 6。图中说明 turbulent flow 对应 higher FD，laminar flow 对应 lower FD。Pre-DurAVR severe AS N=60 柱高约 4.1 %/cm²；Post-DurAVR N=46 约 3.5 %/cm²；Matched controls N=35 约 3.5 %/cm²。Pre-DurAVR 到 Post-DurAVR 比较 P<0.005；Post-DurAVR 与 matched controls 比较 P=0.81。

Systolic flow displacement 定义：distance from the center of the ascending aorta to the highest jet velocity / ascending aorta diameter，并 indexed to aortic area。

### 新发现：MPG 改善和主动脉血流改善独立预测 LVM 回缩

以 LV mass regression ≥10 g 为终点，ΔFRR 和 ΔMean PG 均与 LVM 回缩相关，合并模型解释比例最高。

分析类型为 left ventricular mass regression 的 univariate and multivariate regression analysis。终点定义为 LV mass regression ≥10 grams。柱状图显示：ΔFRR 为 14%，P=0.02；ΔMean PG 为 32%，P=0.001；Combined 为 43%，P=0.0005。

图上方比较标注：ΔFRR 与 ΔMean PG/模型之间有 P=0.005；ΔMean PG 与 Combined 之间标注 P=0.03。Combined 堆叠柱进一步拆分为 Shared 6%，ΔFRR 26%，ΔMPG 68%。

### 下一步

初步 CMR 数据提示，flow 是 LV mass regression 的独立预测因素。仍需进一步研究 THV prosthesis performance 与 laminar flow 的联系。需要评估的器械性能因素包括 frame expansion、circularity、implant position、leaflet mobility、absence of HALT。

### 总结

DurAVR 是首个 biomimetic THV。初步 CMR 结果提示，DurAVR 可将生理层流恢复到接近无主动脉瓣疾病匹配对照的状态。DurAVR 后 6 个月可观察到有意义的左室重构和心脏功能恢复。新发现是平均压差改善和血流改善均与 LV mass regression 独立相关。THV prosthesis performance 与 laminar flow 的联系仍需进一步评估。这些发现将在 PARADIGM trial 中继续验证。
