# 经 Evolut 经导管回拉栓塞 Sapien：TRAPEZE 在 TAV-in-TAV 中抢救栓塞 Sapien

## 核心要点

* 患者为 72 岁男性，既往 2018 年植入 34 mm Evolut，本次因 Evolut 严重 AR、肺水肿和休克转入，外科换瓣风险高，计划 TAVR-in-TAVR。
* 深植入的大号 Evolut 形成明显口径差：流入端 34 mm，腰部约 26 mm。S3 与 Evolut 相互作用后可被推向心室侧，尤其在 node 5或更低位置释放时栓塞风险高。
* 第一枚 26 mm Sapien 3向 LV 栓塞，团队建立体外循环，球囊回拉失败后，在 S3 内释放 29 mm Evolut并上提，把迁移的 S3 拉回 LVOT。
* 抢救后两层瓣架之间出现严重 PVL，第二枚 S3 又因两个 Evolut 的弹簧圈效应迁移到 node 3-4。
* 最终改变策略，第三枚 26 mm Sapien高位释放于 node 7，RCA 位于 node 6并以 Guideliner 保护。
* TRAPEZE 是针对不能手术患者抢救栓塞 S3 的补救技术。患者最终恢复，但因严重 LV 功能不全和心衰需要 LVAD，距本次 index procedure 已超过 1 年。

### 标题与机构

本讲题为“经 Evolut 经导管回拉栓塞 Sapien，TRAPEZE，在 TAV-in-TAV 中抢救栓塞 Sapien”。讲者为 Puvi Seshiah MD FRCP FSCAI，来自 The Christ Hospital, Cincinnati。

### 病例病史

患者为 72 岁男性，主动脉瓣狭窄，既往接受 TAVR；合并冠心病、CABG 术后、严重左室功能不全、CRT-D 植入后和慢性肾病。2018 年接受 TAVR，植入 34 mm Evolut。本次以肺水肿和休克就诊，检查显示 Evolut 瓣重度主动脉瓣反流。因外科主动脉瓣置换风险高，转诊行 TAVR-in-TAVR。

### 严重 AR 的 TEE 表现：瓣叶脱垂

TEE 显示严重主动脉瓣反流，右侧彩色多普勒可见显著反流束，标题标注 flail leaflet。

该页展示严重 AR 的经食道超声证据。左图为二维 TEE，可见人工瓣内瓣叶异常活动；右图为彩色多普勒，显示通过瓣膜的大量反流血流。页面标题明确指出“Severe AR TEE. Flail leaflet”，即严重主动脉瓣反流，伴瓣叶脱垂或飘动。

### 34 mm Evolut 的 TAV-in-TAV CTA 规划

CTA 规划显示既往 34 mm Evolut 瓣架、冠脉对位关系和横断面评估，用于 TAV-in-TAV 策略制定。

该页围绕既往 34 mm Evolut进行 CTA 规划。图中包括冠状面/共面重建、冠脉错位或对位关系、瓣架横断面和三维方向标记，用来判断再植入 Sapien 时的高度、冠脉风险和瓣架约束。

### Evolut 深植入与节点定位

CTA/透视图按 Evolut 瓣架节点标注：原生瓣环在 node 3，左主干在 node 5，右冠在 node 6。

既往 TAVR 为深植入。原生主动脉瓣环位于 node 3；左主干冠脉位于 node 5；右冠位于 node 6。这些节点定义了后续 Sapien 释放窗口，也提示低位释放时向心室侧迁移的风险。

### 初始手术计划：26 mm S3 置于 N5 与 N6 之间

规划将 26 mm Sapien 3 置于 node 5 与 node 6 之间，同时评估右冠 VTC/VTA 和左主干闭塞情况。

术前计划为在 node 5与node 6之间释放 26 mm Sapien 3。页面同时强调 LM CTO，即左主干慢性闭塞；担心右冠瓣叶或右冠窦在 node 6以上被隔离，产生冠脉通路或冠脉闭塞风险。

### Sapien 初始位置与突然向心室侧弹出

TEE/透视引导下，S3 流出端最初位于 index Evolut 的腰部，接触后突然向心室侧移位。

页面记录第一枚 Sapien 的关键失败瞬间：Sapien 流出端最初位于 index Evolut 的腰部；一旦与 Evolut 接触，Sapien 流出端突然“pop”向心室侧，即整体被推向左室方向。

### 第一枚 Sapien 栓塞入 LV，转入体外循环

透视和 TEE 显示第一枚 Sapien 已栓塞到左室内，团队随后置入体外循环支持。

第一枚 Sapien 栓塞进入左室。由于患者处于高危状态，团队将患者置于 bypass 支持。页面左侧为透视下瓣架位置，右侧为 TEE 确认左室内移位。

### 尝试用球囊重新捕获并回拉 S3

团队尝试用球囊进入 S3 并回拉，透视和 TEE 同步评估瓣架位置。

在第一枚 S3 栓塞后，团队尝试重新捕获 S3，并通过球囊回拉。页面显示透视下球囊/导管与瓣架的关系，以及 TEE 监测左室和瓣架的位置。该策略未能作为最终解决方案。

### 在 S3 内释放 29 mm Evolut 并上提至 LVOT

29 mm Evolut 被释放在 S3 内，通过“Evolut pull up”把迁移的 S3 拉回左室流出道。

团队在迁移的 S3 内释放 29 mm Evolut，随后将这套结构上提，使迁移的 S3 回到 LVOT。该步骤构成 TRAPEZE 抢救技术的核心：借助自膨瓣架与球扩瓣架的相互作用，把已栓塞的 S3 经导管拉回。

### S3 抢救成功，但两层瓣架间严重 PVL

TEE 彩色多普勒显示，S3 经 Evolut pull up 成功抢救后，两层瓣架之间仍存在严重瓣周漏。

S3 通过“Evolut pull up”成功抢救。但页面明确指出，在两层瓣架之间存在严重 PVL。多幅彩色多普勒图显示明显反流束，提示虽然瓣架位置被救回，密封和同轴性仍存在严重问题。

### 第二枚 Sapien 初始位置尚可，随后迁移至 node 3-4

第二枚 Sapien 初始在 cine 上位于 node 6 附近，但随后人工瓣迁移到 node 3-4，红箭头标出深移位置。

第二枚 Sapien 部署时，初始 cine 影像显示位置良好，位于 node 6附近。但之后人工瓣再次向深部迁移，最终落在 node 3-4。这提示前一层结构和原有 Evolut 之间的机械相互作用仍会影响球扩瓣释放稳定性。

### 到目前为止的手术状态

连续透视图总结前序过程：S3 栓塞、29 mm Evolut 抢救、随后 S3 因两个 Evolut 的弹簧圈效应再次迁移。

该页总结：已发生一枚 S3 栓塞；用 29 mm Evolut抢救 S3；随后置于 29 mm Evolut 内的 S3 又因两个 Evolut 的 spring coil effect，即弹簧圈效应，发生迁移。

### 改变策略：第三枚 Sapien 高位部署并保护 RCA

策略改为第三枚 Sapien 高位释放至 node 7；由于 RCA 位于 node 6，右冠存在风险，已置入 Guideliner。

团队改变策略：第三枚 Sapien 准备高位释放在 node 7。由于 RCA 在 node 6，右冠存在风险，因此右冠已被导管选择性插入，并放置 Guideliner。

### 第三枚 26 mm Sapien 的最终节点关系

透视图标注第三枚 26 mm Sapien 的高位释放位置、RCA node 6、既往 Evolut 深植入和前两枚 Sapien/Evolut 的关系。

第三枚瓣膜为 26 mm Sapien。节点关系如下：RCA 位于 node 6；第一枚 Evolut 深植入，原生瓣环位于 node 3；第一枚迁移的 Sapien 位于 LVOT；第二枚 Sapien 深植入在 node 3-4；第二枚 Evolut 的流出端位于 node 6；第三枚 Sapien 放在 node 7。

### 第三枚 Sapien 后的超声结果

第三枚 Sapien 释放后，即当天第 4 次 TAVR 操作后的 TEE 结果。

页面标题指出这是“After Third Sapien”，也是当天的第 4 次 TAVR。TEE 彩色多普勒显示第三枚 Sapien 后的血流评估，用于确认最终血流动力学和反流控制情况。

### 学习点：深植入 Evolut 的机械风险

学习点图示 node 6、node 5 和 node 0 的相对位置，强调深植入 Evolut、腰部变窄和 Sapien 向心室侧受力的风险。

深植入 Evolut 瓣架具有挑战性，因为流入端 34 mm与腰部 26 mm之间存在大尺寸差异。如果瓣架有宽大的流入端，尤其是 34 mm规格，在 node 5或更低位置释放会有较高栓塞风险。S3 球囊与 Evolut 相互作用可能迫使 S3 瓣架向心室侧移动。图中红色箭头代表 node 6，蓝色箭头代表 node 5，黄色箭头代表 node 0。

### 结论

TRAPEZE 是“TRAnscatheter Pullback of EmboliZed Sapien with Evolut”的缩写，意为借助 Evolut 经导管回拉栓塞 Sapien。它是一种补救技术，用于无法接受外科手术患者中取回或抢救已栓塞的 S3。患者最终结局尚可，但因严重左室功能不全和心衰需要植入 LVAD；距本次 index procedure 已经超过 1 年。
