# TAVR 后无创压力恢复与超声-有创梯度不一致 - 摘要

## 来源
- 原始文件：[[sources/542-non-invasive-pressure-recovery-and-echo-invasive.md]]
- 收录日期：2026-06-29

## 核心内容
本文报告一项前瞻性双中心研究（189 例重度原生主动脉瓣狭窄 TAVR 患者），首次系统量化压力恢复（PR）作为 TAVR 后超声-有创跨瓣压差不一致（ΔMG）的机制决定因素。结果显示 PR 与 ΔMG 强相关（T0 r=0.50，T1 r=0.80），且 PR 倾向于高估真实不一致幅度（残余误差约 +3 mmHg）。23 mm 自膨瓣亚组的 ΔMG 和 PR 均显著更高，提示小瓣 TAVR 后高超声梯度时需主动考虑压力恢复的贡献。

## 关键实体
- [[实体/evolut|Evolut]]：Evolut 是 Medtronic 生产的自膨式经导管主动脉瓣（TAVR）系列，涵盖 R、PRO、PRO+、FX 等多个亚型，适用尺寸从 23 mm 到 34 mm，以高框架结构和瓣上设计为主要特征。其血流动力学性能经多项研究验证，在小主动脉瓣环（SAA）患者中平均跨瓣压差约 8–9 mmHg，重度人工瓣膜-患者不匹配（PPM）及亚临床瓣叶血栓发生率均低于球扩式瓣膜，但起搏器植入率相对较高。临床实践中，Evolut 既作为初次 TAVR 的主流自膨式平台使用，也频繁出现于瓣中瓣（TAV-in-TAV）、瓣周漏处理及结构性退化后再干预等复杂场景，其 inflow 端向外张开的高框架特性在 Redo TAVR 规划时需特别考量。
- [[实体/gianmaria-calamita|Gianmaria Calamita]]：Gianmaria Calamita 是两项医学研究的主要讲者，在双中心前瞻性研究中承担核心学术角色。其本人均声明无相关财务利益冲突或披露。
- [[实体/acurate-neo2|ACURATE Neo2]]：ACURATE Neo2 是 Boston Scientific 研发的自膨式经导管主动脉瓣膜（TAVR），采用自上而下释放设计。在 SCOPE I/II 及 ACURATE IDE 三项 RCT 中均未能达到非劣效终点，事后分析确认核心问题为钙化瓣环中中段框架展开不足（发生率约 20-21.6%），导致瓣周漏显著增高并与一年不良事件相关。
- [[实体/sapien-resilia|Sapien Resilia]]：Edwards Lifesciences 球扩 TAVR 瓣膜，占本研究 7.7%，为球扩瓣代表。

## 要点
- 压力恢复（PR）与 TAVR 后超声-有创梯度不一致（ΔMG）强相关：T0 r=0.50，T1 r=0.80，均 p<0.001
- PR 倾向于高估真实不一致幅度：残余误差 ε 在 T0 为 +3.4 mmHg，T1 为 +3.1 mmHg
- 23 mm SEV 亚组的 ΔMG 和 PR 均显著高于 25-34 mm 大瓣（p<0.001）
- 出院前（T1）超声平均压差高于术中（T0），差异部分由 PR 变化解释
- TAVR 后高超声梯度时应将压力恢复纳入鉴别诊断，避免误判为人工瓣狭窄
