# 病例解决 (Case Resolution) — Atrial Secondary Mitral Regurgitation - 摘要

## 来源
- 原始文件：[[sources/097-case-resolution.md]]
- 收录日期：2026-06-29

## 核心内容
本报告描述一例91岁高龄女性合并长期房颤、HFpEF和重度房性功能性二尖瓣/三尖瓣反流的治疗过程。术者采用MitraClip XTW进行经导管缘对缘修复（TEER），术后形成双孔二尖瓣（3D MVA 2.08 cm²，MG 3 mmHg），残余轻度MR。出院后优化GDMT，3年随访NYHA I级，提示TEER对此类高危患者可获得持久症状改善。

## 关键实体
- [[实体/mitraclip-xtw|MitraClip XTW]]：MitraClip XTW 是 Abbott 公司生产的宽臂型经导管缘对缘修复（TEER）夹合装置，属于 MitraClip 系列中的宽夹型号（W 型），设计上具有更大的抓瓣面积，适用于接合间隙较大或瓣叶条件要求宽覆盖的病例。临床上主要用于二尖瓣 A2-P2 区域的夹合，通常单枚或与 NT 型联合使用，可将二尖瓣反流即刻降至 trivial 或 0 级，也有用于三尖瓣缘对缘修复的报道。手术中存在 kinking 导致器械受损、瓣叶条件限制引发 SLDA、以及远期复发重度反流等潜在风险。
- [[实体/abbott|Abbott]]：Abbott 是全球领先的医疗器械公司，在结构性心脏病介入治疗领域覆盖多个细分方向，旗下产品包括二尖瓣修复系统 MitraClip、三尖瓣修复系统 TriClip、经导管主动脉瓣置换系统 Navitor 和 Encantor，以及左心耳封堵、房间隔封堵等器械。该公司同时作为多项临床研究（含 EASE FIH、BE TAVI EFS、Navitor IDE 等）的赞助方，并为心血管领域学术教育活动提供资金支持。在披露财务关系的学术讲座中，Abbott 频繁作为顾问费、讲课酬金或研究资助的来源方出现。
- [[实体/nino-mihatov|Nino Mihatov]]：Nino Mihatov 是 Weill Cornell Medicine 与 New York Presbyterian-Brooklyn 的介入心脏病医生，专注于结构性心脏病及 TAVR 后并发症处理，曾作为讲者参与本病例报告。与 Edwards Lifesciences 和 Abbott 存在顾问费关系。

## 要点
- 91岁女性，持续性房颤≥7年，HFpEF，重度atriofunctional MR/TR，6个月内因心衰住院3次
- 使用MitraClip XTW行TEER，术后3D MVA 2.08 cm²，MG 3 mmHg，形成双孔
- 残余MR分内侧（EROA 3 mm²）和外侧（EROA 14 mm²）两个喷射
- 出院GDMT包括dapagliflozin、spironolactone、Bumex、metoprolol、apixaban
- 3年随访mild MR、moderate TR、NYHA I，未再接受额外侵入性治疗
