# 病例汇报：M-TEER 后复发 MR - 摘要

## 来源
- 原始文件：[[sources/444-case-presentation.md]]
- 收录日期：2026-06-29

## 核心内容
本病例报告描述一例 79 岁男性患者，因重度二尖瓣反流和重度三尖瓣反流合并严重 CAD 行 M-TEER，术中第一枚 XTW 夹部署后发生急性单叶装置附着（SLDA），并伴重度 MR 复发。心脏团队选择 bailout TEER 策略，依次部署第二枚 XTR（外侧）和第三枚 XTR（内侧），最终实现 trace 残余 MR，肺静脉血流恢复，MV 平均压差 3 mmHg。本病例展示了复杂 M-TEER 中 SLDA 并发症的实时处理决策流程。

## 关键实体
- [[实体/mitraclip-xtw|MitraClip XTW]]：MitraClip XTW 是 Abbott 公司生产的宽臂型经导管缘对缘修复（TEER）夹合装置，属于 MitraClip 系列中的宽夹型号（W 型），设计上具有更大的抓瓣面积，适用于接合间隙较大或瓣叶条件要求宽覆盖的病例。临床上主要用于二尖瓣 A2-P2 区域的夹合，通常单枚或与 NT 型联合使用，可将二尖瓣反流即刻降至 trivial 或 0 级，也有用于三尖瓣缘对缘修复的报道。手术中存在 kinking 导致器械受损、瓣叶条件限制引发 SLDA、以及远期复发重度反流等潜在风险。
- [[实体/abbott|Abbott]]：Abbott 是全球领先的医疗器械公司，在结构性心脏病介入治疗领域覆盖多个细分方向，旗下产品包括二尖瓣修复系统 MitraClip、三尖瓣修复系统 TriClip、经导管主动脉瓣置换系统 Navitor 和 Encantor，以及左心耳封堵、房间隔封堵等器械。该公司同时作为多项临床研究（含 EASE FIH、BE TAVI EFS、Navitor IDE 等）的赞助方，并为心血管领域学术教育活动提供资金支持。在披露财务关系的学术讲座中，Abbott 频繁作为顾问费、讲课酬金或研究资助的来源方出现。
- [[实体/mitraclip-xtr|MitraClip XTR]]：MitraClip XTR 是 Abbott 公司 MitraClip 系统的延伸型（XTR）夹子，属于经导管缘对缘修复（TEER）器械，可用于二尖瓣或三尖瓣修复。在相关病例中，该型号被用于二尖瓣 SLDA 发生后的 bailout 处置，在外侧与内侧各放置一枚以稳定首枚夹并消除残余 MR；同一患者既往也在三尖瓣位置植入过 2 枚此类器械，后因夹间外科缝合及 TR 复发，形成复杂解剖背景。
- [[实体/elliott-elias|Elliott Elias]]：Elliott Elias 是一位专注于复杂二尖瓣经导管介入领域的临床讲者。他在病例结局报告中聚焦于复杂 M-TEER（二尖瓣经导管缘对缘修复）术中出现的 SLDA 并发症，并系统阐述了相应的 bailout 处理策略。

## 要点
- 患者为 79 岁男性，重度退行性 MR 合并重度 TR，既往 CABG，严重多支 CAD 但桥血管通畅
- 后瓣长度 9-11 mm，MVA>4 cm²，术前决策选用 XTW 夹
- 第一枚夹后发生 SLDA 并伴重度 MR 复发，为严重术中并发症
- 救治策略：外侧放置第二枚 XTR（11/5 方向）以稳定第一枚夹，再于内侧部署第三枚 XTR（1/7 方向）
- 最终结果：trace residual MR，MV mean PG 3 mmHg，肺静脉血流恢复
- 后瓣组织特征（长度和承受机械应力）是 edge-to-edge 修复夹子选择的核心依据
